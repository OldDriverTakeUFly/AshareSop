# 报告模板参考

本文件补充 SKILL.md §2（报告类型路由），给出 4 种报告类型的完整章节结构。写作前根据报告类型选用对应模板。所有模板的结构提炼自 `docs/` 里的黄金标准报告。

## 共通规则（所有类型适用）

### 开头元数据块（必备）

```markdown
# {标题}：{洞察性副标题，暗示结论}

> **研究日期**：YYYY-MM-DD | **股价/数据快照**：{关键数字} | **总市值**：{市值}
> **定位**：{1 句话业务/行业定位}
> **核心逻辑**：{3-5 句论点：关键财务 + 拐点判断 + 为什么重要}
> **数据来源**：{穷举具名来源：Tushare Pro, davis_analyzer, 年报/季报, 具名券商研报(带日期), 行业数据库}
> **方法论说明**（可选）：{引用 docs/方法论/ 下的文档，说明本报告使用的框架}
```

### 结尾固定结构（必备）

1. **风险提示**——≥4 条具体风险（不是泛泛的"市场风险"），编号列出
2. **免责声明**——AI 辅助 + 不构成投资建议 + 风险枚举 + 使用限制 + "市场有风险，投资需谨慎"
3. **报告定位与下游引用**——T0/T1 角色定位、更新频率
4. **版本历史表**：
   ```markdown
   ## 版本历史
   | 版本 | 日期 | 变更说明 (delta) |
   |------|------|-----------------|
   | v1.0 | YYYY-MM-DD | 初版创建... |
   ```

### 量化要求

- 每个小节**至少一个数据表**（markdown 表格）
- 核心洞察用 `> **核心洞察**：...` 引用块，全文 ≥5 个
- 所有指标量化（禁止"明显增长"，用"+37%（来源）"）
- ≥80% 数字带 inline 来源标签

## 模板 A：个股深度研报（景气度 11 章结构）

**适用**：单只 A 股的深度分析。黄金标准：`docs/个股研报/光储新能源/阳光电源深度研报.md`

```markdown
# {公司名}（{代码}.SZ/SH）深度研报：{副标题}

[元数据块]

## 目录
[带锚点的章节链接]

## 第一章：公司概览与股东背景（深度篇——这是研报的灵魂，必须详尽）
> **强制要求**：本章是研报的"灵魂"，必须呈现公司的"故事"——创始人是谁、公司怎么发展来的、现在在行业什么位置。参考 `docs/个股研报/半导体电子/德福科技深度研报.md` 第一章为黄金标准（创始人+发展历程+业务布局+产品矩阵）。
> 严禁只放一张"基本信息表"就结束——那只是占位，不是公司分析。

### 1.1 创始人与管理层（强制，web 搜索补充）
- **实控人/董事长**：姓名、出生年、持股比例、创业路径（如何起家）
- **核心高管（总经理/CTO/技术负责人）**：姓名、学历背景（尤其技术型高管的专业背景）、过往经历
- **管理风格与战略视野**：技术驱动型？资本运作型？市场导向型？
- **管理层特点表格**：实控人/总经理/研发投入/战略方向/员工规模/总部
> 关键洞察：用 1-2 段分析"管理层组合"如何塑造公司基因（如德福"马科战略+罗佳技术"双轮）

### 1.2 发展历程（强制，web 搜索补充——这是公司的"编年史"）
- **完整发展历程表格**：年份 | 里程碑事件 | 战略意义 | 所处阶段
- **阶段划分**：创业起步期 → 转型/切入期 → 扩张期 → 当前阶段（升级/平台期）
- **关键转折点标注**：上市、重大技术突破、并购、战略转向等
> 关键洞察：分析关键转折点（如上市、技术突破）如何改变公司命运

### 1.3 公司定位与业务布局
- **一句话定位**（公司在行业中的核心位置）
- **全球/全国布局**：生产基地数量与分布、研发中心/实验室、客户覆盖
- **股权结构关键节点**：实控人/机构投资者/流通股

### 1.4 股东结构特征
- 总市值、A+H 双重上市（如有）、国资/民企背景、股权稳定性

## 第二章：业务范围与产品矩阵（深度篇——呈现技术方向）
> **强制要求**：必须呈现"公司做什么产品、技术代际如何演进、未来往哪个技术方向发展"。这是判断公司竞争力的核心。

### 2.1 经营范围（法定）
### 2.2 核心产品矩阵——{N} 大业务板块（表格化：产品/应用场景/行业地位）
### 2.3 分业务收入结构（最新年报表格：产品/收入/占比/毛利率/战略定位）
### 2.4 技术代际演进与技术发展方向（强制！）
- **当前技术代际**：公司主流产品的代际位置（如 HVLP1-4 代、KrF/ArF、6μm/4μm）
- **技术演进路径**：从过去到现在到未来的技术迭代方向
- **未来技术布局**：下一代技术、前沿储备（如固态电池铜箔、EUV 光刻胶、先进制程验证）
> 关键洞察：分析技术布局如何构筑护城河、未来增长曲线在哪
### 2.5 海外/新兴业务（如有）

### 9.1 {行业} 公司全景（一句话定位每家同行）
```

## 模板选择决策树

```
用户提到一只具体股票？
  ├─ 是 → 要深度还是快速？
  │       ├─ 深度 → 模板 A（个股 11 章）
  │       └─ 快速 → 模板 D（分析笔记）
  └─ 否 → 提到行业/材料/技术？
          ├─ 是 → 有明确的景气周期信号吗？
          │       │  （价格在涨/交期拉长/产能紧缺/订单爆发/
          │       │   BB Ratio>1/稼动率高/有涨价弹性）
          │       ├─ 有 → 模板 B1（产业链 6 节，景气度导向）
          │       └─ 无/不确定 → 模板 B2（产业链 8 章，全景式）
          └─ 提到投资方法/框架？
                  └─ 是 → 模板 C（方法论 8 章）
```

**B1 vs B2 判断补充**：如果产业链处于"供需缺口 + 涨价 + 稼动率满载"的上行周期（如钽电容、光模块），用 B1。如果是新兴技术早期、格局未定、或以科普/梳理为主（如全光互联、玻璃基板早期），用 B2。不确定时默认 B2，因为 6 节结构需要六维指标数据，缺数据时强写会空洞。

不确定报告类型时，问用户："你想要的是深度研报、产业链全景、方法论梳理、还是快速分析？"
