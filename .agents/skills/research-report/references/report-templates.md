# 报告模板参考

本文件补充 SKILL.md §2（报告类型路由），给出 4 种报告类型的完整章节结构。写作前根据报告类型选用对应模板。所有模板的结构提炼自 `docs/` 里的黄金标准报告。

## 共通规则（所有类型适用）

### 开头元数据块（必备）

```markdown
# {标题}：{洞察性副标题，暗示结论}

> **研究日期**：YYYY-MM-DD | **股价/数据快照**：{关键数字} | **总市值**：{市值}
> **定位**：{1 句话业务/行业定位}
> **核心逻辑**：{3-5 句论点：关键财务 + 拐点判断 + 为什么重要}
> **数据来源**：{穷举具名来源：Tushare Pro, davis_analyzer, 年报/季报, 具名券商研报(带日期), 行业数据库}
> **方法论说明**（可选）：{引用 docs/方法论/ 下的文档，说明本报告使用的框架}
```

### 结尾固定结构（必备）

1. **风险提示**——≥4 条具体风险（不是泛泛的"市场风险"），编号列出
2. **免责声明**——AI 辅助 + 不构成投资建议 + 风险枚举 + 使用限制 + "市场有风险，投资需谨慎"
3. **报告定位与下游引用**——T0/T1 角色定位、更新频率
4. **版本历史表**：
   ```markdown
   ## 版本历史
   | 版本 | 日期 | 变更说明 (delta) |
   |------|------|-----------------|
   | v1.0 | YYYY-MM-DD | 初版创建... |
   ```

### 量化要求

- 每个小节**至少一个数据表**（markdown 表格）
- 核心洞察用 `> **核心洞察**：...` 引用块，全文 ≥5 个
- 所有指标量化（禁止"明显增长"，用"+37%（来源）"）
- ≥80% 数字带 inline 来源标签

## 模板 A：个股深度研报（景气度 11 章结构）

**适用**：单只 A 股的深度分析。黄金标准：`docs/个股研报/光储新能源/阳光电源深度研报.md`

```markdown
# {公司名}（{代码}.SZ/SH）深度研报：{副标题}

[元数据块]

## 目录
[带锚点的章节链接]

## 第一章：公司概览与股东背景
### 1.1 公司基本信息
### 1.2 控股股东与创始人
### 1.3 股东结构特征

## 第二章：业务范围与产品矩阵
### 2.1 经营范围（法定）
### 2.2 核心产品矩阵——{N} 大业务板块
### 2.3 分业务收入结构（最新年报）
### 2.4 海外/新兴业务（如有）

## 第三章：财务分析——{拐点诊断}
### 3.1 营收与利润轨迹（近 8 季度表格：营收/归母净利/同比增速/毛利率/净利率/ROE）
### 3.2 {上升/下降}拐点的成因诊断（关键！列出具体原因）
### 3.3 资产负债表质量
### 3.4 现金流质量

## 第四章：景气度分析——G+ΔG 框架下的周期定位
### 4.1 引擎计算结果（davis_analyzer 输出）
### 4.2 G 与 ΔG 的人工核算（关键诊断）
### 4.3 山峰理论定位（Δg 符号 → 左山坡/右山坡/山后）
### 4.4 30% 阈值检验
### 4.5 二次点火筛选

## 第五章：六维指标体系
### 5.1 {行业}六维评估
[BB Ratio / 稼动率 / 交期 / 库存 / 排产 / LTA 六维表]

## 第六章：强势点与弱势点
### 6.1 强势点（Strengths）
### 6.2 弱势点（Weaknesses）

## 第七章：未来发展规划与战略路径
### 7.1 公司披露的战略方向
### 7.2 隐含的战略逻辑
### 7.3 机构盈利预期

## 第八章：估值分析——历史分位、三模型与情景区间
### 8.1 历史估值分位（近 3 年 PE/PB/PS，10/25/50/75/90/95 分位 + 当前）
### 8.2 三模型估值——PE + PB + PS 三角验证
#### 模型 1：PE 法
#### 模型 2：PB 法
#### 模型 3：PS 法
#### 三角验证综合（三模型收敛/发散判断）
### 8.3 可比公司估值对比
### 8.4 概率加权目标价（悲观20%/中性50%/乐观30%，展示加权公式）
### 8.5 估值结论（定性判断 + 合理区间）

## 第九章：横向对比——{公司} vs {peer}（全景对照）
[≥8 维度对比表]

## 第十章：风险提示与研究局限
### 10.1 主要风险（≥4 条，编号）
### 10.2 研究局限（信息缺乏的诚实声明——标注估算/不可用，"未编造"）

## 第十一章：结论与投资框架定位
### 11.1 一句话结论
### 11.2 最终评估（投资评级 ★ 或 框架定位）

## 附录
### 附录一：数据来源索引（D1-Dn 表，带可靠度评级）
### 附录二：引擎调用记录（davis_analyzer 计算明细）
### 附录三：历史研报存档（若整合了旧报告）

## 版本历史
[结尾固定结构]
```

**亏损标的变体**：第八章改为三角框架——困境评分（Angle A）+ PS/DCF 双模型（Angle B）+ 全球同业锚定（Angle C），遵循 `valuation-loss-making-targets` skill。禁止 PE 主估值。

## 模板 B：产业链研报

**适用**：一个行业/材料/技术的全景分析。黄金标准：`docs/产业链研报/钽电容产业链深度研报.md`

有两种子结构，按主题景气度导向选用。

### B1：景气度 6 节结构（推荐，景气导向的产业链）

```markdown
# {主题}产业链深度研报——{slogan 副标题}

[元数据块]

## 目录

## 方法论框架导读
[表格：节次 → 核心问题 → 方法论 → 输出结论]
[声明：本报告用景气度六节结构，与既有 XX 报告的边界]

## 第一节：供需格局与缺口量化
### 1.1 上游供给（全球集中度、产能、扩产计划）
### 1.2 加工/中游环节
### 1.3 下游需求驱动（AI/新能源等）
### 1.4 供给的弹性/刚性分析
### 1.5 缺口量化（供需增速剪刀差 + 供需平衡表）

## 第二节：景气度六维指标监控
### 2.1 BB Ratio（订单出货比）
### 2.2 稼动率（产能利用率）
### 2.3 交期（Lead Time）
### 2.4 经销商库存
### 2.5 排产周期
### 2.6 LTA（长期供货协议）
### 2.7 六维综合评分

## 第三节：二阶导周期定位
### 3.1 山峰理论定位（Δg → 左/右山坡）
### 3.2 成长股投资时钟（4 象限）

### 3.4 与历史周期对比
### 3.5 个股分化（行业 β ≠ 个股 α）

## 第四节：国产替代与卡脖子分析
### 4.1 中国的强项
### 4.2 中国的短板（国产化率量化）
### 4.3 卡脖子环节精确定位
### 4.4 国产替代进度
### 4.5 "卡位咽喉"企业识别

## 第五节：涨价弹性与紧缺预测
### 5.1 上游价格行情
### 5.2 中游/成品价格传导
### 5.3 涨价红利分配（谁能吃到最多）
### 5.4 紧缺预测（何时缓解）
### 5.5 情景分析（乐观/中性/悲观）

## 第六节：标的横向对比与投资框架
### 6.1 A 股标的全景对比表（≥15 维度）
### 6.2 按产业链位置分类（梯队）
### 6.3 投资框架（周期位置 × 产业链位置 → 策略）

## 方法论框架整合小结
```

## 模板选择决策树

```
用户提到一只具体股票？
  ├─ 是 → 要深度还是快速？
  │       ├─ 深度 → 模板 A（个股 11 章）
  │       └─ 快速 → 模板 D（分析笔记）
  └─ 否 → 提到行业/材料/技术？
          ├─ 是 → 景气导向？
          │       ├─ 是 → 模板 B1（产业链 6 节）
          │       └─ 否 → 模板 B2（产业链 8 章）
          └─ 提到投资方法/框架？
                  └─ 是 → 模板 C（方法论 8 章）
```

不确定时，问用户："你想要的是深度研报、产业链全景、方法论梳理、还是快速分析？"
