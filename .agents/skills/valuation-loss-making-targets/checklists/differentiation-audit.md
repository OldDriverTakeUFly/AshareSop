# Differentiation Audit Checklist

当撰写新的亏损标的估值报告时，在报告撰写完成后、终审前使用此清单。逐轴确认新报告与已有报告存在实质性差异，避免内容重复。

## 量化困境评分

- [ ] 新报告包含 davis_analyzer 三层信号困境评分（财务确认层 / 反转可行性层 / 拐点激活层），输出综合反转概率评分（0-100），替代已有报告的定性景气度打分
- [ ] 已有报告缺乏可复现的量化反转模型，新报告三层信号（含权重与阈值）构成方法论增量

## WACC×永续敏感性矩阵

- [ ] 新报告构建 WACC×永续增长率 5×5 敏感性矩阵（WACC 5 档、永续 5 档），输出 25 个估值结果
- [ ] 已有报告 DCF 仅 WACC 单点估值，新报告标注单点在矩阵中的位置，展示估值边界与重心

## 扩展财务深度

- [ ] 新报告对以下 8 项指标做深度拆解（多季度趋势 + 同业横向对比）：毛利率、研发费率、费用率、合同负债、在建工程、存货周转率、现金消耗率、资本开支强度
- [ ] 已有报告财务分析停留在基础营收/净利/毛利率，新报告补齐领先指标与季度趋势

## 筹码结构分析

- [ ] 新报告纳入筹码结构分析，覆盖：股东人数趋势、前十大股东、流通结构、限售解禁日历、北向资金动向、大股东增减持
- [ ] 已有报告完全无交易面/筹码面分析，新报告补齐"基本面+交易面"双轮视角

## 全量国内外同业

- [ ] 新报告覆盖不少于 3 家国内外可比标的（建议覆盖 6 家以上以提升可比性：海外上市巨头 + 国内上市同业 + 未上市龙头），多维雷达图对比
- [ ] 已有报告同业对比范围更窄或相关性弱，新报告扩充同业覆盖面并提升可比性

## 结论判定

- [ ] 5 轴全部通过 → 差异化充分，可进入终审
- [ ] 任一轴不通过 → 返回报告撰写阶段补充该轴增量内容
