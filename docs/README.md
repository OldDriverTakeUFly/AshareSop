## 🔧 维护说明

### 研报版本管理规范

1. **日期标注**：每篇研报正文头部标注"研究日期：YYYY-MM-DD"
2. **版本历史**：每篇研报末尾（免责声明前）维护版本历史表：
   ```
   ## 版本历史
   | 版本 | 日期 | 变更说明 (delta) |
   |------|------|-----------------|
   | v1.0 | YYYY-MM-DD | 初版创建... |
   | v1.1 | YYYY-MM-DD | 新增... |
   ```
3. **同标的多份研报**：优先整合为一份主报告 + 附录存档，删除独立文件（参考阳光电源 v1.2 整合方式）
4. **文件命名**：使用中文文件名，日期只在正文（文件名不加日期前缀，保持链接稳定）

### 新增研报时

1. 放入对应的行业子文件夹
2. 正文头部加"研究日期"
3. 末尾加"版本历史"表
4. 更新本 README.md 索引
