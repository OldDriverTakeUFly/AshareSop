## 1. SST技术概述

### 1.1 定义与核心原理

固态变压器（Solid-State Transformer，SST），又称电力电子变压器（Power Electronic Transformer，PET）或智能变压器，是一种基于先进电力电子技术（如SiC、GaN宽禁带半导体）和高频变压器技术的新型电能转换设备。其核心是通过半导体器件替代传统变压器的铁芯和铜绕组结构，实现电压变换、电气隔离及电能质量控制等功能。

SST的基本架构由三级组成：
- **输入级（AC/DC整流）**：将工频交流电转换为直流电
- **隔离级（DC/DC高频变换）**：通过高频变压器实现电压变换和电气隔离
- **输出级（DC/AC逆变）**：将直流电转换为所需频率和电压的交流电

### 1.2 SST vs 传统变压器对比

| 特性 | 传统变压器（LFT） | 固态变压器（SST） |
|------|-------------------|-------------------|
| 核心技术 | 铁芯+铜绕组 | 功率半导体（SiC/GaN）+高频变压器 |
| 工作频率 | 50/60 Hz | 1 kHz–200 kHz |
| 体积/重量 | 大且重 | 缩小50-80% |
| AC/DC转换 | 需外部整流器 | 集成AC/DC能力 |
| 双向功率流 | 有限 | 完全双向支持 |
| 电网智能 | 被动 | 实时控制、故障隔离 |
| 效率 | 97-98%（AC/AC） | 97-99%（含AC/DC功能时综合效率更高） |
| 相对成本 | 基准 | 当前为传统变压器的2-10倍 |
| 寿命 | 30-40年 | 待验证（电力电子寿命通常15-25年） |

**关键洞察**：SST不是简单的"变压器替代品"，而是一个集成了变压器、整流器、逆变器、无功补偿、电能质量治理等多功能于一体的智能电力枢纽。在AC/DC应用场景中（如数据中心、EV充电），SST的系统级效率优势更为显著——ETH Zurich研究表明，在AC/DC应用中SST损耗仅为传统方案的一半。

### 1.3 发展历程

- **1970年**：William McMurray在专利中首次提出具有高频链接的变换电路（US Patent 3,517,300）
- **2009年**：GE Vernova采用10kV SiC器件完成13.8kV/265V SST首次示范验证
- **2011-2012年**：ABB在瑞士联邦铁路Ee 933调车机车上部署1.2MW PETT（电力电子牵引变压器），这是最常被引用的西方商业SST部署案例
- **2020年**：日本新干线N700S搭载SiC牵引变流器投入运营（功能上等同于SST）
- **2021年**：中国国家电网在江苏吴江投运±10kV/750V/2MW SST示范工程
- **2024-2025年**：全球多家企业集中发布SST产品，DG Matrix推出首款商用多端口SST
- **2026年**：四方股份数智SST 1.0实现量产（2.4MW），Heron Power获1.4亿美元B轮融资建设40GW年产能工厂，Enphase发布基于GaN BDS的分布式SST平台

## 3. 上游：宽禁带半导体材料与器件

### 3.1 SiC衬底——SST的成本核心

SiC衬底是SST产业链中**成本占比最高、技术壁垒最深**的环节。在SiC器件制造成本结构中，衬底成本占比约47%，外延成本约23%。

### 3.3 10kV SiC——SST的未来制高点

**10kV SiC MOSFET是SST技术路线的关键变量**。当前SST多采用1700V或2100V SiC MOSFET模块级联方式来适配13.8kV母线电压，流程繁琐且效率有损。10kV SiC MOSFET可以大幅减少器件级联数量、提升系统效率、降低综合成本。

**布局企业**：
- **纳微半导体**：2019年推出15kV SiC MOSFET，已应用于宇航设备和智能电网示范
- **瞻芯电子**：2025年6月与浙江大学联合发表10kV等级SiC MOSFET芯片，击穿电压超12kV
- **杰平方半导体**：判断2030年前后万伏级SiC器件将实现万伏千安级工程化落地
- **德氪微电子**：10kV SiC技术布局中
- **天岳先进**：液相法P型衬底为10kV SiC提供材料基础

**技术瓶颈**：
1. 厚外延缺陷控制——超厚外延层的缺陷密度是良率的关键
2. 栅氧可靠性——高压下栅氧层寿命是核心问题
3. 大尺寸芯片一致性——10mm×10mm以上芯片良率挑战
4. P型衬底需求——6500V以上器件可能需要P型衬底（天岳先进液相法技术领先）

### 3.4 GaN——SST的替代技术路线

虽然SiC是SST主流技术路线，但GaN正在成为有力竞争者：

- **Enphase Energy**（2026年4月发布）：采用GaN双向开关（BDS）技术推出IQ SST平台，每个1.25MW SST机架由342个智能4kW功率模块组成，2026年底演示、2028年批量出货
- **GaN优势**：更高的开关频率、更低的EMI（零电压/零电流软开关）、塑料封装（SiC通常需陶瓷基板）
- **GaN挑战**：目前电压等级限于650V/900V，中压SST仍需SiC

## 4. 上游：高频磁性材料

### 4.1 纳米晶/非晶软磁合金——SST高频变压器的核心

SST中的高频变压器需要工作在1kHz-200kHz范围，传统硅钢片在此频率下损耗极高，必须采用纳米晶或非晶软磁合金。

#### 纳米晶 vs 铁氧体对比

| 特性 | 纳米晶（FINEMET/VITROPERM） | 铁氧体（MnZn） |
|------|---------------------------|----------------|
| 饱和磁感应强度 Bs | ≥1.2 T | ~0.4 T |
| 最大磁导率 | 500-100,000 | ~15,000 |
| 铁损 W/kg (100kHz, 0.2T) | 20-35 | 120 |
| 居里温度 | 550-570°C | 200-300°C |
| 最高工作温度 | 150°C | 100°C |
| 矫顽力 | 0.5-5 A/m | 较高 |

**关键优势**：纳米晶材料饱和磁感应强度是铁氧体的3倍，铁损仅为铁氧体的1/4-1/6，是SST高频变压器的理想铁芯材料。

#### 全球纳米晶/非晶材料供应商

| 企业 | 国家 | 产品/地位 | SST相关动态 |
|------|------|-----------|-------------|
| **日立金属（Proterial，原Hitachi Metals）** | 日本 | FINEMET®纳米晶+Metglas®非晶全球龙头 | F3BC纳米晶块芯可处理100kW以上功率；5-20kHz开关频率最优 |
| **VAC（Vacuumschmelze）** | 德国 | VITROPERM®纳米晶欧洲龙头 | 专为SST和MF变压器提供纳米晶铁芯解决方案；1kHz-200kHz工作范围 |
| **安泰科技（AT&M）** | 中国 | 中国纳米晶/非晶带材龙头之一 | 市占约47%；技术比肩日立金属；已通过英伟达、华为认证；非晶带材产能5万吨/年 |
| **云路股份** | 中国 | 纳米晶磁芯龙头 | 非晶带材产能11万吨/年（全球最大）；纳米晶在200kHz高频损耗比铁氧体低40%；已通过金盘科技认证；即将推出SST专用非晶纳米晶带材 |

**投资视角**：安泰科技和云路股份是SST高频磁性材料的直接受益标的。安泰科技估值弹性更大（市值较小），云路股份产能规模领先且已在SST客户认证中取得进展。

### 4.2 高频变压器制造

### 7.2 AI数据中心——SST的杀手级应用

**为什么AI数据中心是最先落地的场景？**

1. **英伟达800V HVDC架构推动**：英伟达Blackwell/Rubin架构功耗飙升（单机架>100kW），传统48V配电已不可行，800V直流配电成为必然选择。SST可直接将电网中压（13.8-34.5kV）转换为800V DC，省去传统方案中的工频变压器+低压开关+UPS+配电柜等多级设备。

2. **空间和效率优势**：
   - 四方股份SST 1.0：占地面积减少90%，效率提升6.5%
   - 台达SST：占地面积减少50%，1㎡承载MW级
   - 维谛/英伟达合作方案：省去灰色空间房间（UPS+配电间）

3. **客户付费意愿强**：AI数据中心运营商（Google、Microsoft、AWS、Meta、阿里云、腾讯云等）对电力转换效率和空间密度的需求极为迫切，成本敏感度低于电网客户。

4. **传统变压器交期极长**：大型电力变压器平均交期128周（2.5年），部分配电变压器价格自2019年以来上涨95%。SST可以更快部署。

**SST在AI数据中心的架构变革**：
```
传统方案（5级变换）：
电网13.8kV → 工频变压器降至480V → 低压开关 → UPS → PDU → 服务器电源 → 主板

SST方案（1级变换）：
电网13.8kV → SST直接转换为800V DC → 服务器机架
```

## 9. 技术瓶颈深度分析

### 9.1 成本问题——最大商业化障碍

**当前状态**：
- SST成本为传统变压器的**2-10倍**
- 50kVA SST：$60,000-$100,000 vs 传统变压器$6,000-$10,000
- 5MW MV-SST的SiC模块单项成本：$10,000-$50,000/单元

**成本下降路径**：
| 驱动因素 | 预期效果 |
|----------|----------|
| SiC器件成本下降 | 过去5年已降50%+；8英寸产线将再降30%+ |
| 规模效应 | Heron Power 40GW/年产能将带来数量级降本 |
| 模块化设计 | 减少定制化，提高生产效率 |
| 拓扑简化 | 从三级拓扑→两级→单级，减少功率器件数量 |

**历史类比**：
- 光伏逆变器：2010年$0.80/W → 2025年$0.07/W（15年降91%）
- 锂电池：2010年$1,100/kWh → 2025年<$140/kWh（15年降87%）
- SST行业预测：2028-2032年SST成本/kVA可与铁芯配电变压器持平

**关键拐点**：2026年SiC模块成本曲线与SST系统收益曲线发生"金叉"。虽然单模块成本仍高于IGBT，但考虑SST系统层面减少的铜材、硅钢片、冷却系统及占地面积，**SST的综合TCO（全生命周期成本）将在2026年首次具备对传统变压器的竞争力**。

### 9.2 效率问题

**当前效率水平**：
- 四方股份SST 1.0：端到端效率98.5%（10kV AC→800V DC）
- 金盘科技SST：转换效率98%+
- 台达SST：效率98.5%+
- 道通科技SST：99.0%+
- EPFL（使用纳微SiC）：250kW SST效率>97%
- CSA Catapult：3/2电平DAB SST，15kW输出效率95%（目标100kW）
- 理论最优（ETH Zurich研究）：在AC/DC应用中，SST损耗仅为传统方案的一半

**效率分析**：当前SST效率已达到实用门槛（>97%），且在AC/DC应用场景中（如数据中心800V架构），SST的**系统级效率**显著优于传统多级方案。这不是瓶颈。

### 9.3 容量限制

**当前最大容量**：
- 单机：2.4-5MW（四方、金盘、Heron Power）
- 模块化组合：10MW+（GE Vernova的MMC方案、伊戈尔规划35kV/10MVA）
- Heron Power 2027年目标：40GW年产能

**配电站级需求**：典型配电站容量在10-100MVA范围。当前SST单机容量已达到MW级，通过模块化并联可扩展到10MW+，已覆盖大部分配电站需求。

### 9.4 可靠性/寿命问题

**这是被低估的最大风险**：
- 传统变压器寿命30-40年，维护极少
- SST包含数千个电力电子器件，任一故障可能影响系统
- **缺乏长期运行数据**：全球商用SST累计运行时间极少，25年寿命目标尚无法验证
- SST支持模块化热插拔（如Heron Link故障模块10分钟可更换），但在10kV中压场景下热插拔实现难度大
- 日本新干线SiC牵引变流器的运行数据（可能最丰富）在三菱电机和日立铁路内部，不公开

### 9.5 散热问题

SST功率密度是传统变压器的50-100倍，散热是核心挑战：
- 液冷是MW级SST的标配（成本占比8-12%）
- 高频变压器液冷设计复杂
- SiC器件结温可达175°C，有利于散热设计
- 基本半导体ED3模块采用Si3N4 AMB载板，散热性能优于传统Al2O3 DBC

### 9.6 标准化问题

**缺乏统一标准是SST大规模部署的制度性障碍**：
- 数据中心领域缺乏国家标准指引
- SST的测试、认证、并网标准尚未统一
- 中国企业（如中国电科院）正在主导国际SST标准制定
- 国家电网和南方电网通过"并网试运行"机制为SST提供准入路径

### 9.7 电磁兼容（EMC）

高频开关（50-200kHz）带来的EMI问题：
- 纳米晶铁芯有助于降低EMI（低磁致伸缩）
- Enphase的GaN BDS方案采用谐振拓扑实现ZVS/ZCS，EMI极低
- SST内部需要精心设计的EMC滤波器

## 10. 依赖性分析与地缘政治影响

### 10.1 SST对SiC器件的依赖

**SST与SiC的关系是不可分割的**：
- 当前所有已发布和已宣布的商用SST产品都使用SiC MOSFET作为中压开关器件
- 没有SiC（或等效宽禁带半导体），SST的高频、高效率、高功率密度优势将大幅缩水
- GaN是替代技术路线（Enphase），但目前限于低压（<900V），中压SST仍需SiC
- Wolfspeed的200mm SiC产能是"整个SST制造扩建的最大外部约束"

### 10.2 国产SiC替代能力

| 环节 | 国产化率 | 主要企业 | 能否满足SST需求 |
|------|----------|----------|----------------|
| SiC衬底 | ~30%（国内市场份额） | 天岳先进、天科合达 | 6英寸基本满足，8英寸加速追赶 |
| SiC外延 | ~25% | 瀚天天成、天域半导体 | 基本满足 |
| SiC MOSFET器件 | ~15% | 斯达半导、士兰微、基本半导体 | 中低压可替代，高压（>3.3kV）仍有差距 |
| 10kV SiC MOSFET | 极低 | 瞻芯电子、杰平方半导体 | 实验室阶段，2030年工程化 |
| SiC模块 | ~20% | 斯达半导、基本半导体 | 基本可替代 |

**关键判断**：
- 中低压SST（<1700V器件）：国产SiC基本可替代，不受地缘政治影响
- 中压SST（3.3kV-10kV器件）：当前依赖进口（英飞凌、Wolfspeed、ST），短期有"卡脖子"风险
- 但中国SST企业（四方股份、金盘科技等）目前主要使用1700V SiC MOSFET级联方案，**在10kV SiC国产化之前，可通过级联方案规避进口依赖**

### 10.3 Wolfspeed的系统性风险

Wolfspeed的处境直接影响全球SST供应链：
- 2024年9月申请Chapter 11破产保护
- 重组后资本开支承诺降低
- Wolfspeed不只是一个器件供应商——它是西方SiC衬底的主要来源
- 英飞凌不从自己的晶体生长SiC晶锭，依赖Wolfspeed/SiCrystal供应
- 如果Wolfspeed产能爬坡停滞，整个西方SST制造计划都将受限

**对中国的影响**：Wolfspeed的困境反而可能加速中国SiC供应链的自主化进程。天岳先进等企业正在填补产能缺口。

### 10.4 地缘政治对SST供应链的影响

| 风险因素 | 影响程度 | 影响环节 |
|----------|----------|----------|
| 美国对华半导体出口管制 | 中等 | 高压SiC MOSFET（>3.3kV） |
| 2025年美国关税框架调整 | 中等 | SST整机进出口 |
| Wolfspeed产能不确定性 | 高 | 全球SiC衬底供应 |
| 电网基础设施国家安全审查 | 中低 | SST并网审批 |
| 关键矿物（镓、锗）出口管制 | 低 | GaN器件（反向影响西方） |

**总结**：地缘政治对SST供应链的短期影响有限（因为中国SST企业主要使用国产或非受限渠道获取SiC器件），但长期来看，10kV级高压SiC器件的国产化是确保供应链安全的关键。

## 13. 结论与投资建议

### 13.1 核心结论

1. **SST正处于从技术验证到早期商用的关键拐点**。2026年是中国SST产业爆发元年，四方股份SST 1.0实现量产标志着行业从实验室走向工程化。

2. **AI数据中心是SST的杀手级应用**。英伟达800V HVDC架构的推动下，SST在数据中心的渗透将是最快的。2027年有望看到规模化部署。

3. **成本仍是最大障碍，但下降路径清晰**。SST当前成本是传统变压器的2-4倍，但SiC器件成本过去5年已降50%+，行业预测2028-2032年SST成本/kVA可与传统变压器持平。

4. **中国企业在SST产业链中的参与度正在快速提升**。从SiC衬底（天岳先进、天科合达）到磁性材料（安泰科技、云路股份）到整机（四方股份、金盘科技、中国西电），中国已形成基本完整的SST产业链。

5. **客观面对商业化进度的不确定性**。全球商用SST实际部署量仍在个位数，2026-2028年是验证窗口期。投资者应关注具体订单和交付数据，而非仅看产品发布。

### 13.2 风险提示

- **技术风险**：SST长期运行可靠性缺乏数据支撑
- **成本风险**：SiC产能过剩可能导致上游企业盈利恶化
- **政策风险**：800V数据中心标准尚未统一
- **竞争风险**：传统变压器企业也在升级（高效非晶变压器）
- **估值风险**：部分SST概念股已透支未来2-3年预期
- **执行风险**：Heron Power等初创企业能否兑现40GW产能目标存疑
