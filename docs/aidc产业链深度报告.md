> 研究范围：AI智算中心（AIDC）全产业链梳理、投资确定性排序、核心环节标的分析
> 报告日期：2026年5月

## 一、用户认知纠偏与补充

### 1.1 用户原有认知框架

用户对AIDC产业链的初始认知覆盖了以下环节：

- AIDC电源
- 算力中心建设
- 算力芯片供应商
- 算力提供商
- 液冷散热
- 电力供应

这个框架抓住了产业链的主干，但存在**多处认知偏差**和**大面积遗漏**。下面逐一纠偏。

### 1.2 核心纠偏

**纠偏一："电源"远不止UPS**

多数投资者提到AIDC电源，脑子里浮现的就是UPS（不间断电源）。实际上，AIDC供电链路至少包含**7个子环节**：

| 供电链路环节 | 功能定位 | 代表公司 |
|:---|:---|:---|
| 变压器（高压/配电） | 电网接入、电压变换 | 特变电工、许继电气、金盘科技 |
| UPS/HVDC/SST | 不间断供电、直流配电、固态变压器 | 科华数据、科士达、禾望电气 |
| 服务器PSU（电源模块） | 服务器内部电源，功率密度持续提升 | 麦格米特、欧陆通、台达电子 |
| BBU（后备电池组） | 机柜级备电，替代传统UPS架构 | 亿纬锂能、宁德时代、南都电源 |
| 超级电容 | 瞬态功率补偿，应对AI训练尖峰功耗 | 江海股份、法拉电子 |
| 芯片电感（合金软磁） | GPU供电滤波，单颗GPU需20-30颗 | 铂科新材、东睦股份、天通股份 |
| 配电柜/PDU | 机柜级精密配电 | 科华数据、施耐德（合资） |

UPS只是中间一环，上游的变压器、配电设备，下游的服务器PSU、芯片级被动元件，每一个都是独立的投资逻辑。

**纠偏二："液冷"上游还有一整条供应链**

液冷散热不仅仅是冷板和管路。它的上游还包括：

- **压缩机**：汉钟精机（涡旋压缩机龙头，液冷系统核心部件）
- **泵**：大元泵业（液冷循环泵，国产替代进行中）
- **冷却液**：巨化股份（氟化液，3M退出后的国产替代窗口）、新宙邦
- **冷板/快速接头**：高澜股份、飞荣达
- **管路/连接器**：川环科技、中航光电

如果只看到液冷集成商，会错过上游这些增速更高、壁垒更厚的标的。

**纠偏三：最大遗漏，光模块/光通信**

这是A股AI算力投资中**确定性最高的赛道**，却没有出现在用户的认知框架里。

为什么光模块如此重要？AIDC内部GPU集群的互联依赖高速光通信。一个万卡集群需要数万只800G/1.6T光模块，单只价格数千元，这是纯粹的量价齐升逻辑。中际旭创、新易盛、天孚通信等公司已经连续多个季度业绩超预期，验证了这一判断。

**纠偏四：PCB/覆铜板是被低估的环节**

AI服务器用PCB层数从普通服务器的12-16层提升到**24-30层**，单价提升3-5倍。覆铜板（CCL）作为PCB的上游原材料，同样受益于高频高速材料的需求升级。沪电股份、深南电路在高端PCB领域有明确产能瓶颈，景气度远未被市场充分定价。

**纠偏五：HBM存储是独立且重要的赛道**

HBM（高带宽内存）不是普通DRAM，它是绑定在GPU封装内的存储芯片，技术壁垒极高，由SK海力士、三星、美光三家垄断。A股相关标的主要是封装测试环节（通富微电）和接口IP（芯原股份），以及国内HBM攻关的潜在受益者（兆易创新、北京君正）。

### 1.3 完整遗漏清单

除上述纠偏外，用户框架还缺失以下重要环节：

| 遗漏环节 | 重要程度 | 说明 |
|:---|:---|:---|
| 光模块/光通信 | 极高 | A股AI算力最确定赛道，800G→1.6T迭代 |
| PCB/覆铜板 | 高 | 高端供需紧张，层数/单价齐升 |
| HBM存储 | 高 | 独立超级周期，国产替代紧迫 |
| 交换机/网络设备 | 高 | InfiniBand/以太网交换机，AIDC组网核心 |
| 光纤光缆 | 中高 | AIDC内部互联+城域/长途光网络扩容 |
| 铜连接（高速铜缆） | 中高 | 短距互联的性价比方案，安费诺替代 |
| 柴油发电机组 | 中高 | AIDC备电刚需，供需缺口明显 |
| 燃气轮机 | 中 | 大型AIDC的供配电方案升级方向 |
| 变压器（高压配电） | 中 | AIDC建设直接拉动，配电容量需求翻倍 |
| 连接器 | 中 | 高速背板连接器、光连接器国产替代 |
| 被动元件（电容/电感） | 中 | GPU供电侧用量暴增 |
| BBU（后备电池） | 中 | 机柜级锂电池备电新趋势 |
| 超级电容 | 中低 | AI训练瞬态功耗补偿 |
| 芯片电感/合金软磁 | 中低 | GPU供电必需，单机价值量提升 |
| 管理软件/调度平台 | 中低 | 算力调度、集群管理、运维 |

### 2.2 上游设施层：基建类

#### 2.2.1 土建工程

AIDC的土建要求远高于传统数据中心。单栋智算楼承重需达到**12-15kN/m²**（普通住宅约2kN/m²），楼层层高需6-8米以容纳液冷管路和高压配电。大型AIDC园区用地规模达数百亩。

| 细分方向 | 代表公司（A股） |
|:---|:---|
| 数据中心建设/总包 | 数据港、润建股份、中国电信 |
| 钢结构/机房模块化 | 精工钢构、东南网架 |
| 消防/安防系统 | 青鸟消防、辰安科技 |

#### 2.2.2 供配电系统

AIDC的供配电系统是投资链条最长、子环节最多的基建环节。从电网接入到芯片供电，经历**4-5级变换**：

|:---|:---|:---|
| 高压变压器 | 110kV/10kV降压 | 特变电工、金盘科技、许继电气 |
| 中压开关柜 | 配电与保护 | 国电南瑞、许继电气 |

## 三、按投资确定性排名

|:---:|:---|:---:|:---|:---:|
| 1 | 光模块/光通信 | ★★★★★ | 800G放量、1.6T迭代，量价齐升，出口份额持续提升 | 80%+ |
| 2 | AIDC运营商 | ★★★★★ | 核心稀缺资产，大客户长单锁定，租金/上架率双升 | 30-40% |
| 3 | AI服务器整机 | ★★★★☆ | 算力直接载体，订单爆发式增长，合同负债大幅攀升 | 50%+ |

### 3.1 光模块/光通信：确定性 No.1

**核心逻辑**：AIDC内部GPU集群互联需要大量高速光模块。以万卡集群为例，GPU间全互联需约2-3万只800G光模块，单只价格约5000-7000元，对应10-20亿元光模块采购额。从800G向1.6T迭代过程中，量价齐升逻辑清晰。

### 3.3 AI服务器整机：确定性 No.3

**核心逻辑**：AI服务器是算力的直接物理载体，需求爆发式增长。头部厂商合同负债大幅增长反映供不应求格局。但需注意，整机环节毛利率普遍偏低（3-5%），利润弹性弱于收入弹性。

### 3.4 液冷散热：确定性 No.4

**核心逻辑**：当单机柜功率超过15kW，风冷已无法胜任。AI服务器机柜普遍40-120kW，液冷渗透率正从2024年的约15%向2027年的70%+跃迁。这是典型的渗透率加速拐点，相关公司业绩增速有望持续超预期。

2026年全球800G和1.6T光模块需求出现叠加爆发。800G尚未退坡，1.6T已开始规模出货，这种"双代际共振"在光模块历史上罕见，也是当前标的高增速的底层支撑。

### 6.1 中际旭创(300308) — 全球光模块双寡头之一

中际旭创和Coherent（原II-VI）构成全球高速光模块双寡头格局。公司2025年全年营收382.40亿元，同比增长60%；归母净利润107.97亿元，同比增长109%。2026年一季度进一步加速: 营收194.96亿元（同比+192%），归母净利润57.35亿元（同比+262%）。

**风险提示**: 客户集中度过高，若北美大客户资本开支放缓将直接冲击业绩；1.6T价格下行压力可能压缩毛利率。

### 6.3 天孚通信(300394) — 光器件上游龙头

**竞争壁垒:**

- **无源器件壁垒极高**: 精密光学加工、薄膜镀膜、微纳结构制造等技术门槛极高，全球能做高品质无源器件的厂商屈指可数
- **光模块上游核心供应商**: 中际旭创、新易盛等光模块龙头均是天孚通信的核心客户
- **客户粘性强**: 无源器件一旦通过认证进入BOM，替换需要重新进行可靠性验证，周期长、成本高

天孚通信的价值在于，它卖的是"铲子"而非"金子"。无论光模块价格如何波动，无论哪家光模块厂商胜出，只要高速光模块的需求在增长，天孚通信作为上游核心供应商就能持续受益。

## 八、供电系统（Top 3） — 算力尽头是电力

"算力尽头是电力"不是一句口号，而是AIDC产业链的真实写照。单个GPU功耗突破1000W，单机柜功率突破100kW，一个万卡集群的功耗已经相当于一个中型工业园区。供电系统不再只是基础设施的配角，而是决定算力上限的核心约束。

**AIDC供电链路:**

```
10kV市电 → 变压器 → 开关柜 → UPS/HVDC → PDU → 服务器PSU → GPU
```

这条链路上，每一个环节都在经历功率升级和架构变革。UPS从传统交流方案向HVDC（高压直流）演进，PSU从3kW向5kW+升级，配电密度持续提升。供电系统的投资占AIDC建设总成本的15-20%，且在功率密度提升趋势下占比还在上升。

### 15.2 伊戈尔(002922) — 数据中心变压器

**核心逻辑**：伊戈尔专注于工业控制与电源领域磁性元器件，数据中心变压器是核心增长方向之一。公司产品覆盖移相变压器、整流变压器等数据中心配电应用场景。

伊戈尔的竞争优势在于成本控制能力与客户响应速度。在AIDC建设加速的背景下，变压器产能成为制约因素之一，伊戈尔凭借柔性制造能力可实现快速扩产交付。

## 十六、连接器/被动元件（Top 3）

> AIDC内部互连从铜缆到光通信再到铜光电复合（CPC），技术路线快速迭代。连接器和被动元件虽"小"，却是决定信号完整性和电源质量的关键节点。

### 16.1 立讯精密(002475) — 224G铜连接+液冷+电源全布局

**核心逻辑**：立讯精密是AIDC互连领域的全布局标的，产品线覆盖224G高速铜连接、液冷快插接头、电源连接器三大方向，同时深度参与GPU模组SXM/PCIe代工。

224G铜连接是当前AIDC机柜内互连的主流方案，立讯作为NVIDIA核心供应商深度受益。CPC（铜光电复合）是下一代互连方案，将铜线的供电能力与光纤的传输带宽结合，立讯在该方向已启动预研布局。

立讯的独特优势在于"一站式"互连解决方案能力，客户不需要分别对接铜连接、液冷、电源等多个供应商，这大幅简化了AIDC的系统集成复杂度。

### 16.2 中航光电(002179) — 液冷连接器龙头

**核心逻辑**：中航光电依托军工技术积累，将航空领域的高可靠性连接器技术转化为数据中心液冷连接器产品。液冷系统的连接器需要在高压、高温冷却液环境下长期稳定运行，对密封性和耐腐蚀性要求极高，这恰好是中航光电的传统优势领域。

公司在液冷盲插连接器领域占据国内领先地位，已进入多家头部服务器厂商供应链。

### 16.3 铂科新材(300811) — 芯片电感

**核心逻辑**：铂科新材是国内芯片电感的龙头供应商，芯片电感是AIDC电源模块的核心被动元件。GPU供电对电感的体积、损耗、饱和电流均提出了严苛要求，铂科的合金软磁粉末加压制成型技术路线在高频大电流场景下具有明显优势。

AIDC电源模块中，每颗GPU需要搭配多颗芯片电感，GPU功耗从300W向1000W演进的过程中，电感的单机价值量同步提升。铂科新材作为国内稀缺的芯片电感能量产企业，直接受益于AIDC放量。

### 17.3 关键风险提示

**1. AI资本开支周期反转风险**

全球AI资本开支高度集中在少数几家公司（微软、Google、Meta、Amazon合计占全球AI Capex的60%以上）。一旦头部厂商因ROI不达预期而放缓投入，整条产业链将面临需求端断崖式下行。历史上，电信行业的资本开支周期就曾导致光通信产业链长达数年的低迷。

**2. 客户集中度风险**

部分标的客户集中度极高。以寒武纪为例，前五大客户营收占比高达88.66%，单一客户变动就可能对业绩造成重大冲击。这一风险在国产AI芯片标的中普遍存在。

**3. GPU四小龙收敛风险**

海光信息、寒武纪、景嘉微、龙芯中科被称为"国产GPU四小龙"，但最终可能只有1至2家能真正跑出来。投资者在分散配置的同时需警惕最终"赢家通吃"格局下的估值重估风险。

**4. HBM瓶颈**

AI训练对高带宽内存（HBM）的需求持续攀升，但国内尚无HBM量产能力。这一瓶颈可能在特定时期限制国产AI芯片的实际性能发挥，间接影响相关标的的成长天花板。

**5. 毛利率压力**

服务器行业毛利率长期承压。以浪潮信息为例，综合毛利率仅4.88%，几乎属于"搬砖"生意。在AI服务器放量过程中，若无法通过高附加值产品提升毛利率，量增可能无法转化为利润增长。

**6. 地缘政治升级风险**

美国对华芯片出口管制持续加码，从GPU到EDA工具再到HBM，管制范围不断扩大。国产替代标的短期受益于"自主可控"逻辑，但若管制进一步升级至设备、材料层面，可能影响产业链上游的正常运转。

**7. 液冷竞争加剧**

液冷赛道参与者数量快速增加，冷板领域已出现价格竞争迹象。未来1至2年可能出现行业洗牌，部分缺乏核心竞争力的参与者将被淘汰，但短期内价格战可能压缩行业整体利润率。
