# 关键标的深度研究：中际旭创（硅光）& 天孚通信（隔离器）

> 本报告为光通信产业链投资研究系列报告之一，聚焦中际旭创硅光技术进展与天孚通信光隔离器能力，所有数据均来自公开披露信息与行业调研。

---

## 一、中际旭创 (InnoLight, 300308.SZ) — 硅光技术深度研究

### 1.1 硅光模块产品代际演进

中际旭创在硅光技术路线上已实现从400G到1.6T的全面量产覆盖，3.2T处于研发阶段。

| 代际 | 硅光状态 | 封装形态 | 关键信息 |
|---|---|---|---|
| 400G | 2023年起已量产 | OSFP / QSFP-DD | 基于Tower PH18M平台 |
| 800G | 已量产，硅光占比快速提升 | OSFP (DR8, 2xFR4, DR8+) | 截至2026年初硅光占比超过50% |
| 1.6T | 2025年Q2起量产 | OSFP-XD / OSFP224 | 以硅光方案为主 |
| 3.2T | 研发阶段 | 待定 | OFC 2025展示单波400G演示 |

数据来源：公司公告、OFC 2025展会信息。

### 1.2 硅光芯片供应链

中际旭创的硅光芯片采用**自研设计 + Tower代工**的模式。

**芯片设计：** 自研。2025年9月公司明确表示"公司自研的硅光芯片已批量应用和大规模出货"。

**代工合作方：** Tower Semiconductor (TSEM)，独家合作。

- 2023年9月：初始合作，采用Tower PH18M平台量产400G/800G硅光模块
- 2025年3月：扩大合作范围，下一代硅光平台将每模块所需激光器数量减半
- Tower晶圆厂位于以色列(Migdal HaEmek)和美国(Malta, NY)，采用200mm硅晶圆

**未采用其他代工厂：** 硅光芯片未使用TSMC、中芯国际或GlobalFoundries。

**DSP来源：** Broadcom (Sian系列) 和 Marvell (Spica/Ara系列)。

### 1.3 产量、良率与硅光占比

中际旭创硅光模块出货占比在过去两年内经历了快速攀升：

| 时间段 | 硅光占比（约） | 毛利率 |
|---|---|---|
| 2024年上半年 | 低个位数百分比 | 33.8% |
| 2025年Q2 | 快速攀升 | 41.5% |
| 2025年Q3 | 占800G出货主力 | 42.8% |
| 2026年Q1 | 超过50%（800G与1.6T合计） | 约46% |

**关键数据点：**

- 行业调研显示800G硅光模块良率约95%
- 2025年全年光模块出货量：2109万只
- 截至2025年底年化产能：约2800万只

### 1.4 1.6T量产时间线

1.6T是中际旭创当前最重要的增长驱动力之一，从演示到大规模部署的时间线如下：

| 时间 | 里程碑 |
|---|---|
| 2023年3月 | OFC 2023现场演示1.6T OSFP-XD DR8+ |
| 2024年下半年 | 客户测试完成 |
| 2024年12月15日 | 首批小批量发货 |
| 2025年Q2 | 以硅光方案为主的量产启动 |
| 2025年Q3 | 核心超大规模客户开始部署 |
| 2026年Q1 | 营收195亿元（同比+192%） |

### 1.5 核心客户结构

中际旭创的客户群体以海外超大规模数据中心为主：

- **Google** — 核心超大规模客户
- **Nvidia** — 中际旭创为其主要光模块供应商
- **Amazon (AWS)**
- **Microsoft**
- **Meta**
- 国内云计算厂商

2024年海外收入占总收入比例为**86.8%**。

### 1.6 硅光 vs EML/InP 技术对比

硅光方案相对于传统EML（电吸收调制激光器）和InP（磷化铟）方案具有系统性成本优势：

| 维度 | 硅光方案 | EML/InP方案 |
|---|---|---|
| BOM成本 | 显著更低 | 较高 |
| 激光器需求 | 仅需CW激光器（更简单、供应商更多） | 需要复杂调制激光器 |
| 激光器数量 | Tower新平台可减少50% | 每通道独立激光器 |
| 毛利率驱动 | 从33.8%(2024)提升至46%(2026 Q1) | 受限于物料成本 |

硅光是公司明确认定的未来发展方向，在成本结构和可扩展性上持续优于EML/InP方案。

### 1.7 竞争优势总结

1. **硅光自研能力：** 掌握核心芯片设计，不依赖外部芯片供应商
2. **代工深度绑定：** 与Tower独家合作，新平台持续优化激光器用量
3. **良率领先：** 800G硅光模块良率约95%，行业领先水平
4. **客户卡位：** Nvidia主要光模块供应商，同时覆盖Google、AWS、Meta、Microsoft
5. **规模效应：** 2025年出货超2100万只，年化产能2800万只，规模优势明显

## 二、天孚通信 (TFC, 300394.SZ) — 光隔离器深度研究

### 2.1 光隔离器产品类型

天孚通信的光隔离器产品主要分为两类：

- **自由空间型隔离器（Free-Space Isolator）：** 核心产品，嵌入TOSA（光发射组件）封装中使用
- **偏振无关型隔离器（Polarization-Independent）：** 面向高速光模块应用场景

需要特别注意的是，天孚通信的隔离器**不是独立SKU出售**，而是嵌入其一站式OSA（光组件）封装方案中作为整体解决方案交付。这意味着隔离器的价值体现为提升封装方案的完整性和客户黏性，而非单独贡献收入。

### 2.2 法拉第旋光晶体：关键发现

2025年12月12日，天孚通信官方明确回应："公司产品中包括光隔离器，暂不生产法拉第旋光片。"

这一表态意味着：

- TGG（铽镓石榴石）和YIG（钇铁石榴石）等法拉第旋光晶体材料**完全依赖外部采购**
- 天孚通信在隔离器环节不具备从晶体材料到成品的垂直整合能力
- 上游核心材料的供应稳定性和成本可控性存在外部依赖

### 2.3 光隔离器竞争格局

在天孚通信不掌握法拉第旋光片自产能力的同时，国内已有多家竞争对手实现了全链条或部分链条的自主生产：

| 公司 | 法拉第旋光片自产能力 | 竞争策略 |
|---|---|---|
| 东田微 (301183) | 全链条自主生产 | 规模产能 + 成本控制 |
| 光库科技 (300620) | 高功率隔离器技术突破 | 聚焦特种应用场景 |
| 长飞光纤 (YOFC) | TGG自产 | 垂直整合路线 |
| 天孚通信 (300394) | 不自产 | 一站式平台 / JDM模式 |

从竞争策略上看，天孚通信选择了与东田微、光库科技、长飞光纤完全不同的路线。它不追求在单一器件上实现垂直整合，而是以一站式光组件平台为核心，通过JDM（联合设计制造）模式为客户提供完整封装方案。隔离器只是其平台中的一个组件。

### 2.4 1.6T光引擎就绪度

天孚通信在1.6T光引擎上的进展如下：

- 2025年4月22日公司确认："1.6T光引擎处于量产状态，因个别物料缺料尚未达到预期产量"
- 全球首家交付800G和1.6T光引擎
- EML单波200G光发射器件已实现小批量供应
- 1.6T光引擎市占率约65%，核心客户为英伟达/Mellanox

### 2.5 营收结构分析（2024年）

| 业务板块 | 营收 | 同比增长 | 占比 |
|---|---|---|---|
| 无源器件 | 15.76亿元 | +33.21% | 48.5% |
| 有源器件 | 16.55亿元 | +121.91% | 50.9% |

- 光隔离器约占无源器件的5-10%（估算值）
- 2025年预计总收入约51.15亿元，同比增长58.54%

有源器件的增速（+121.91%）远超无源器件（+33.21%），反映出天孚通信正从无源器件供应商向有源+无源一体化方案商加速转型。

### 2.6 泰国工厂建设进展

天孚通信的泰国工厂是其应对地缘政治风险和海外客户"中国+1"需求的重要布局：

| 阶段 | 时间 | 进展 |
|---|---|---|
| 一期 | 2024年6月投产 | 2025年4月通过核心客户认证 |
| 二期 | 2025年6月交付 | 预计2026年大规模量产 |

- 总投资约5亿元
- 总规划面积4万㎡

泰国工厂的推进节奏与海外客户（尤其是英伟达/Mellanox链）的供应链多元化需求高度契合。

### 2.7 核心客户结构

| 客户 | 备注 |
|---|---|
| Fabrinet | 第一大客户，2024年销售额20亿元，同比增长92.3% |
| 英伟达/Mellanox | 1.6T光引擎核心客户 |
| 华为 | 国内重要客户 |
| 中际旭创 | 光模块厂商客户 |
| 新易盛 | 光模块厂商客户 |

Fabrinet作为全球最大的光组件代工厂，是天孚通信的第一大客户。2024年对Fabrinet销售额达20亿元，同比大增92.3%，反映出天孚通信通过Fabrinet间接供应英伟达等超大规模客户的业务规模在快速膨胀。

### 2.8 竞争定位矩阵

天孚通信在光隔离器领域的竞争定位可以概括为：

| 维度 | 天孚通信 | 东田微/光库科技/长飞光纤 |
|---|---|---|
| 隔离器定位 | 平台组件之一 | 核心产品 |
| 法拉第旋光片 | 外采 | 自产 |
| 竞争壁垒 | 一站式封装能力+客户黏性 | 垂直整合+成本控制 |
| 收入贡献 | 较小（无源器件的5-10%） | 核心收入来源 |

天孚通信的核心竞争力不在于单一器件的垂直整合，而在于其一站式光组件平台能为客户提供从设计到封装的完整解决方案。这种模式的壁垒来自长期的JDM合作深度和客户验证周期，而非器件层面的材料自产能力。

## 四、风险提示

本报告仅为行业研究数据整理，不构成任何投资建议。以下风险因素值得关注：

### 中际旭创风险

1. **代工厂集中风险：** 硅光芯片完全依赖Tower Semiconductor独家代工，Tower晶圆厂分布于以色列和美国，存在地缘政治和供应集中风险。
2. **DSP外部依赖：** DSP芯片来自Broadcom和Marvell，属于美国制裁敏感器件，供应链存在不确定性。
3. **地缘政治风险：** Tower使用位于美国纽约州的晶圆厂，可能受到出口管制政策变化影响。
4. **技术替代风险：** CPO（共封装光学）和NPO（近封装光学）技术路线未来可能对可插拔硅光模块形成替代压力。

### 天孚通信风险

1. **法拉第旋光片不自产：** 光隔离器核心材料完全依赖外部采购，在上游供应紧张时可能面临成本和交期压力。
2. **有源业务占比提升或稀释利润率：** 有源器件收入增速（+121.91%）远超无源器件（+33.21%），但有源业务毛利率通常低于无源器件，业务结构变化可能影响整体利润水平。
3. **隔离器非核心主打产品：** 光隔离器仅占无源器件的5-10%，并非天孚通信的核心收入驱动力，市场对隔离器能力的关注度可能高于其实际业务贡献。
