## 第五章 竞争格局

### 5.1 vs Adobe（Premiere Pro/Firefly）

| 维度 | 万兴Filmora | Adobe Premiere |
|------|-----------|---------------|
| **定位** | 创作者/Prosumer | 专业制作/企业级 |
| **桌面市场份额** | 垂直小众 | **35%**（全球第一） |
| **AI生成** | Sora2/Veo3.1/Seedance 2.0（集成） | Firefly（自研） |
| **价格** | $7-10/月 | $23/月（单应用） |
| **生态** | 独立产品 | Creative Cloud全家桶（PS/AE/AU） |
| **移动端** | ✅强大 | ❌薄弱（Rush功能有限） |
| **核心优势** | AI生成+编辑一体化、移动端、性价比 | 专业深度、行业标准、企业级协作 |

**竞争态势**：Adobe在专业桌面视频编辑领域份额35%+难以撼动，但万兴在AI生成+编辑一体化和移动端形成差异化。两者客群重合度不高。

### 5.2 vs Canva

| 维度 | 万兴 | Canva |
|------|------|-------|
| **定位** | AI视频创意为核心 | 全品类设计平台 |
| **年化收入** | ~15.3亿元（~2.1亿美元） | ~$33亿（2026年初） |
| **估值** | ~100亿元人民币 | **$260亿**（2026年） |
| **AI能力** | 天幕+多模型集成 | Magic Studio |
| **视频编辑** | 专业级AI视频编辑 | 基础视频编辑 |
| **设计能力** | 基础 | 极强（250K+模板） |
| **企业客户** | 初期 | 10,000+（含Salesforce/Sony） |
| **付费用户** | 290万+ | 1,690万+（美图）→ Canva更高 |

**竞争态势**：Canva是设计平台龙头，万兴是视频创作垂直玩家。两者在视频编辑功能上有部分重叠，但核心用户群不同。Canva更偏营销设计，万兴更偏视频创作。

## 总结与投资建议

**核心投资逻辑**：万兴科技是A股最纯正的AI创意软件全球化标的，具有三大独特优势——

1. **全球化稀缺性**：93%收入来自海外，覆盖200+国家/地区，是A股软件出海标杆
2. **AI转型确定性**：AI原生应用收入+90%至1.3亿，AI服务器调用量13亿+次（+140%），AI商业化已从概念进入收入贡献阶段
3. **移动化新引擎**：移动端收入+60%占软件业务近25%，Filmora移动端+90%，对标CapCut的移动端付费升级空间巨大

**核心风险**：短期仍在亏损（2025年-0.91亿），AI投入高企压缩利润空间，移动端面临CapCut免费模式的激烈竞争。

**结论**：万兴科技正处于"AI投入→产品升级→用户增长→收入放量→盈利兑现"的转型关键期。2026-2027年是观察AI商业化拐点的重要窗口。若AI原生应用收入持续翻倍增长、漫剧业务快速放量、费用率持续改善，公司有望在2027年确立盈利，届时PS估值将向PE估值切换，带来显著估值修复空间。
