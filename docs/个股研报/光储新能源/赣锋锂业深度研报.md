### 1.2 定位概述

赣锋锂业是全球锂行业极少数实现"资源开采→锂化合物/金属锂加工→锂电池制造→电池回收"全产业链闭环的企业，也是中国锂企中全球化程度最高的公司之一。公司权益锂资源量超1亿吨LCE，覆盖澳大利亚硬岩锂矿、阿根廷盐湖、非洲锂辉石、国内锂云母四大资源类型。2025年，锂化合物及金属锂板块贡献营收163.52亿元（占63.2%），锂电池板块贡献82.34亿元（占31.8%），固态电池和电池回收作为战略储备板块加速落地。公司客户覆盖特斯拉、LG新能源、宝马、大众、丰田等全球主流车企及电池厂商。

> **核心定位**：全球锂产业链纵向一体化龙头，资源端多元化布局+中游加工能力全球领先+下游电池与回收形成闭环。

## 二、企业发展历程（关键里程碑时间线）

### Phase 1：创业起步期（1997-2010）

李良彬1967年生于江西，1986年进入宜春钽铌矿（国企）工作。1997年国企改制，李良彬辞职下海，以14万元承包河下镇一家金属锂厂。2000年正式成立赣锋锂业，从金属锂加工切入，逐步向下游锂化合物延伸。

| 时间 | 里程碑事件 |
|:---|:---|
| 1997年 | 李良彬从宜春钽铌矿辞职，承包河下金属锂厂 |
| 2000年3月 | 赣锋锂业正式成立，主营金属锂生产 |
| 2003年 | 建成国内首条电池级金属锂生产线 |
| 2006年 | 布局锂化合物深加工，丁基锂产品投产 |
| 2008年 | 引入战略投资者，完成股份制改造 |
| 2010年8月 | 深交所中小板上市（002460），募资6.4亿元 |

> 这一阶段的核心逻辑：从金属锂这一细分品种切入，避开与天齐锂业等在碳酸锂领域的正面竞争，以"人无我有"的产品策略建立初始壁垒。

### Phase 2：上游资源拓展+产品线完善（2011-2017）

上市后赣锋开始系统性布局上游锂矿资源，先后投资加拿大、阿根廷、澳大利亚等地锂项目，同时完善锂化合物产品矩阵，切入电池回收和固态电池研发。

| 时间 | 里程碑事件 |
|:---|:---|
| 2011年 | 投资加拿大美洲锂业（Litium Americas），首次布局盐湖资源 |
| 2014年 | 入股阿根廷Mariana盐湖项目 |
| 2015年 | 切入锂电池制造领域，成立赣锋电池科技有限公司 |
| 2015年 | 投资澳大利亚Mt Marion锂辉石矿，获取50%权益 |
| 2016年 | 正式进入电池回收领域，成立赣锋循环科技有限公司 |
| 2016年 | 启动固态电池研发，组建专项研发团队 |
| 2017年 | 投资阿根廷Cauchari-Olaroz盐湖项目 |
| 2017年 | 赣锋锂业总市值突破500亿元 |

> 这一阶段的核心逻辑：从纯加工型企业向"资源+加工"一体化转型。2015年同时切入锂电池制造和电池回收，开始构建全产业链雏形。

### Phase 3：全球化加速+A+H双上市（2018-2020）

2018年港股上市打开国际融资渠道，同时与全球头部车企和电池厂商签订长期供货协议，确立全球锂行业核心供应商地位。

| 时间 | 里程碑事件 |
|:---|:---|
| 2018年10月 | 港交所主板上市（01772.HK），募资约40亿港元 |
| 2018年 | 与特斯拉签订电池级氢氧化锂长期供货协议 |
| 2019年 | 与LG化学签订五年期氢氧化锂供货协议 |
| 2019年 | 与宝马集团签订长期供货协议，指定为电池级锂盐供应商 |
| 2020年 | 与大众集团签订十年期供货协议 |
| 2020年 | Cauchari-Olaroz盐湖一期（4万吨碳酸锂/年）建设推进 |
| 2020年 | 电池级氢氧化锂产能扩至8.1万吨/年 |

> 这一阶段的核心逻辑：以长期供货协议锁定下游需求，以A+H双上市打通融资渠道，以全球资源布局对冲单一地区政策风险。

### Phase 4：锂价暴涨到深度调整（2021-2024）

锂价在2022年冲上历史高点后急速回落，赣锋业绩经历"过山车"式波动，同时推进马里Goulamina等关键资源项目。

| 时间 | 里程碑事件 |
|:---|:---|
| 2021年 | 锂价开始暴涨，电池级碳酸锂从~5万元/吨升至年底~28万元/吨 |
| 2022年 | 碳酸锂价格最高突破60万元/吨，公司全年营收418.3亿元，归母净利润205.0亿元（历史峰值） |
| 2022年7月 | 收购马里Goulamina锂辉石矿100%权益 |
| 2023年 | 锂价暴跌，碳酸锂从年初~50万元/吨跌至年底~10万元/吨 |
| 2023年 | 全年营收326.7亿元，归母净利润降至47.5亿元（同比-76.8%） |
| 2024年 | 碳酸锂价格维持低位（7-11万元/吨区间），全年营收195.8亿元，归母净利润亏损20.73亿元 |
| 2024年 | Goulamina一期（50.6万吨精矿/年）投产 |

> **关键数据对比**：2022年净利205亿 vs 2024年亏损20.7亿，两年间利润波动超过225亿元。这既是锂行业高周期性的真实写照，也暴露了公司在中游加工端的利润弹性远大于资源端的现实。

### Phase 5：复苏与新格局（2025至今）

锂价企稳回升，公司凭借资源自给率提升和锂电池板块放量实现扭亏为盈，同时明确三大核心板块战略定位。

| 时间 | 里程碑事件 |
|:---|:---|
| 2025年Q1 | 扭亏为盈，单季度归母净利润4.67亿元 |
| 2025年H1 | 上半年营收131.5亿元，归母净利润9.83亿元 |
| 2025年全年 | 营收258.66亿元（同比+32.1%），归母净利润22.08亿元 |
| 2025年 | 锂盐出货18.48万吨LCE（同比+42%），锂电池出货17.82GWh（同比+117%） |
| 2025年 | Goulamina全年产出锂精矿33.7万吨，Cauchari-Olaroz产碳酸锂3.41万吨 |
| 2025年 | 确认三大核心板块：锂矿资源、锂化合物及金属锂、锂电池 |
| 2025年 | 磷酸铁锂回收市占率达行业第一，锂回收率94% |
| 2026年Q1 | 延续盈利态势，固态电池装车测试推进中 |

> 2025年是赣锋的"交卷年"：资源自给率提升后成本端改善，锂电池放量拉动收入增长，电池回收形成独立利润贡献点。三大板块的协同效应开始显现。

#### 板块一：锂矿资源开发

| 指标 | 数据 |
|:---|:---|

| 全球权益资源量 | 超1亿吨LCE |
| 资源类型覆盖 | 硬岩锂矿、盐湖卤水、锂云母、锂黏土（四种全覆盖） |

| 战略意义 | 提升资源自给率，降低周期波动对利润的冲击 |

#### 板块二：锂化合物及金属锂

| 指标 | 数据 |
|:---|:---|
| 起家业务 | 2000年从金属锂起家 |
| 2025年出货量 | 18.48万吨LCE（同比+42%） |
| 总产能 | ~34.8万吨/年（含在建产线） |
| 主要产品 | 电池级碳酸锂、电池级氢氧化锂、氯化锂、金属锂、丁基锂、氟化锂等 |
| 2025年板块营收 | 163.52亿元（占总营收63.2%） |
| 客户结构 | 特斯拉、LG新能源、宝马、大众、丰田、宁德时代、比亚迪等 |
| 竞争优势 | 全球最大氢氧化锂生产商之一，电池级产品纯度行业领先 |

> 氢氧化锂是高镍三元正极材料的核心原料，纯度要求远高于碳酸锂。赣锋的电池级氢氧化锂纯度可达99.56%以上，这是绑定特斯拉和LG的核心技术壁垒。

#### 板块三：锂电池制造

| 指标 | 数据 |
|:---|:---|
| 切入时间 | 2015年 |
| 2025年营收 | 82.34亿元（占总营收31.8%） |
| 2025年出货量 | 17.82GWh（同比+117%） |
| 产品覆盖 | 动力电池（磷酸铁锂+三元）、储能电池、消费类电池 |
| 主要客户 | 特斯拉（储能）、广汽、东风、赛力斯等 |
| 产能布局 | 江西（新余/赣州）、重庆、东莞、德国（规划中） |
| 增长驱动 | 储能电池放量（特斯拉Megapack供应链）、动力电池客户拓展 |

> 锂电池板块2025年同比翻倍增长，储能电池是核心增量。作为特斯拉Megapack的磷酸铁锂电池供应商之一，这部分业务的确定性较高。

#### 板块四：固态电池

| 指标 | 数据 |
|:---|:---|
| 研发启动 | 2016年 |
| 技术路线 | 硫化物+氧化物+聚合物三条路线并行 |
| 能量密度梯度 | 320Wh/kg（一代）→ 400Wh/kg（二代）→ 500Wh/kg（三代） |
| 一代产品 | 320-350Wh/kg，已通过部分车企装车测试 |
| 二代产品 | 400Wh/kg，中试阶段 |
| 三代产品 | 500Wh/kg，实验室阶段 |
| 生产基地 | 重庆（固态电池专用产线） |
| 战略定位 | 技术储备+品牌溢价，非短期利润贡献点 |

> 赣锋是国内最早布局固态电池的锂企之一。三条技术路线并行是保守策略，但也意味着研发资源分散。硫化物路线（日韩主流选择）是重点突破方向，已解决部分电解质稳定性问题。

#### 板块五：电池回收

| 指标 | 数据 |
|:---|:---|
| 正式进入时间 | 2016年 |
| 磷酸铁锂回收市占率 | 行业第一（2025年） |
| 锂回收率 | 94% |
| 镍钴锰回收率 | >98% |
| 处理能力 | 年处理废旧电池~10万吨 |
| 生产基地 | 江西（新余/赣州） |
| 核心优势 | 前端自有锂电池产能+后端回收形成闭环，退役电池来源有保障 |
| 战略意义 | 稳定锂资源供给的第二渠道，降低对矿产资源的依赖 |

> 电池回收是赣锋区别于天齐锂业等纯资源型公司的关键板块。94%的锂回收率意味着废旧电池中的锂几乎可以全部再生利用，随着动力电池退役潮到来，这一板块的战略价值将持续提升。

## 五、财务分析：营收、利润与毛利率

### 5.1 营收变化（2019-2025年）

| 年份 | 营收（亿元） | 同比 | 备注 |
|:----:|:----------:|:----:|:----:|
| 2019 | 53.42 | +6.75% | 行业底部，平稳过渡 |
| 2020 | 55.24 | +3.41% | 疫情冲击下韧性增长 |
| 2021 | 111.62 | +102.07% | 新能源需求爆发，锂价起飞 |
| 2022 | 418.22 | +274.68% | 碳酸锂峰值60万元/吨，营收巅峰 |
| 2023 | 329.72 | -21.16% | 锂价崩跌，量增价缩 |
| 2024 | 189.06 | -42.66% | 碳酸锂跌破8万元/吨，周期至暗时刻 |
| 2025 | 258.66 | +32.14% | 止跌回升，锂电池+回收贡献增量 |

赣锋锂业营收走势呈现教科书级的周期股特征。2021至2022年，碳酸锂价格从约8万元/吨飙升至近60万元/吨，营收两年翻了近4倍。这种涨幅并非来自产能扩张，而是价格驱动。2022年418.22亿元的营收中，锂系列产品贡献了345.80亿元，占比82.7%，是典型的"价格型繁荣"。

2023至2024年，碳酸锂价格从50万元/吨暴跌至7-8万元/吨区间，跌幅超过85%。出货量虽然在增长，但价格的断崖式下跌导致营收从418.22亿元缩水至189.06亿元。2025年营收回升至258.66亿元，同比+32.14%，标志着底部确认和新一轮修复周期的开始。回升动力来自三个方面：锂价企稳在8-10万元/吨区间、锂电池业务收入快速增长（出货17.82GWh，同比+117%）、电池回收业务贡献增量。

> **关键判断**: 2024年是本轮周期的营收底部（189.06亿元），2025年止跌回升确认右侧信号。但258.66亿元的营收仅为2022年峰值的61.8%，距离前高仍有较大空间。

### 5.2 归母净利润、毛利率、净利率趋势

| 年份 | 归母净利润（亿元） | 毛利率 | 净利率 | ROE |
|:----:|:---------------:|:------:|:------:|:---:|
| 2019 | 3.58 | ~23.5% | ~6.7% | 4.38% |
| 2020 | 10.25 | ~21.4% | ~18.6% | 11.07% |
| 2021 | 52.28 | ~39.8% | ~46.8% | 31.04% |
| 2022 | 205.04 | ~49.2% | ~49.0% | 61.41% |
| 2023 | 49.47 | ~13.7% | ~15.0% | 10.84% |
| 2024 | -20.73 | 10.82% | -13.91% | -4.60% |
| 2025 | 22.08 | ~15.7% | ~8.5% | ~6.52% |

这张表揭示了一个残酷事实：**锂行业的利润弹性远大于营收弹性**。

2022年毛利率高达49.2%，净利率49.0%，ROE达到惊人的61.41%。这意味着当年每卖出100元的锂产品，有近一半变成了净利润。这种利润率水平在全市场制造业中极为罕见，本质上是锂价暴涨带来的"资源租金"。

2024年则是镜像反转。毛利率从49.2%暴跌至10.82%，归母净利润由盈转亏（-20.73亿元）。这是赣锋上市以来首次年度亏损。亏损的核心原因有三：锂价跌破部分产线的现金成本线、高价库存带来的存货跌价损失、锂电池板块竞争加剧导致毛利率同步下滑。

2025年毛利率回升至约15.7%，归母净利润22.08亿元，实现扭亏。净利率约8.5%，ROE约6.52%（基于2025年净资产338.52亿元计算），距离2021至2022年的盈利水平仍有巨大差距。这表明锂行业已进入"温和盈利"的新常态，暴利时代难以重现。

> **核心洞察**: 毛利率从49.2%→10.82%→15.7%的走势，清晰地刻画了锂行业的完整周期。2024年10.82%为利润率底部，2025年开启修复。但15.7%的毛利率水平更接近行业长期中枢，49.2%的峰值属于极端行情，不可作为估值参考。

### 5.3 业务板块收入拆分

| 业务板块 | 2019年 | 2022年 | 2023年 | 2025年 |
|:-------:|:-----:|:-----:|:-----:|:-----:|
| 锂化合物及金属锂 | 41.61亿（77.9%） | 345.80亿（82.7%） | 244.65亿（74.2%） | 163.52亿（63.2%） |
| 锂电池系列 | 6.03亿（11.3%） | 64.78亿（15.5%） | 77.07亿（23.4%） | 82.34亿（31.8%） |
| 其他业务 | 5.78亿（10.8%） | 7.64亿（1.8%） | 8.00亿（2.4%） | 12.80亿（5.0%） |
| **合计** | **53.42亿** | **418.22亿** | **329.72亿** | **258.66亿** |

这个拆分表是理解赣锋战略转型的核心数据。两个趋势一目了然。

**趋势一：锂化合物占比从82.7%降至63.2%。** 2022年锂系列产品贡献345.80亿元，到了2025年仅163.52亿元，绝对金额下降52.7%。占比从82.7%降至63.2%，这不是主动收缩，而是周期下行导致的价格坍塌。但即便在锂价低迷期，锂化合物及金属锂仍是赣锋的收入基石，贡献了超过六成营收。

**趋势二：锂电池占比从11.3%升至31.8%。** 从2019年的6.03亿元增长至2025年的82.34亿元，6年CAGR约54%。在锂系列收入腰斩的背景下，锂电池业务持续放量增长，成为周期下行的关键缓冲。2023年锂价暴跌时，锂电池收入77.07亿元反而同比增长19.2%，逆周期属性凸显。2025年锂电池出货17.82GWh，同比+117%，是公司增速最快的业务板块。

> **战略意义**: 赣锋正从"锂矿+锂盐"的单轮驱动，转向"锂化合物+锂电池"的双轮驱动。锂电池6年CAGR约54%，是公司增长最快的业务。当锂系列产品收入从345.80亿跌至163.52亿时，锂电池从64.78亿增长至82.34亿，部分对冲了周期冲击。业务结构正从2022年的"锂盐一家独大"（82.7%）演变为2025年的"锂化合物63.2%+锂电池31.8%"的更均衡格局。

### 5.4 各板块毛利率对比

| 板块 | 2019年 | 2022年 | 2023年 | 2024H1 | 2025年 |
|:---:|:-----:|:-----:|:-----:|:------:|:-----:|
| 锂化合物及金属锂 | 27.17% | 49.50% | 12.53% | 11.75% | ~15.5% |
| 锂电池系列 | 13.79% | 17.00% | 17.96% | 9.88% | ~14.6% |
| 其他业务（含回收） | 7.25% | — | — | 0.40% | ~21.7% |
| **综合毛利率** | **23.5%** | **49.2%** | **13.7%** | **10.8%** | **~15.7%** |

三个板块的毛利率走势呈现截然不同的周期特征。

**锂化合物是周期放大器。** 毛利率波动幅度最大，从27.17%到49.50%再到10.75%，振幅接近39个百分点。这直接反映了锂价对资源型业务利润的放大效应。2022年49.50%的毛利率是碳酸锂价格畸高的产物，2023至2024年锂价回归后毛利率跌至11-12%，更接近锂盐加工环节的真实盈利水平。

**锂电池是稳定器。** 毛利率波动相对温和，2019至2023年基本维持在13-18%区间。2024H1降至9.88%，主要受动力电池行业价格战影响。2025年回升至约14.6%，表明行业竞争格局有所改善。锂电池业务对锂价的敏感度远低于锂系列产品，在公司收入结构中的占比提升有助于平抑整体利润波动。

**其他业务（含回收）是增长极。** 2025年毛利率跃升至约21.7%，不仅远超历史水平，也高于锂化合物和锂电池板块。电池回收业务的高毛利率源于两个因素：废旧电池原材料成本低廉、锂回收率94%带来较高的金属回收价值。随着退役动力电池放量，这个板块的利润贡献将持续提升。

> **周期定位**: 锂化合物是赣锋的"进攻武器"，在锂价上行期贡献超额利润；锂电池是"防御盾牌"，在锂价下行期提供利润缓冲；回收业务是"未来引擎"，独立于锂价周期，受益于退役电池放量。三者的毛利率组合正在从"单极依赖"走向"三足鼎立"。

三家公司的财务画像差异极大，折射出各自商业模式的本质区别。

**盐湖股份以约57.9%的毛利率和57.7%的净利率一骑绝尘。** 这是因为盐湖提锂成本仅在3-4万元/吨，远低于锂辉石提锂和锂云母提锂。当碳酸锂价格在8万元/吨附近时，盐湖股份仍有50%以上的毛利率，而赣锋和天齐的锂盐业务已接近盈亏平衡线。盐湖股份的资产负债率仅约14%，财务结构极为稳健，是典型的"现金奶牛"。

**天齐锂业毛利率约39.5%看似可观，但净利率仅4.5%。** 这个剪刀差反映了天齐的核心问题：高昂的财务成本和管理费用。天齐在2018年收购SQM 23.77%股权时承担了大量债务，虽然资产负债率已从高峰期的约73%降至约41%，但利息支出仍然侵蚀利润。2025年SQM利润大幅缩水，天齐的投资收益同步下滑。天齐的优势在于拥有Greenbushes全球最优质锂辉石矿的控股权，锂精矿成本约5.5-6万元/吨（折碳酸锂当量），低于赣锋的7.1-7.5万元/吨。

**赣锋锂业营收规模最大（258.66亿元），但利润率最低。** 毛利率约15.7%、净利率约8.5%在三家公司中垫底。这并非经营效率低下，而是业务结构不同。赣锋的锂电池业务（2025年收入82.34亿元）本质上是制造业，毛利率约14.6%，拉低了整体水平。如果仅看锂化合物及金属锂板块，赣锋的盈利能力与天齐更为接近。值得注意的是，赣锋2025年经营性现金流净额高达62.35亿元，远超归母净利润22.08亿元，现金流质量优异。

> **核心结论**: 盐湖股份是"成本之王"，天齐锂业是"资源之王"，赣锋锂业是"规模之王"。三者的利润率差异主要由资源禀赋和业务结构决定，而非管理能力差距。

2024年是分水岭。天齐锂业巨亏79.05亿元，跌幅最为惨烈。这主要因为SQM投资收益大幅缩水（SQM本身在2024年利润暴跌）、锂精矿价格深度回调、以及前期高价库存的跌价损失。赣锋亏损20.73亿元，幅度远小于天齐，锂电池业务的逆周期缓冲效应在此刻发挥了关键作用。盐湖股份2024年仍盈利72.43亿元，同比仅下降19.1%，展现了极强的抗周期能力。

2025年，赣锋率先扭亏（22.08亿元），天齐仅微利4.63亿元，盐湖维持在84.76亿元。赣锋的利润修复速度领先于天齐，得益于锂电池业务持续放量（出货+117%）和电池回收贡献增量。经营性现金流62.35亿元更是远超净利润，表明盈利质量扎实。

> **周期弹性排名**: 天齐锂业（利润波动最大，弹性最高）> 赣锋锂业（中等弹性，锂电池缓冲）> 盐湖股份（波动最小，防御性最强）。投资者在锂价上行期应偏好天齐，下行期应偏好盐湖，全周期配置可选择赣锋。

### 6.3 战略差异对比

| 维度 | 赣锋锂业 | 天齐锂业 | 盐湖股份 |
|:---:|:-------:|:-------:|:-------:|
| 商业模式 | 全产业链垂直整合（矿→盐→电池→回收） | 上游资源集中型（优质矿+SQM投资） | 盐湖提钾+提锂双主业 |
| 核心资源 | Mt Marion/Cue/COVAL/SDLA等全球多矿 | Greenbushes（全球最优质锂辉石矿）+SQM股权 | 察尔汗盐湖（中国最大钾锂资源） |
| 资源类型 | 硬岩+盐湖+锂云母+锂黏土（四种全覆盖） | 硬岩（Greenbushes）+盐湖（SQM Atacama） | 盐湖卤水 |
| 资源自给率 | 50-60% | ~100% | ~100%（自有盐湖） |
| 提锂成本 | 7.1-7.5万/吨（锂辉石） | 5.5-6万/吨（锂辉石） | 3-4万/吨（盐湖） |
| 下游延伸 | 锂电池+固态电池+回收 | 无下游业务 | 无下游业务（钾肥为主） |
| 抗周期能力 | 中等（锂电池缓冲） | 较弱（纯上游，受锂价冲击最大） | 最强（钾肥对冲+成本优势） |
| 成长性 | 最高（固态电池先发+回收放量） | 中等（依赖锂价和SQM分红） | 较低（盐湖产能扩张有限） |
| 海外布局 | 全球化（澳洲+阿根廷+非洲+欧洲） | 集中（澳洲+智利） | 纯国内（青海察尔汗） |
| 财务风险 | 中等（资产负债率~58.6%） | 中等（负债率~41%，但有SQM减值风险） | 最低（负债率~14%） |

> **战略定位总结**: 赣锋是"全产业链选手"，赌的是垂直整合带来的长期竞争优势和固态电池的弯道超车。天齐是"优质矿主"，赌的是锂价中枢长期上行和Greenbushes的资源溢价。盐湖是"成本堡垒"，在锂价低迷期也能盈利，但成长性受限。三种模式各有优劣，投资者应根据锂价预期和风险偏好选择。

## 七、竞争格局：上中下游全面对比

### 7.1 上游锂矿资源竞争

全球锂资源争夺已进入白热化阶段。中国锂企业面临的共同挑战是：国内锂资源禀赋不佳（以锂云母和盐湖为主，品质低于澳洲锂辉石），必须通过海外并购获取优质资源。

**主要玩家资源布局：**

| 企业 | 核心资源 | 资源类型 | 权益储量（LCE） | 自给率 | 提锂成本 |
|:---:|:------:|:------:|:----------:|:-----:|:------:|
| 赣锋锂业 | Mt Marion（澳）、Goulamina（马里）、Cauchari-Olaroz（阿）、Mariana（阿）、PPGS（阿） | 硬岩+盐湖+锂云母+锂黏土 | 超1亿吨 | 50-60% | 7.1-7.5万/吨 |
| 天齐锂业 | Greenbushes（澳，51%）、SQM（智利，23.77%） | 硬岩+盐湖 | ~3,200万吨 | ~100% | 5.5-6万/吨 |
| 盐湖股份 | 察尔汗盐湖（青海） | 盐湖卤水 | ~1,600万吨 | ~100% | 3-4万/吨 |
| 紫金矿业 | 3Q（阿根廷）、LAC（加拿大） | 盐湖+硬岩 | ~1,500万吨 | 0→规划中 | 待定 |

**成本梯队：**

```
第一梯队（3-4万/吨）：盐湖股份（察尔汗盐湖）— 成本优势碾压级
第二梯队（5.5-6万/吨）：天齐锂业（Greenbushes）— 全球最优锂辉石矿
第三梯队（7.1-7.5万/吨）：赣锋锂业（澳洲锂辉石）— 多矿组合对冲
第四梯队（12-13万/吨）：锂云母提锂企业（永兴材料等）— 碳酸锂<10万即亏损
```

**赣锋上游资源强弱分析：**

| 维度 | 评价 | 说明 |
|:---:|:---:|:---|
| 储量规模 | ★★★★★ | 权益资源量超1亿吨LCE，全球第一梯队 |
| 资源类型多样性 | ★★★★★ | 硬岩+盐湖+锂云母+锂黏土，全球最全 |
| 地域分散度 | ★★★★★ | 澳洲+南美+非洲+中国，风险分散 |
| 成本竞争力 | ★★★☆☆ | 7.1-7.5万/吨，高于天齐和盐湖 |
| 自给率 | ★★★☆☆ | 50-60%，仍有40-50%依赖外购 |
| 资源控制力 | ★★★★☆ | 多数为包销+合资，部分100%控股 |

> **上游关键结论**: 赣锋强在储量大（超1亿吨LCE）、类型全（四种资源全覆盖）、地域分散（三大洲布局），弱在成本和自给率。2024年碳酸锂跌破8万元/吨时，赣锋部分外购矿石产线已接近或触及现金成本线。自给率从50-60%提升至70%+是未来3年的核心任务，Goulamina达产（85万吨精矿/年）和Mariana盐湖放量是关键节点。

### 7.2 中游锂化合物竞争

赣锋在中游锂化合物领域拥有全球领先的规模和技术优势。

**产能规模对比：**

| 产品 | 赣锋产能 | 全球排名 | 市占率 |
|:---:|:------:|:------:|:-----:|
| 碳酸锂/氢氧化锂 | 34.8万吨/年（含在建） | 全球第一 | ~20% |
| 金属锂 | 5,000吨/年 | 全球第一 | 60%+ |
| 硫化锂 | 5,000吨/年 | 全球第一 | ~80%（先发垄断） |
| 丁基锂 | 6,000吨/年 | 全球前三 | ~30% |
| 氯化锂 | 10,000吨/年 | 中国第一 | ~40% |

**核心客户矩阵：**

| 客户类型 | 代表客户 | 合作深度 |
|:------:|:------:|:------:|
| 电芯企业 | 宁德时代、LG新能源、比亚迪、三星SDI | 长协+合资 |
| 整车企业 | 特斯拉、宝马、大众、奔驰、现代 | 战略长协 |
| 正极材料 | 容百科技、当升科技、德方纳米 | 批量供应 |
| 传统工业 | 3M、陶氏化学 | 细分垄断 |

赣锋的锂盐长协客户覆盖率超过70%，在锂价下行期提供了稳定的需求锚点。与特斯拉、LG、宁德时代的长协合同通常为3-5年期，采用公式定价机制（挂钩碳酸锂现货价+加工费），在价格剧烈波动期为双方提供了一定的可预测性。

**硫化锂的先发垄断地位值得特别关注。** 5,000吨硫化锂年产能是全球最大规模，主要用于固态电池硫化物电解质原料。随着固态电池产业化加速，硫化锂的需求将呈指数级增长。赣锋在硫化锂领域的先发优势，类似于其在金属锂领域的长期垄断，是未来3-5年核心竞争壁垒之一。

> **中游关键结论**: 赣锋在锂化合物领域是绝对的"规模+品类"双料冠军。34.8万吨锂盐产能、金属锂60%+全球市占、硫化锂先发垄断，构成了坚固的中游护城河。但绝对成本仍高于天齐锂业，在锂价低迷期利润率承压。优势在于规模和客户，劣势在于成本。

### 7.3 下游固态电池竞争（六强对比）

固态电池是锂行业未来十年最具想象力的赛道。以下是目前国内主要竞争者的全面对比：

| 企业 | 技术路线 | 能量密度（Wh/kg） | 量产进度 | 核心优势 | 赣锋差异 |
|:---:|:------:|:---------------:|:-------:|:------:|:-------:|
| **赣锋锂业** | 硫化物为主+氧化物 | 320-500 | **全球首家装车量产**（东风E70） | 唯一全链路一体化 | — |
| 宁德时代 | 三路线并行（硫化物+氧化物+聚合物） | 360-450 | 半固态已装车（极氪009），全固态2027年 | 规模+专利+资金 | 赣锋有全链路优势 |
| 比亚迪 | 硫化物 | 480 | 2026Q3小批量装车 | 整车一体化+成本控制 | 赣锋装车进度领先 |
| 清陶能源 | 氧化物 | 400+ | 已装车（智己L6） | 上汽深度绑定 | 赣锋资源端更强 |
| 卫蓝新能源 | 半固态→全固态 | 360-400 | 半固态已量产（蔚来ET7） | 中科院技术背景 | 赣锋量产更早 |
| 亿纬锂能 | 阶梯式升级 | 300→400+ | 2026年3月60Ah电芯下线 | 大圆柱+储能布局 | 赣锋先发优势明显 |

**赣锋的独特定位：从矿石到电芯的唯一全链路玩家。**

```
锂矿石 → 碳酸锂/氢氧化锂 → 硫化锂 → 硫化物电解质 → 金属锂负极 → 固态电芯 → 电池系统
  ↑                                                                          ↑
 赣锋                                                                       赣锋
```

这条产业链上，其他竞争对手最多覆盖2-3个环节，而赣锋是唯一能从锂矿石一路做到电池系统的企业。这意味着：

1. **原材料自主可控**：硫化锂自产自给，不受制于外部供应商
2. **成本优势**：省去中间环节的利润加成和物流成本
3. **质量可控**：从矿石到电芯全程品控，减少供应链质量风险
4. **技术协同**：上游材料研发与下游电芯设计形成正向反馈

> **固态电池核心判断**: 赣锋是全球首家实现固态电池装车量产的企业（东风E70，2023年），这个先发优势在产业早期极为珍贵。但宁德时代凭借资金、规模和客户资源，正在加速追赶。赣锋的时间窗口约2-3年，需要在这段时间内将产能从目前的GWh级提升至10GWh+，才能建立持续的竞争壁垒。

### 7.4 电池回收竞争

动力电池退役潮正在到来，2025年预计退役量超过50万吨，2030年将突破300万吨。电池回收成为锂行业新的战略高地。

**主要竞争者对比：**

| 企业 | 核心能力 | 处理规模 | 锂回收率 | 市占率 | 基地布局 |
|:---:|:------:|:------:|:------:|:-----:|:------:|
| **赣锋锂业** | 湿法回收+闭环整合 | 10万吨+（规划） | **94%** | 磷酸铁锂回收第一 | 国内多基地 |
| 宁德时代（邦普） | 规模化回收+前驱体 | 27万吨 | ~90% | 三元回收第一 | 4大基地 |
| 格林美 | 专业回收+多金属提取 | 15万吨 | ~92% | 综合回收第二 | 6大基地 |
| 华友钴业 | 钴锂综合回收 | 8万吨 | ~88% | 钴回收领先 | 3大基地 |
| 光华科技 | 磷酸铁锂专项回收 | 5万吨 | ~90% | 快速成长 | 2大基地 |

赣锋在电池回收领域的核心优势不在规模（邦普27万吨更大），而在于**与上游锂盐和下游电池的闭环整合**。回收得到的碳酸锂直接进入赣锋自身的锂盐生产线，回收得到的磷酸铁锂前驱体供应给锂电池板块。这种"废料进、产品出"的闭环模式，省去了回收产品的市场销售环节，利润率更高。

2025年回收业务毛利率达到约21.7%，已超过锂化合物（约15.5%）和锂电池（约14.6%），成为赣锋利润率最高的业务板块。随着磷酸铁锂电池退役量激增（2020-2021年装车的磷酸铁锂电池将在2025-2027年集中退役），回收业务有望在2027-2028年贡献更大的收入和利润增量。

> **回收业务关键判断**: 赣锋的回收业务是独立于锂价周期的增长极。锂回收率94%处于行业领先水平，磷酸铁锂回收市占率行业第一。当锂价低迷时，废旧电池原材料成本同步下降，回收业务利润率反而可能提升。这是赣锋抗周期能力的重要拼图。

## 八、锂行业景气度与供需分析

### 8.1 锂价走势回顾（2020-2026年关键节点）

碳酸锂与氢氧化锂过去六年经历了一轮完整的超级周期:从2020年底部4万元起步,暴涨至2022年末近60万元历史极值,随后崩跌90%至2025年不足6万元,又在2026年强劲反弹至20万元附近。

| 时间 | 碳酸锂（万元/吨） | 氢氧化锂（万元/吨） | 关键事件 |
|------|-------------------|---------------------|----------|

### 8.5 盐湖提锂 vs 矿石提锂成本对比

不同技术路线的成本差异决定了各企业在锂价下行周期中的生存能力,也决定了价格底部支撑位。

| 技术路线 | 代表企业 | 完全成本（万元/吨LCE） | 现金成本（万元/吨LCE） | 成本优势 |
|---------|---------|---------------------|---------------------|---------|
| 盐湖(吸附法+膜法) | 盐湖股份 | 3.65 | 2.8 | 极强,锂价5万仍盈利 |
| 盐湖(吸附法) | 藏格矿业 | 3.50 | 2.6 | 极强,成本最低梯队 |
| 盐湖(浓缩沉淀法) | SQM/Albemarle | 3.8-4.5 | 3.0-3.5 | 强,南美优质盐湖 |
| 锂辉石(澳洲精矿) | 天齐锂业 | 6.0 | 4.8 | 中等,拥有Greenbushes股权 |
| 锂辉石(澳洲精矿) | 行业主流 | 5.5-6.0 | 4.5-5.0 | 中等 |
| 锂辉石(自有+外购) | 赣锋锂业 | 7.1-7.5 | 5.5-6.0 | 一般,原料对外依存度较高 |
| 锂云母(宜春) | 永兴材料等 | 7.5-9.0 | 6.0-7.5 | 较弱,锂价<8万即亏损 |
| 锂云母(低品位) | 行业边缘产能 | 9.0-12.0+ | 7.5-10.0 | 很弱,锂价<10万即停产 |

> **核心结论**: 盐湖提锂完全成本3.5-4.5万元/吨,仅为矿石提锂的一半。赣锋锂业锂辉石路线完全成本7.1-7.5万元,在锂价5.95万元底部时已深度亏损。锂云母路线成本最高,是锂价下行周期中最先退出的边际产能,构成价格底部支撑。

**成本曲线对价格的含义**:
- 锂价5.95万时:盐湖全部盈利,锂辉石大部分亏损,锂云母全面停产
- 锂价8万时:盐湖+优质锂辉石盈利,赣锋接近盈亏平衡,锂云母部分复产
- 锂价12万时:全行业盈利,锂云母大规模复产
- 锂价20万时:全行业暴利,远超全部技术路线成本

### 8.6 行业景气度判断

**底部已确认**: 2025年4-6月为本轮锂价周期的绝对底部。碳酸锂现货价格在5.95万元/吨企稳,氢氧化锂在6.3万元/吨见底。底部特征包括:锂云母提锂全面停产、澳洲部分矿山减产保价、行业龙头普遍亏损、贸易商库存降至历史低位。

**反转的三大驱动**:

1. **储能需求爆发**: 2025年全球储能装机量同比+74%,远超市场年初预期,成为锂需求增长最快的应用场景
2. **供给端主动收缩**: 津巴布韦原矿出口禁令、宁德时代锂盐产线停产、高成本锂云母产能退出、多个绿地项目推迟
3. **库存降至极低**: 下半年锂价持续上涨过程中,中游正极材料厂和电池厂库存消费比降至2-3周的历史低位,补库需求旺盛

**2026年景气度判断**: 处于上行周期中后段。价格中枢预计在19-20万元/吨,对应锂辉石提锂企业净利润8-12万元/吨。但需注意,锂价已从底部反弹227%,进一步上行空间取决于储能需求能否持续超预期。

**2027年前瞻**: 确定性短缺年份。2025-2026年新投产项目有限,而新能源车和储能需求持续高增长,供需缺口可能扩大至3-10万吨LCE。如果锂价维持在15万元以上,部分推迟项目可能在2027年下半年投产,但远水解不了近渴。

## 十、综合投资建议

### 10.1 核心投资逻辑

**逻辑一:周期底部已过,盈利确定性恢复**

赣锋锂业2024年归母净亏损20.7亿元,2025年实现扭亏,2026年Q1营收同比暴增144%至122亿元。盈利恢复的核心驱动力是锂价从5.95万底部反弹至19.5万,叠加公司锂盐产量从2024年的17.4万吨提升至2026年目标23万吨。量价齐升之下,2026年全年净利润有望突破60亿元,2027年进一步增至80-100亿元。

**逻辑二:锂电池第二曲线从量变到质变**

锂电池业务收入从2022年的6亿元飙升至2025年的82亿元,占公司总收入比例从11%升至36%。2026年锂电池出货量目标40+GWh,对应收入有望突破100亿元。这条第二曲线显著降低了赣锋对锂价的单一依赖,使公司从纯周期股向"周期+成长"双属性切换。

**逻辑三:固态电池差异化,全链路一体化是杀手锏**

赣锋是全球唯一实现"锂资源提取→锂盐加工→正负极材料→电芯制造→固态电解质→电池回收"全链路一体化的锂业企业。固态电池已建成4GWh产能,2026年预计装车验证。在宁德时代、比亚迪统治的锂电池红海中,固态电池是赣锋差异化的核心武器,也是资本市场给予其高于天齐锂业50%估值溢价的主要理由。

**逻辑四:资源端持续加码,2030年60万吨LCE目标**

公司权益资源量从2019年的2,800万吨LCE增长至2025年的5,400万吨LCE,产能从6万吨提升至规划60万吨(2030年)。Goulamina、Cauchari-Olaroz、PPGS等项目有序推进,长期资源保障能力持续增强。

### 10.2 关键风险提示

**风险一:锂价二次探底风险**

本轮锂价反弹高度依赖储能需求的持续超预期。如果2026年下半年储能装机增速放缓,或锂价涨至20万后刺激锂云母产能大规模复产,锂价可能二次回落至12-15万区间。对应赣锋锂盐单吨利润从12万降至5-8万,净利润预期下调30-50%。

**风险二:高负债率与流动性压力**

截至2025年末,公司资产负债率约53%,总负债超500亿元。其中流动负债超过流动资产约967亿元(主要因扩产资本开支),短期偿债压力较大。如果锂价再次大幅下跌,现金流可能紧张,迫使公司放缓扩张节奏或进行融资。

**风险三:固态电池商业化不确定性**

固态电池目前尚处于产业化早期,4GWh产能规模较小。技术路线(氧化物vs硫化物vs聚合物)尚未收敛,国际车企装车验证时间表可能推迟。如果2027年前无法实现规模化装车,市场可能下调对固态电池的估值溢价。

**风险四:海外项目地缘政治风险**

赣锋的核心资源项目分布在阿根廷(Cauchari-Olaroz)、马里(Goulamina)、墨西哥(Sonora)等地。墨西哥Sonora粘土提锂项目已面临政府仲裁争端,阿根廷政策不确定性(外汇管制、出口税),非洲资源民族主义抬头(津巴布韦出口禁令),均可能影响海外项目的产能释放和利润汇回。

**风险五:锂电池竞争加剧**

锂电池业务面临宁德时代(全球份额37%)、比亚迪(17%)、中创新航(5%)等巨头的激烈竞争。赣锋锂电池主要聚焦储能和动力电池细分市场,但行业产能过剩背景下,利润率可能承压。2025年锂电池板块毛利率仅12.8%,显著低于锂盐业务的25-30%。

### 10.3 关键时间节点

| 时间 | 预期事件 | 投资影响 |
|------|---------|---------|
| 2026年H1 | 锂盐产量达23万吨/年,锂电池出货40+GWh | 量价齐升兑现,业绩高增确定性增强 |
| 2026年H2 | PPGS可研完成,Goulamina可研推进 | 资源端叙事延续,支撑长期估值 |
| 2026-2027年 | 固态电池国际车企装车验证 | 核心催化剂,若兑现估值中枢上移30%+ |
| 2027年 | Cauchari-Olaroz二期4.5万吨LCE投产 | 盐湖低成本产能放量,改善成本结构 |
| 2027年 | Goulamina二期100万吨精矿扩产 | 资源自给率提升,降低原料成本 |

| 2030年 | 60万吨LCE产能目标 | 长期愿景,全球锂业龙头地位巩固 |

### 10.4 综合研判

**定位:传统锂业龙头+固态电池唯一全链路一体化企业**

赣锋锂业在锂产业链中的定位可以用"四个象限"来概括:

上游资源端,赣锋是"最大但不最便宜"的玩家。公司拥有全球锂业上市公司中最大的权益资源储量(5,400万吨LCE),但受制于原料对外依存度较高(自给率约40%),完全成本7.1-7.5万元/吨,不如天齐锂业(拥有Greenbushes股权)的6.0万元和盐湖股份的3.65万元。资源端的不足,公司正通过Goulamina、Cauchari-Olaroz扩产和PPGS等新项目来弥补。

中游锂盐端,赣锋是"规模最大+客户最强"的绝对龙头。2026年锂盐产能23万吨LCE,全球第一。客户覆盖特斯拉、宝马、LGES、宁德时代、比亚迪等顶级客户群,长单覆盖率高,在锂价下行周期中具有较强的订单韧性。

下游锂电池端,赣锋选择了"固态电池差异化突围"的路径。在宁德时代和比亚迪统治的液态锂电池红海中,赣锋避开了正面竞争,转而押注固态电池这条差异化赛道。全链路一体化(从锂矿到固态电解质再到电芯)是其核心竞争力,也是其他锂电池企业难以复制的护城河。

回收端,赣锋构建了"闭环协同"的再生体系。锂回收率94%,全球领先。回收业务与矿产提锂、盐湖提锂形成三大原料来源,降低了对单一矿山的依赖,也为固态电池提供了稳定的前驱体供应。

**投资策略建议**:

当前锂价处于上行周期中后段,碳酸锂19.5万元对应赣锋锂盐单吨净利润约10-12万元,年化利润60-70亿元。以2026年5月14日80元股价计算,Forward PE约18倍,在周期股中属于合理偏高水平。

短期(3-6个月):锂价仍有上行惯性,中枢可能突破20万。但股价已从92元高点回调至80元,市场在博弈锂价见顶时点。建议关注2026年Q2-Q3储能装机数据,如果持续超预期,锂价可能挑战25万,对应股价有20-30%上行空间。

中期(6-18个月):核心变量是固态电池装车验证。如果2026年底至2027年上半年实现国际车企装车,市场将从周期股估值框架切换到"周期+成长"估值框架,目标价中枢可上移至100-120元。如果装车推迟或性能不达预期,股价可能回落至60-70元区间。

长期(2-3年):2027-2028年Cauchari-Olaroz二期和Goulamina二期投产后,公司资源自给率有望从40%提升至60%以上,成本结构显著改善。叠加固态电池20GWh产能释放,2030年60万吨LCE目标如果如期实现,赣锋有望从周期股蜕变为具有长期成长性的锂业+电池一体化平台型企业。

> **风险收益比评估**: 当前80元股价对应2026E PE 18倍,处于合理区间。锂价上行+储能爆发的确定性较高,提供下行保护;固态电池装车验证提供上行期权。综合判断,当前位置的赣锋锂业具备较好的风险收益比,适合作为锂行业上行周期的核心配置标的,但需密切关注锂价走势和固态电池进展,做好动态调整。
