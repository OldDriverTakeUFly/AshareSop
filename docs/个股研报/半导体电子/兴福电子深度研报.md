## 第一章：公司概览——兴发集团孵化的电子化学品平台

### 1.1 创始人与管理层

兴福电子并非个人创业企业，而是**精细磷化工龙头兴发集团（600141）孵化的子公司**。公司的核心领军人物是**李少平**。

**李少平**，兴福电子董事长、法定代表人，**全国人大代表**。李少平的成长轨迹是典型的"产业人崛起"：

| 时间 | 职务 | 阶段 |
|------|------|------|
| 2000–2008 | 兴发集团白沙河化工厂：工人→工艺员→车间主任→副厂长→主任工程师→副总工程师 | 基层成长 |
| 2008 | 深度参与兴福电子创建 | 创业起点 |
| 2017–2022 | 兴发集团总工程师→副总经理 | 集团技术领军 |
| 2022 至今 | 兴福电子董事长 | 独立平台掌舵 |

> **核心洞察**：李少平从兴发集团化工厂**一线工人**做起，历经工艺员、车间主任、副总工程师，最终成为兴福电子董事长，并当选全国人大代表。这个成长路径说明他**深谙磷化工的底层工艺**——而电子级磷酸的核心壁垒正是"从工业黄磷提纯到 99.9999%（6N）级别"的提纯工艺。一个从车间一线成长起来的技术型掌舵人，是兴福电子能把电子级磷酸做到国内最大的深层原因。同时，全国人大代表身份为公司带来政策层面的战略视野。

### 1.2 发展历程

兴福电子的发展可划分为三个阶段：孵化培育期（2008–2022）、股份制改造与扩产期（2022–2024）、科创板上市与平台化期（2025 至今）。

**完整发展历程：**

| 年份 | 里程碑 | 战略意义 | 阶段 |
|------|--------|----------|------|
| 2008 | 兴发集团基于精细磷化工积累，成立兴福电子，进军电子化学品 | **创业起点**：磷化工→电子化学品延伸 | 孵化期 |
| 2008–2021 | 持续研发电子级磷酸提纯工艺，突破至 6N（99.9999%）级别 | **技术攻坚**：打破海外垄断 | 孵化期 |
| 2022.05–07 | 兴福有限整体变更为股份有限公司 | 规范化，筹备上市 | 改造期 |
| 2022 | 李少平任兴发集团副总工程师后转任兴福电子董事长 | 核心领军到位 | 改造期 |
| 2023.07 | 3 万吨/年电子级磷酸项目（新建）建成投产 | **产能跃升**：电子级磷酸实际产能达 5.3 万吨 | 改造期 |

| 2025 | 大基金二期（6.94%）+ SK 海力士（2.74%）+ 长江存储（4.7%）入股 | **战略股东加持** | 上市期 |

> **核心洞察**：2008 年成立到 2025 年上市，兴福电子走了 17 年——这是一家"长期主义"的公司。兴发集团作为磷化工龙头，有稳定的黄磷原料供应和成熟的提纯工艺积累，这是兴福电子能突破 6N 电子级磷酸的根基。2025 年上市+大基金/SK 海力士/长江存储入股，标志着兴福电子从"兴发集团的电子化学品事业部"升级为"半导体供应链的独立平台"。

### 1.3 公司定位与业务布局

兴福电子是国内**湿电子化学品**领域的领军企业，核心产品为电子级磷酸（国内产能最大）。公司依托兴发集团的磷化工原料优势，从工业黄磷提纯至 6N（99.9999%）级别电子级磷酸，并延伸至电子级硫酸、双氧水、氨水、功能湿电子化学品（蚀刻液/清洗剂）等全品类。

**一句话定位**：兴发集团孵化的、以电子级磷酸为核心的湿电子化学品平台，半导体国产替代的"磷系材料"承载者。

**业务布局：**

| 类别 | 详情 |
|------|------|
| 总部 | 湖北宜昌 |
| 生产基地 | 宜昌本部（依托兴发集团磷化工基地） |
| 电子级磷酸产能 | 6 万吨/年（实际最大 5.3 万吨/年） |
| 总产能 | 37.4 万吨/年（含各类湿电子化学品） |
| 客户 | 中芯国际、长江存储等头部半导体企业 |

### 1.4 股东结构特征

兴福电子的股东结构是其最大亮点——**半导体国家战略与头部客户的资本深度绑定**：

| 股东 | 持股比例 | 战略意义 |
|------|:--------:|----------|
| **兴发集团（600141）** | 39.93%（控股） | 磷化工原料支撑，实际控制人（兴山县国资） |
| **国家大基金二期** | 6.94% | 国家半导体战略投资，流通股第一 |
| **长江存储（长存资本）** | 4.7% | 核心客户深度绑定 |
| **SK 海力士** | 2.74% | 国际存储龙头，2026/1 解禁 |
| 厦门联和/合肥石溪 | 各约 2% | 集成电路产业基金 |

> **核心洞察——"明星股东阵容"的战略含义**：大基金二期（国家意志）+ 长江存储（核心客户）+ SK 海力士（国际客户）的组合，意味着兴福电子不是一家普通的材料公司，而是**国家半导体供应链安全战略的"磷系材料"承载者**。大基金二期投资通常伴随长期订单承诺，长江存储入股直接绑定采购，SK 海力士代表国际市场准入——这种股东结构为兴福电子提供了强大的客户护城河。但 SK 海力士 2026/1 解禁后的减持压力需关注。

## 第二章：业务范围与产品矩阵——从电子级磷酸到湿电子化学品平台

### 2.1 经营范围

湿电子化学品的研发、生产和销售，涵盖通用湿电子化学品与功能湿电子化学品两大类。

### 2.2 核心产品矩阵

兴福电子的产品矩阵以电子级磷酸为核心，向全品类湿电子化学品延伸：

| 产品大类 | 核心产品 | 应用场景 | 行业地位 |
|----------|----------|----------|:--------:|
| **通用湿电子化学品** | **电子级磷酸** | 半导体湿法蚀刻、清洗 | ⭐ **国内产能最大（5.3万吨）** |
| | 电子级硫酸 | 晶圆清洗 | 强 |
| | 电子级双氧水 | 清洗、氧化 | 强 |
| | 电子级氨水/氨气 | 中和、氮化 | 强 |
| **功能湿电子化学品** | 高选择性蚀刻液 | 精细蚀刻 | 成长中 |
| | 电子级清洗剂 | 湿法清洗 | 强 |
| | 干法蚀刻后清洗剂 | 先进制程清洗 | 新品 |
| **前瞻产品** | 硅基前驱体（165吨/年在建） | CVD/ALD 工艺 | 布局中 |
| | 电子级高纯红磷 | 半导体掺杂 | 研发阶段 |

> **核心洞察**：兴福电子的产品矩阵是"**磷酸为核、全品类延伸、前瞻布局前驱体**"。电子级磷酸是湿法工艺中用量最大的酸之一（晶圆制造中反复使用），兴福的 5.3 万吨产能是国内的绝对龙头。从磷酸向硫酸/双氧水/氨水横向扩展（通用湿电子），再向蚀刻液/清洗剂（功能湿电子）、硅基前驱体（先进制程材料）纵向升级，是典型的"核心单品类→平台化"演进路径。

### 2.4 技术代际演进与技术发展方向（核心章节）

**当前技术代际——电子级磷酸 6N 级别**

电子级磷酸的核心技术壁垒是**纯度**。兴福电子的技术代际演进：

| 纯度级别 | 别名 | 应用 | 兴福进展 |
|----------|------|------|----------|
| 工业级 | 黄磷 | 普通化工 | 兴发集团原料基础 |
| 电子级 3N–4N | 初级 | 低端电子 | 早期突破 |
| 电子级 5N | G3-G4 | 中端半导体 | 已量产 |
| **电子级 6N（99.9999%）** | **G5** | **先进半导体** | ⭐ **已量产（国内领先）** |
| 电子级 7N+ | 前沿 | 极紫外/先进制程 | 研发中 |

> **核心洞察**：电子级磷酸从工业黄磷提纯到 6N（99.9999%）是巨大的技术跨越——每提升一个 N，杂质含量降低 10 倍，提纯难度指数级上升。兴福电子依托兴发集团的黄磷资源和 17 年工艺积累，做到了国内 6N 电子级磷酸的最大产能（5.3 万吨），这是其核心护城河。

**技术发展方向——平台化与先进制程**

兴福电子的技术演进路径清晰：**磷酸单品（基础）→ 全品类湿电子（横向）→ 前驱体/红磷（纵向先进制程）**

| 方向 | 当前进展 | 未来潜力 |
|------|----------|----------|
| 通用湿电子全品类 | 硫酸/双氧水/氨水已量产 | 替代进口，扩大份额 |
| 功能湿电子（蚀刻液） | 高选择性蚀刻液成长中 | 先进制程精细化 |
| **硅基前驱体** | 165 吨/年在建 | CVD/ALD 高附加值材料，打开第二曲线 |
| **电子级高纯红磷** | 研发阶段 | 半导体掺杂材料，前沿布局 |

> **核心洞察**：硅基前驱体是兴福电子最重要的技术方向——前驱体是先进制程（CVD/ALD）的核心材料，单价和毛利率远高于通用湿电子。兴福从"卖磷酸"升级到"卖前驱体"，是从大宗材料商向先进制程材料商的质变。这是其未来最大的增长曲线。
