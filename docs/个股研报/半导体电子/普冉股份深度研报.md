## 七、未来发展规划与战略路径

### 7.1 公司披露的战略方向（D5 年报/官网）

1. **NOR Flash**：推进 40nm → 25nm 制程迭代，扩展大容量 1Gbit+ 产品
2. **MCU**：PY32 系列从 M0+/M4 向高性能延伸，深化"Memory+"协同
3. **车规**：EEPROM 全系列 AEC-Q100 认证已完成，重点拓展汽车电子（IVI/ADAS/仪表盘）
4. **AI 算力配套**：受益于 AI 服务器存储容量升级需求

### 7.2 隐含的战略逻辑

普冉的战略主线是**"存储工艺平台化 + 应用领域升级"**：
- **平台化**：NOR Flash SONOS 工艺 → 衍生 EEPROM → 衍生 MCU（共用存储工艺线）
- **应用升级**：消费电子（低功耗利基）→ 工业 → 汽车（车规认证）→ AI 服务器（容量升级）

> **关键风险点**：NOR 25nm 制程迭代需要代工厂配合（中芯/华虹），普冉作为 Fabless 无自主控制力。若代工厂产能紧张或优先服务大客户，普冉制程追赶可能受阻。
