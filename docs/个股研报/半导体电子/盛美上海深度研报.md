## 第一章：公司概览与股东背景

| 项目 | 内容 |
|------|------|
| 证券简称 | 盛美上海 |
| 证券代码 | 688082.SH（科创板） |
| 总市值 | 约 2084.8 亿（2026/6/30） |
| 板块定位 | 半导体清洗设备国产龙头 |
| 母公司 | 盛美半导体（ACMR，美股上市） |
| 数据来源 | Tushare Pro stock_basic |

盛美上海是美国盛美半导体（ACMR）在华子公司，主营半导体清洗设备。其核心技术包括 SAPS（空间交变相移增强）等独家清洗技术，在国产清洗设备领域处于绝对领先地位。

> **核心洞察**：盛美的核心壁垒是"**独家清洗技术 + 已进入全球头部客户供应链**"——其设备已进入三星、SK 海力士等韩国大厂，这是 8 只半导体标的中供应链确定性最高的。母公司 ACMR 美股上市的背景也带来技术与海外客户资源。
