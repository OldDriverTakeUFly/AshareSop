## 第一章：公司概览与卡位定位

### 1.1 创始人与管理层

姚力军，1967年出生，日本归国博士，江丰电子创始人、实际控制人、首席技术官。2005年，姚力军放弃海外优厚待遇，带着"做中国自己的半导体靶材"的信念回到宁波余姚，创办了江丰电子。在此之前，中国高端溅射靶材几乎100%依赖进口，日本JX日矿金属、美国霍尼韦尔等跨国巨头垄断了全球市场。

姚力军是中国半导体材料行业最具技术底色的创业者之一。他不仅是一名管理者，更是一名深耕超高纯金属材料数十年的科学家。2021年，他主导的"超高纯铝钛铜钽溅射靶材制备技术及应用"项目获得国家技术发明二等奖，这是中国半导体材料领域获得的最高荣誉之一。2025年5月，姚力军主动辞去董事长职务，转任首席技术官，将管理权交给总经理边逸军，自己则全身心投入技术攻关。

这一人事调整在中国半导体圈引发广泛关注。58岁的创始人主动退居技术一线，将公司运营交给职业经理人，在中国民营企业中并不多见。这既体现了姚力军对技术的极致追求，也说明江丰电子正在从"创始人驱动"向"专业团队治理"转型。

边逸军，1980年出生，材料科学与工程博士后，高级工程师，2017年起担任江丰电子总经理，2025年5月接任董事长。他在江丰电子深耕近十年，主导了公司从单一靶材向"靶材+零部件"双轮驱动的战略转型，对半导体材料产业链有深刻理解。

**管理层特点：**

| 维度 | 描述 |
|------|------|
| 创始人/实控人 | 姚力军，1967年生，日本归国博士，国家技术发明二等奖获得者 |
| 董事长 | 边逸军，1980年生，材料科学博士后，2017年起任总经理 |
| 管理风格 | 技术驱动+专业化治理，创始人专注技术，职业经理人负责运营 |
| 研发投入 | 2025年研发费用2.62亿元，研发费用率5.69%，全部费用化 |
| 战略视野 | 从靶材单一业务向"靶材+零部件+静电吸盘"平台化拓展 |
| 员工规模 | 约3,000人 |
| 总部 | 浙江宁波余姚 |

> 核心洞察：姚力军是中国半导体材料行业少数兼具顶尖学术能力和产业化能力的创始人。他辞去董事长转任首席技术官，不是退场，而是把最宝贵的时间精力投入到最关键的技术攻坚中。这种"让专业的人做专业的事"的治理理念，在A股半导体公司中相当少见。

### 1.3 主动卡位定位

"主动卡位"意味着企业在某个细分领域拥有不可替代的供应能力和客户认证壁垒，下游客户想换也换不掉，或者换的成本极高。江丰电子在高端溅射靶材领域的卡位，是中国半导体材料产业最典型的主动卡位案例。

**卡位的核心逻辑：耗材模型。**溅射靶材是半导体制造中的消耗品。每一片晶圆在PVD（物理气相沉积）工艺中都需要消耗靶材，一块靶材的寿命通常在几十到几百片晶圆之间。这意味着只要晶圆厂在运转，靶材的需求就是持续且刚性的。与设备不同，耗材不存在"买一台用十年"的情况，而是"天天都要用，用完就要换"。

这种耗材模型赋予了靶材供应商两个独特的优势：第一，客户粘性极强。靶材一旦通过晶圆厂认证进入供应链，更换供应商的风险和成本极高（可能导致良率波动），因此客户通常不会轻易更换。第二，收入持续且可预测。只要晶圆厂在产，靶材订单就不会断。

**江丰电子的五大卡位优势：**

| 卡位维度 | 具体表现 | 卡位强度 |
|----------|----------|----------|
| 客户卡位 | 台积电、SK海力士、中芯国际、联华电子核心供应商 | ★★★★★ |
| 制程卡位 | 全球仅4-5家能供应3nm制程靶材，中国唯一 | ★★★★★ |
| 品类卡位 | 铝/钛/铜/钽/钨全品类覆盖，全球仅少数企业能做到 | ★★★★ |
| 耗材模型 | 靶材是消耗品，晶圆厂每片晶圆都要用，收入持续 | ★★★★★ |
| 国产替代 | 国内12英寸晶圆厂靶材市占率73%，绝对龙头 | ★★★★★ |

**在全球半导体供应链中的位置：**

| 层级 | 角色 | 代表企业 | 江丰电子的位置 |
|------|------|----------|----------------|
| 上游 | 高纯金属原材料 | 同创普润、东方钽业、日矿金属 | 供应商/部分自给 |
| **中游** | **溅射靶材/精密零部件** | **江丰电子、有研新材、日矿金属、霍尼韦尔** | **全球出货量第一** |
| 中游 | 半导体设备 | 北方华创、中微公司、Applied Materials | 零部件供应商 |
| 下游 | 晶圆制造 | 台积电、中芯国际、SK海力士 | 核心靶材供应商 |
| 终端 | 芯片应用 | 苹果、英伟达、华为、特斯拉 | 终端需求驱动 |

> 核心洞察：靶材是典型的"耗材模型"卡位。一台半导体设备卖一次就结束了，但靶材是晶圆厂每天都在消耗的材料。只要AI芯片在制造、只要晶圆厂在运转，靶材的需求就不会停。江丰电子已经进入了台积电3nm供应体系，这意味着每一颗3nm芯片的制造都在消耗江丰的靶材。这种"每一片晶圆都是你的收入"的商业模式，才是真正的卡位。

### 1.4 核心市场地位

截至2025年末，江丰电子在全球半导体溅射靶材市场已建立起无可争议的领导地位。

**核心市场地位数据：**

| 指标 | 数值 |
|------|------|
| 全球半导体靶材出货量排名 | **第1位**（2024年起，富士经济报告） |
| 全球半导体靶材市场份额 | **约25-28%** |
| 国内12英寸晶圆厂靶材市占率 | **约73%** |
| 营收规模（2025年） | **46.04亿元** |
| 先进制程覆盖 | **3nm批量供货，全球仅4-5家** |
| 客户覆盖 | 台积电、SK海力士、中芯国际、联华电子、京东方等 |
| 10年营收增长 | **约18倍**（2015年~2.5亿到2025年46.04亿） |

> 核心洞察：从2.5亿到46.04亿，10年18倍的营收增长，这不是靠一个爆款产品实现的，而是靠"从零到全球第一"的系统性追赶。更重要的是，出货量全球第一不代表技术全球第一，江丰电子在7N+超高纯度和12英寸大尺寸靶材上与日矿金属仍有差距。出货量领先靠的是规模和性价比，技术领先才是下一个目标。

## 第二章：财务分析——双轮驱动下的规模跃迁

### 2.1 核心财务数据（2022-2025）

江丰电子的财务数据讲述的是一家半导体材料公司从追赶者到全球出货量第一的成长故事。2022年到2025年，公司营收从18.75亿元增长到46.04亿元，三年CAGR约35%。净利润从2.53亿元增长到5.00亿元，但净利率始终在10%左右徘徊。

**核心财务数据一览：**

| 指标 | 2022年 | 2023年 | 2024年 | 2025年 |
|------|--------|--------|--------|--------|
| 营业收入（亿元） | 18.75 | 26.02 | 36.05 | **46.04** |
| 营收增速 | +41% | +39% | +38.57% | **+27.72%** |
| 归母净利润（亿元） | 2.53 | 2.55 | 4.01 | **5.00** |
| 净利润增速 | +26% | +1% | +56.79% | **+24.70%** |
| 毛利率 | ~29% | ~29% | ~28% | **27.17%** |
| 净利率 | ~13.5% | ~9.8% | ~11.1% | **~10.9%** |
| 研发费用（亿元） | 1.25 | 1.72 | 2.17 | **2.62** |
| 研发费用率 | ~6.7% | ~6.6% | ~6.0% | **5.69%** |
| 扣非净利（亿元） | 1.83 | 1.56 | 3.04 | **3.60** |
| 扣非净利增速 | — | -15% | +95% | **+18.74%** |

**五年营收复合增速：**

| 区间 | CAGR |
|------|------|
| 2020-2025营收CAGR | ~30% |
| 2020-2025净利润CAGR | ~25% |
| 2023-2025营收CAGR | ~33% |

> 核心洞察：江丰电子的营收增速（27.72%）和净利润增速（24.70%）基本同步，说明增长主要靠规模驱动而非杠杆。但值得警惕的是，毛利率从2022年的29%下降到2025年的27.17%，净利率在10%左右徘徊。这说明公司正处于"以利润率换市场份额"的阶段，精密零部件业务的快速放量（毛利率仅14.88%）拉低了整体毛利。

### 2.2 双轮驱动：靶材与零部件的收入拆解

2025年，江丰电子的收入结构清晰地展示了"靶材+零部件"双轮驱动的格局。

**按产品收入结构（2025年）：**

| 产品 | 收入（亿元） | 占比 | 毛利率 | 同比增速 |
|------|-------------|------|--------|----------|
| 超高纯靶材 | 28.50 | 61.90% | 34.24% | +22.13% |
| 精密零部件 | 10.84 | 23.54% | 14.88% | +22.24% |
| 其他 | 6.70 | 14.56% | 16.98% | +74.19% |

靶材仍然是绝对核心，贡献了62%的收入和更高的利润（毛利率34.24%）。精密零部件虽然毛利率只有14.88%，但增速与靶材持平（22.24%），且品类在快速拓展，是公司的第二增长曲线。

**靶材业务毛利率提升趋势：**

| 年份 | 靶材毛利率 | 变化 |
|------|-----------|------|
| 2023 | ~28-29% | 基准 |
| 2024 | ~31.35% | 提升 |
| 2025 | 34.24% | +2.89pct |

靶材毛利率从2023年的约29%提升到2025年的34.24%，两年提升了约5个百分点。这主要得益于高端靶材（钽靶、铜锰合金靶）占比提升，以及产品结构从铝/钛向铜/钽升级。

**零部件业务毛利率下降原因：**

| 因素 | 影响 |
|------|------|
| 新产品大量试制转放量 | 初期成本高，毛利率被稀释 |
| 品类快速拓展 | 从简单的金属件向陶瓷/树脂件扩展，工艺复杂度增加 |
| 产能爬坡 | 新产线投产初期良率和效率偏低 |
| 2025年靶材涨价潮中零部件成本传导滞后 | 原材料涨价但零部件定价调整有滞后 |

> 核心洞察：靶材毛利率34.24%是历史新高，说明产品结构升级（铜/钽占比提升）正在兑现。零部件毛利率14.88%看着低，但这是品类扩张期的正常现象。随着新产品良率爬坡和规模效应显现，零部件毛利率有望回升至20%以上。当靶材的"高毛利"遇上零部件的"高增长"，双轮驱动的威力才会真正释放。

### 2.3 按地区收入拆解

| 地区 | 收入（亿元） | 占比 | 毛利率 | 同比增速 |
|------|-------------|------|--------|----------|
| 内销 | 30.36 | 65.93% | 24.71% | +40.57% |
| 外销 | 15.68 | 34.07% | 31.93% | +8.51% |

内销增速（40.57%）远高于外销（8.51%），说明国产替代逻辑在国内市场加速兑现。但外销毛利率（31.93%）高于内销（24.71%），这主要因为海外客户以台积电、SK海力士等高端客户为主，产品附加值更高。

**内外销毛利率差异的原因：**

| 因素 | 说明 |
|------|------|
| 客户结构差异 | 外销客户以台积电、SK海力士为主，产品要求更高，定价也更高 |
| 产品结构差异 | 外销以高端铜/钽靶材为主，内销包含更多铝/钛等中端产品 |
| 零部件占比 | 内销中零部件占比更高（国产替代加速），拉低了整体毛利率 |

> 核心洞察：内销增速40.57%说明国产替代正在加速兑现。国内12英寸晶圆厂73%的靶材市占率，意味着江丰电子几乎垄断了国内高端靶材供应。但外销增速只有8.51%，海外市占率还有很大提升空间。韩国工厂的建设正是为了突破海外产能瓶颈，加速对SK海力士、三星等客户的供应。

### 2.4 盈利质量与现金流

| 指标 | 2024年 | 2025年 | 变化 |
|------|--------|--------|------|
| 经营活动现金流（亿元） | -0.96 | 4.70 | 扭转 |
| 净利润（亿元） | 4.01 | 5.00 | +24.7% |
| 现金流/净利润 | 负值 | 94% | 大幅改善 |
| 研发费用（亿元） | 2.17 | 2.62 | +20.6% |
| 研发费用率 | 6.0% | 5.69% | -0.31pct |

2024年经营现金流为负（-0.96亿元），2025年大幅改善至4.70亿元。现金流从负转正，主要得益于销售回款改善和票据支付比例提升。

## 第三章：产品矩阵——从靶材到零部件的技术纵深

### 3.1 超高纯溅射靶材——全球出货量第一

溅射靶材是半导体晶圆制造中物理气相沉积（PVD）工艺的核心材料。简单说，就是把超高纯度的金属做成特定形状的"靶"，在高真空环境中用离子束轰击它，让金属原子溅射出来，均匀地沉积在硅片表面，形成芯片内部精密的金属布线层。靶材的纯度、均匀性和加工精度，直接决定了芯片的性能和良率。

**江丰电子靶材产品矩阵：**

| 产品 | 纯度 | 应用制程 | 技术难度 | 市场地位 |
|------|------|----------|----------|----------|
| 高纯铝及铝合金靶 | 4N-5N5 | 130nm及以下成熟制程 | 中等 | 成熟量产 |
| 高纯钛靶及环件 | 4N5-7N | 130-5nm | 中高 | **7N钛靶领先国际** |
| 高纯铜及铜合金靶 | 4N-6N | 90-3nm | 高 | 量产，全球领先 |
| 高纯钽靶及环件 | 3N5-5N | 90-3nm | **极高** | 全球少数供应商 |
| 高纯钨靶 | — | 存储芯片 | 高 | 多家国际存储厂批量应用 |
| 铜锰合金靶 | — | 90-3nm | **极高** | 全球供应链紧张 |
| HCM靶材 | — | 先进制程 | 极高 | 自主开发，年销售额数千万 |

**各靶材产品详细拆解：**

**铝靶材：**铝及其合金是最广泛使用的半导体芯片配线薄膜材料。芯片制造要求纯度达到99.9995%（5N5）以上。铝靶是江丰电子最早突破的产品，技术成熟，产能充足。

**钛靶材及环件：**钛是最常用的阻挡层薄膜材料，应用于130-5nm工艺。江丰电子的钛靶纯度达到7N（99.99999%），**领先于国际巨头的主流规格6N**。这是一个非常重要的技术里程碑，说明江丰在特定材料上已经超越了日美竞争对手。

**钽靶材及环件：**钽是90-3nm先进制程中必需的阻挡层材料，技术难度最高、品质一致性要求最苛刻的尖端产品。在此之前仅有少数几家跨国公司能生产。特别是钽环件，目前全球只有江丰电子和头部跨国企业掌握核心技术。随着高端芯片需求增长，钽靶材及环件的需求大幅增长，全球供应链极其紧张。

**铜及铜合金靶材：**铜及铜合金是先端半导体导电层薄膜材料，应用于90-3nm制程。铜锰合金靶材制造难度极高，目前只有江丰和头部跨国企业掌握。300mm铜锰合金靶材产量大幅攀升，已批量进入国际著名芯片制造企业。

**钨靶材：**超高纯钨靶已在多家国际知名存储芯片制造商实现批量应用。先端存储芯片用高纯300mm硅靶实现稳定批量供货，这是存储芯片制造中的关键突破。

> 核心洞察：7N级钛靶量产技术领先国际巨头（国际主流6N），这是江丰电子最具技术含量的成果之一。钽靶和铜锰合金靶则是"卡脖子中的卡脖子"，全球只有4-5家能做，江丰是其中之一。在靶材领域，"能做"和"做得好"是两个概念。江丰已经从"能做"跨入了"做得好"的阶段。

**零部件业务与北方华创/中微公司的联动关系：**

| 协同维度 | 说明 |
|----------|------|
| 客户重叠 | 晶圆厂既是设备商的客户，也是靶材/零部件的客户 |
| 产品互补 | 设备+耗材形成完整解决方案 |
| 国产替代同步 | 设备国产化和耗材国产化可以同步推进 |
| 技术协同 | 零部件业务需要理解设备设计，与设备商深度合作 |

> 核心洞察：4万多种可量产零部件，说明江丰电子的零部件业务不是"做几个简单零件"，而是覆盖了半导体设备耗材的全品类。从靶材到零部件，江丰正在从"单一耗材供应商"进化为"半导体耗材平台"。当一台设备上的所有耗材都能从一家供应商采购时，客户的粘性将指数级提升。

2025年，江丰电子通过定增募集资金19.48亿元，其中9.98亿元用于"年产5,100个集成电路设备用静电吸盘产业化项目"。

> 核心洞察：靶材行业真正的技术壁垒不在"加工"，而在"提纯"。高纯金属的提纯技术（5N以上）仍主要掌握在日美企业手中。江丰电子通过同创普润进行上游布局，是一个明智的垂直整合策略。但高纯金属的国产化程度仍然不够，原材料端的自主可控是江丰未来需要持续攻克的课题。

### 7.1 五大增长引擎

**引擎一：AI芯片扩产驱动的靶材需求爆发**

| 维度 | 预期 |
|------|------|
| 驱动力 | AI训练/推理芯片扩产，GPU/HBM存储靶材需求激增 |
| 靶材需求增速 | 先进制程靶材需求CAGR 15%+ |
| 受益产品 | 铜靶、钽靶、钨靶（存储芯片用） |
| 关键客户 | 台积电（3nm扩产）、SK海力士（HBM扩产） |
| 增量空间 | 2025-2030年AI芯片靶材增量需求超100亿元 |

AI芯片对靶材的需求量远超传统芯片。一颗AI GPU需要的硅面积是传统芯片的5-10倍，制造过程中的金属互连层数更多，对铜靶、钽靶等高端靶材的消耗量成倍增长。HBM存储芯片的3D堆叠结构进一步增加了靶材消耗。

**引擎二：靶材国产替代加速**

| 维度 | 预期 |
|------|------|
| 当前国产化率 | ~20%（半导体靶材整体） |
| 2030年目标 | 50%+ |
| 江丰国内市占率 | 73%（12英寸晶圆厂） |
| 增量空间 | 国内靶材市场规模约250亿元，国产替代空间约200亿元 |
| 催化因素 | 政策支持、供应链安全需求 |

国内靶材整体国产化率约20-30%，先进制程靶材国产化率更低。江丰电子在国内12英寸晶圆厂73%的市占率，使其成为国产替代的最大受益者。

**引擎三：精密零部件第二增长曲线**

| 维度 | 预期 |
|------|------|
| 2025年收入 | 10.84亿元 |
| 增长趋势 | CAGR 30%+ |
| 品类拓展 | 从金属件到陶瓷/树脂件，4万多种 |
| 新品突破 | 静电吸盘（2027-2028E量产） |
| 市场空间 | 2027年全球半导体零部件市场约5,428亿元 |

**引擎四：靶材涨价周期**

| 维度 | 预期 |
|------|------|
| 2026Q1涨价幅度 | 20-30% |
| 持续性 | 至少持续2-3个季度 |
| 利润弹性 | 增量收入的50-70%可转化为利润 |
| 对2026年净利润影响 | +30-70% |

**引擎五：全球化产能布局**

| 维度 | 预期 |
|------|------|
| 韩国工厂 | 2027-2028E投产，覆盖SK海力士/三星 |
| 马来西亚工厂 | 已投产，服务东南亚客户 |
| 海外收入占比提升 | 当前34%，目标40%+ |
| 议价能力 | 属地化制造提升海外客户服务能力 |

> 核心洞察：五大引擎中，最被低估的是"靶材涨价周期"。市场普遍关注AI需求和国产替代，却忽略了靶材作为大宗特种材料的周期性涨价能力。2026年Q1靶材涨价20-30%，在产能利用率超过100%的背景下，涨价的利润弹性极大。这不是概念，而是实打实的利润增厚。

### 7.2 硅基消费生态中的联动关系

在硅基消费的产业链中，江丰电子与北方华创、中微公司等设备商形成天然的协同关系。设备出厂后需要配套靶材才能运行，设备运转后需要持续更换耗材（靶材+零部件），这就形成了"设备+耗材"的联动模型。

**联动逻辑：**

| 设备商 | 设备类型 | 配套靶材/零部件 | 江丰供应 |
|--------|----------|----------------|----------|
| 北方华创 | PVD设备 | 铝/铜/钛/钽靶材+Shield/Ring | 全品类覆盖 |
| 中微公司 | 刻蚀设备 | 静电吸盘+气体分配盘 | 静电吸盘开发中 |
| 拓荆科技 | CVD设备 | 靶材+加热盘+气体分配盘 | 部分覆盖 |

**耗材模型的价值：**

| 特征 | 设备 | 靶材（耗材） |
|------|------|-------------|
| 采购频率 | 一次性 | 持续性（每片晶圆消耗） |
| 收入持续性 | 低（取决于新厂建设） | 高（只要晶圆厂在运转） |
| 客户粘性 | 中（设备可替代） | 高（更换供应商风险大） |
| 周期性 | 强 | 弱（耗材需求刚性） |

> 核心洞察：设备商和靶材商的关系不是竞争，而是共生。北方华创卖出一台PVD设备，就意味着一个持续的靶材消耗需求。AI芯片扩产驱动设备采购，设备运转驱动靶材消耗，靶材消耗驱动江丰收入增长。这条传导链是"硅基消费"中最确定的受益路径之一。
