### 2.3 技术代际演进与技术发展方向（强制小节）

| 板块 | 当前技术代际 | 演进路径 | 未来技术布局 |
|------|------------|---------|------------|
| **紫外固化材料** | 第 3 代（高性能低聚物树脂合成） | 单一光纤涂覆 → 多场景 UV 油墨/涂料 | AI 算力驱动的光纤互连景气受益（D8） |
| **屏幕显示材料** | 混合液晶主流代际 | 国产替代 → 收购 JNC 华业务+专利 → 全球供应 | OLED 用材料、新型显示化学品 |
| **半导体材料-光刻胶** | **I线光刻胶（部分客户量产）+ KX 系列晶圆级封装光刻胶** | I线 → KrF → ArF（演进中） | 先进封装光刻胶、KrF/ArF 浸没式 |
| **半导体材料-湿电子化学品** | G2-G3 等级 | 显影/蚀刻/剥离/电镀液全系列 | 先进制程用高纯化学品 |
| **半导体材料-临时键合** | **验证导入期（公司 7/3 口径），三种解绑方案（热力/机械/激光）** | 验证导入 → 中试 → 量产 | **HBM 专用临时键合胶（苏州凯芯基地，预计 2027 初投产，雪球非官方）** |
| **半导体材料-封装料** | 环氧塑封料（EMC）+ 封装锡球（成熟量产） | 传统封装 → 先进封装 | LMC 液态封装料（仍验证中，D8） |

> **核心洞察**：飞凯的技术代际定位是"**成熟产品全面量产 + 先进产品验证导入**"的二元结构。紫外固化（60% 市占率）、液晶（JNC 加持）、封装料（EMC/锡球量产）是已兑现的成熟产能；而 **HBM 临时键合、KrF/ArF 光刻胶、LMC 封装料是仍在验证导入的下一代技术**——后者正是市场此前炒作的"AI/HBM 概念溢价"来源，但公司 7/3 明确承认尚未规模化/未量产，这是当前估值消化的核心矛盾。

### 2.4 半导体材料布局详述（HBM 矛盾厘清）

**官方口径（权威，以公司公告/互动平台为准）**：
- 7/3 互动平台：HBM 材料"**可用于制造工艺但尚未形成规模化营收，对业绩影响较小**"（新京报 20260703）
- 7/3 互动平台：临时键合材料"**正处于验证导入阶段，尚未实现量产**"，生产线正式开工时间待定（每日经济新闻 20260703）
- 6/18 互动平台+年报：I线光刻胶部分客户已量产、湿电子化学品（显影/蚀刻/剥离/电镀）多场景量产、环氧塑封料+封装锡球成熟量产（D4/D8）

**市场传言（非官方，需交叉验证，不作估值依据）**：
- 雪球/股吧：临时键合胶"国产唯一量产、国内市占率约 30%"、"通过 SK 海力士原厂认证"、"供货台积电 CoWoS"、"长鑫存储第一大客户"——**均未经公司公告证实**，公司 7/3 回复与"已量产/市占率 30%"的说法直接矛盾
- 经济观察网 6/19：多款半导体材料已量产，**但临时键合及 LMC 封装料仍处验证导入中**（D8，与公司口径一致）

> **核心洞察**：报告对 HBM/临时键合的判断**严格遵循公司官方口径**——即"验证导入、未规模化"。市场传言之所以盛行，是因为股价从 2025 年底至 2026/6/29 涨幅巨大（一度突破 64 元），资金需要"国产 HBM 唯一标的"的叙事支撑估值。**7/3 公司的谨慎回复等于正式否定了"已量产/市占率 30%"的乐观叙事**，这是暴跌 -13% 的预期破灭本质。本报告估值 strictly 基于官方口径，不将未经证实的传言纳入定价。

### 4.2 G 与 ΔG 的人工核算

- **G（增速绝对值）**：2025 全年净利 +49.6%、2026Q1 净利 +12.5%、2026Q1 营收 +24.6%
- **ΔG（增速边际变化）**：营收同比从 +3.8%（2025H1）→ +7.9%（Q3）→ +10.6%（全年）→ +24.6%（2026Q1），**逐季加速，ΔG 为正且扩大**
- **engine delta_g = +8.34**（与人工核算方向一致）

### 4.3 山峰理论定位

山峰理论按 ΔG 符号定周期位置：
- ΔG > 0 → **左山坡上行段**（增速加快）
- ΔG < 0 → 右山坡下行段（增速放缓）
- G 接近 0 → 山后（衰退）

**飞凯定位**：ΔG +8.34 > 0 且逐季扩大 → **左山坡上行段（上升拐点确认）**。

### 4.4 30% 阈值检验

| 指标 | 数值 | 阈值 | 判断 |
|------|:---:|:---:|------|
| 2026Q1 净利增速 | +12.5% | >30% | ❌ 低于 |
| 2026Q1 营收增速 | +24.6% | >30% | ❌ 接近但低于 |
| 2025 全年净利增速 | +49.6% | >30% | ✅ 超过 |

> **核心洞察**：飞凯是"稳健加速"而非"高速爆发"。2025 全年净利增速 49.6% 超 30% 阈值，但 2026Q1 因基数效应降至 12.5%。按 industry-prosperity skill，**净利增速降至 30% 以下时超额收益大概率下滑**——当前 2026Q1 已低于阈值，提示短期估值溢价消化压力大，这与 7 月暴跌的市场表现一致。

### 4.5 二次点火筛选

二次点火筛选条件：G > 0 且 ΔG > 0（增速为正且边际改善）。

**飞凯结果**：G > 0（净利+营收均正增长）+ ΔG +8.34 > 0 → **满足二次点火筛选条件**。

> 但需注意：二次点火是"景气度信号"，不等于"股价必涨"——还需结合估值（§8）与筹码（§3.5）综合判断。飞凯当前景气度信号成立，但估值偏高+筹码分散构成对冲。

## 七、未来发展规划与战略路径

### 7.2 隐含的战略逻辑

> **关键风险点**：半导体材料（HBM/临时键合）是公司从未做过"从 0 到 1 量产"的新方向——既往的成功（液晶/封装料）都是买成熟业务，而 HBM 临时键合需要自研+客户认证+量产爬坡，路径不同，不确定性更高。这也是 7/3 公司谨慎回复的内在逻辑。
