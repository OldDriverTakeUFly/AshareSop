## 一、ADC 是什么——"生物导弹"原理

```
ADC = 抗体（Antibody）+ 连接子（Linker）+ 毒素（Payload）
       ↑                    ↑                    ↑
   精准靶向肿瘤          连接桥梁           强力杀伤（化疗毒素）
   （GPS 导航）          （焊条）           （弹头）
```

**ADC 的革命性**：传统化疗"敌我不分"（杀死癌细胞也杀死正常细胞），ADC 通过抗体精准将毒素送到肿瘤细胞——**疗效提升 + 副作用降低**。

### 技术迭代

| 代际 | 特征 | 代表 |
|------|------|------|
| 第一代 | 不稳定、毒性大 | Mylotarg（撤市后重批） |
| 第二代 | 改良连接子 | Adcetris/Kadcyla |
| **第三代** | **定点偶联+新毒素（TOPO1i）** | Enhertu（DS-8201，革命性） |
| **下一代** | **双抗 ADC+新型连接子** | 百利天恒 BL-M071D |

## 二、产业链结构

```
上游：抗体 + 毒素（Payload）+ 连接子（Linker）原材料
  ↓
中游：
  ├── ADC CDMO（药明合联/凯莱英）——生产外包
  └── ADC 创新药企（百利天恒/科伦博泰/荣昌/恒瑞）——研发
  ↓
下游：医院/患者（肿瘤治疗）
```

## 四、中国为什么全球第一

| 因素 | 说明 |
|------|------|
| **工程师红利** | 大量抗体工程/化学偶联人才 |
| **药监局改革** | 2015 后创新药审评加速 |
| **BD 模式成熟** | License-out→首付款→MNC 扫货 |
| **技术追赶完成** | 从跟随 Enhertu 到原创双抗 ADC |
| **产能优势** | 药明合联 ADC CDMO 全球龙头 |
