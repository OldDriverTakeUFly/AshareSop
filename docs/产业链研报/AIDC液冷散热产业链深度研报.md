## 第一章：液冷技术——AI算力的散热刚需

### 1.1 为什么AIDC必须液冷？

AI大模型训练对算力的需求呈指数级增长，GPU单芯片功耗已从2023年H100的700W飙升至2026年Rubin的2300W。传统风冷散热的物理极限是15-25kW/机柜——空气的体积热容仅为水的3300分之一，100kW机柜若用风冷需在柜内产生飓风级气流。当机柜功率超过40kW，风冷已完全不可行。

**液冷不是"可选项"，而是"刚需"——这是物理定律决定的。**

| 对比维度 | 风冷极限 | 液冷能力 |
|---------|---------|---------|
| 最大机柜功率密度 | 15-25 kW/柜 | 冷板80-132 kW/柜；浸没200+ kW/柜 |
| PUE下限 | ~1.3（实测1.3-1.6） | 冷板1.05-1.15；浸没1.01-1.08 |
| GPU结温控制 | 85-95°C | 40-65°C |
| 噪音 | >90 dB | 显著降低 |
| 水耗 | 2-5百万加仑/MW/年 | 封闭循环/近零 |

**GB200 NVL72**单机柜功耗达132kW，是风冷极限的5-8倍；**Rubin NVL72**达190-230kW，**Rubin Ultra（2027）目标600kW/柜**——液冷已从"优化选项"变为"架构强制需求"。

**数据来源**：ToneCooling、Castle Rock Digital、NVIDIA官方规格

### 1.2 液冷技术分类体系

液冷技术按冷却液是否与电子元件直接接触，分为**非接触式**（冷板式）和**接触式**（浸没式、喷淋式）两大类：

| 技术路线 | 冷却液接触元件 | 典型方案 | 液冷应用占比 | PUE | 最大机柜密度 |
|---------|:---:|------|:---:|------|------|
| **冷板式（DLC/DTC）** | ✗ | 铜微通道冷板 + 水乙二醇 | >90% | 1.05-1.15 | 80-132 kW/柜 |
| **单相浸没式** | ✓ | 介电液体（合成油/氟化液） | ~8% | 1.02-1.08 | 100-200+ kW/柜 |
| **两相浸没式** | ✓ | 低沸点氟化液（相变） | ~2% | 1.01-1.05 | 250+ kW/柜 |
| **喷淋式** | ✓ | 介电液体喷射 | <1% | ~1.05 | 100+ kW/柜 |
| **背门热交换器（RDHx）** | ✗ | 水冷盘管替代机柜后门 | 补充方案 | 1.2-1.4 | 30-50 kW/柜 |

**核心结论**：冷板式液冷占液冷数据中心应用超90%，是当前绝对主流，原因是技术成熟度最高、与现有服务器架构兼容性好、改造成本低。NVIDIA官方推荐冷板式方案。

### 1.3 液冷 vs 风冷全面对比

| 对比维度 | 风冷 | 冷板式DLC | 单相浸没 | 两相浸没 |
|---------|------|----------|---------|---------|
| **散热能力（kW/柜）** | 15-25 | 80-132 | 100-200+ | 250+ |
| **PUE** | 1.3-1.6 | 1.05-1.15 | 1.02-1.08 | 1.01-1.05 |
| **GPU结温（°C）** | 85-95 | 50-65 | 45-55 | 40-50 |
| **初始CapEx（$/kW）** | $1,800-3,200 | $3,500-5,000 | $4,300-6,500 | $6,000-9,500 |
| **运行OpEx/kW** | 高 | 低 | 极低 | 最低 |
| **噪音** | >90 dB | 中 | 近零 | 近零 |
| **占地面积** | 基准 | -30~45% | -60~75% | -60~75% |
| **水耗** | 2-5M加仑/MW/年 | 封闭循环 | 近零 | 近零 |
| **技术成熟度** | 成熟 | 最成熟 | 新兴 | 实验性 |
| **改造复杂度** | 基准 | 高 | 极高 | 极高 |
| **10年TCO（64柜AI集群）** | $42M | $31M | $28M | 受PFAS影响 |

**关键洞察**：浸没式TCO优势最大（vs风冷节省20.3%），但冷板式改造门槛最低。液冷的TCO优势在电价>$0.10/kWh、机柜密度>30kW时显著——2026年AI数据中心平均机柜密度约27kW，已越过拐点。

**数据来源**：Adam Silva Consulting、Energy Solutions Intelligence、NVIDIA/Vertiv PUE研究

### 2.2 英伟达GB200液冷架构详解


**核心组件详解**：

**NVIDIA供应商认证要求**：CFD热仿真验证、7-15天原型交付（MOQ 5片）、12 bar爆破压力测试、PPAP Level 3文档、±5%流量均衡验证。

### 2.3 Rubin/Rubin Ultra的散热革命

| 参数 | Blackwell GB300 NVL72 | Vera Rubin NVL72 |
|------|---------------------|-----------------|
| 单GPU功耗 | ~1400W | ~2300W |
| 机柜功耗 | ~150 kW | **190-230 kW** |
| 冷却方式 | 80%+液冷 | **100%液冷，无风冷选项** |
| 进液温度 | 32-45°C | **45°C（高温进液）** |
| 冷却液流量 | 30-40 L/min | 45-60 L/min |
| 供电架构 | 48V | **800V DC** |
| 电源架构 | 集成电源 | **独立Power Rack** |

**Rubin的五大革命性变化**：

1. **45°C进液温度**：可利用更高温度冷却水，多数气候条件下**无需冷水机组**，大幅延长自然冷却时间
2. **100%液冷**：完全消除风扇，机柜内气流需求降低~80%，电源模块移至独立Power Rack
3. **800V DC供电**：减少铜缆重量200kg/柜，端到端减少5%电力损耗
4. **模块化托盘设计**：5分钟安装（Blackwell需2小时）
5. **回水温度高达65°C**：可直接用于区域供暖

**Rubin Ultra（2027）目标600kW/柜**：需要更先进的两相液冷技术，可能推动封装级液冷（MCL）加速落地。

**数据来源**：NVIDIA Vera Rubin平台规格、NVIDIA Blog OCP、gandgcontrols、AI Consulting Network

## 第三章：电源液冷——被忽视的第二战场

### 3.1 电源液冷的需求来源

电源液冷是AIDC液冷产业中**被严重低估但至关重要的维度**。随着机柜功耗从几十kW跃升至数百kW，电源系统（SST、HVDC、UPS、PCS、PSU）的热管理需求同样剧增。

| 电源类型 | 功率范围 | 热耗（效率损耗） | 液冷必要性 |
|---------|---------|----------------|---------|
| **SST（固态变压器）** | 200kW-5MW | 1-4%×额定 = 数十kW | 高（室内部署） |
| **HVDC整流模块** | MW级 | ~3%×MW级 | 高 |
| **UPS** | 传统7-13%损耗 | 高 | 被SST替代趋势 |
| **PCS（储能变流器）** | 450kW/单元 | ~0.7% | 高（阳光电源已液冷） |
| **PSU（服务器电源）** | 3-6.6kW/单元 | 4-10% | 快速提升 |

**核心驱动力**：GB200液冷覆盖率仅70-80%（电源仍风冷），**Rubin平台将首次实现100%液冷覆盖（含电源）**，创建独立Power Rack。

### 3.2 SST固态变压器液冷

SST是下一代AI数据中心的核心电力设备，直接将中压交流电（13.8-34.5kV）转换为800V DC，替代传统变压器+UPS的多级转换架构。

| 厂商 | 产品 | 功率 | 冷却方案 | 半导体技术 |
|------|------|------|---------|---------|
| **Enphase** | IQ SST | 1.25MW/柜 | 室外风冷/室内液冷选项 | GaN HEMT |
| **DG Matrix** | Interport Flex | 200-400kW/单元 | 模块化液冷 | Infineon SiC |
| **SolarEdge** | SST | 2-5MW模块化 | 液冷设计 | SiC |
| **Heron Power** | Heron Link | MV级模块化 | 液冷 | HV SiC MOSFET |
| **华为数字能源** | 全链供电Power POD | 3.2MW | 液冷 | 4T融合技术 |

**SST液冷的核心意义**：
1. SST效率虽高（96-99%），但兆瓦级设备仍有10-15kW热耗
2. 功率密度极高（体积仅为传统变压器的1/15），散热需求集中
3. 室内部署需要液冷避免热量进入白空间
4. SST比传统UPS节省约8.5%输入能量

**产业节奏**：2026年验证送样→2027年小规模导入→2028年后批量落地。市场空间预计500-1000亿元。

**数据来源**：Enphase白皮书、DG Matrix白皮书、Wolfspeed、国金证券SST研报

### 3.5 服务器液冷 vs 电源液冷技术差异

| 维度 | 服务器液冷（GPU/CPU） | 电源液冷（SST/HVDC/PCS） |
|------|---------------------|------------------------|
| **热流密度** | 极高（500-1500W/cm²芯片级） | 中高（功率器件级分布） |
| **冷却精度** | 结温控制±1-2°C | 温差控制±2.5°C |
| **冷却液类型** | 水乙二醇/去离子水（冷板）；氟化液（浸没） | 水乙二醇（主流） |
| **冷却温度** | 进液30-45°C，回液55-65°C | 环境温度到60°C |
| **流量需求** | 2-3 L/min/模块（精细控制） | 大流量（系统级循环） |
| **安全性要求** | 极高（漏水损坏昂贵GPU，万元级/卡） | 高（电力设备，可切换冗余） |
| **标准化程度** | OCP/ASHRAE标准成熟 | 多样化（各厂商自定义） |
| **功率级别** | 单柜100-600kW | 单设备200kW-5MW |
| **散热占比** | 机柜功耗100%需要管理 | 设备功耗的1-4%（效率损耗热） |
| **技术路线** | 冷板式为主（>90%） | 强制风冷→液冷过渡中 |
| **维护频率** | 定期检查，冷却液更换 | 长周期维护 |

**核心结论**：服务器液冷的核心挑战是**超高热流密度的精确温控**，电源液冷的核心挑战是**大功率系统的可靠散热与环境适应性**。两者技术路线不同但正在趋同——Rubin Ultra 600kW/柜将迫使电源和计算使用统一液冷架构。

## 第四章：CDU与液冷基础设施

### 4.1 CDU（冷量分配单元）

CDU是液冷系统的**核心枢纽**，连接IT设备冷却回路与设施冷却回路。

**核心功能**：
- 热交换（钎焊板式换热器隔离一/二次回路）
- 泵送（冗余泵+自动切换）
- 温度控制（维持GPU进液温度稳定）
- 压力控制（6-8 bar一次回路）
- 泄漏检测与紧急关断
- 流量监控与均衡

**CDU功率分类**：

| CDU类型 | 功率范围 | 部署方式 | 适用场景 |
|---------|---------|---------|---------|
| 机柜级CDU | 50-200kW | 底部/顶部安装 | 单机柜/小集群 |
| 行级CDU | 200-600kW | 行间独立柜 | 4-8机柜行 |
| 设施级CDU | 600kW-2MW+ | 独立机房/模块化 | 大规模集群 |

**GB200 NVL72典型配置**：Vertiv XDU600（600kW/skid，4柜）、Chilldyne CDU-1500（1.5MW）、Delta L2L CDU（2000kW）、DCX ECDU（600kW-2.6MW）。

### 4.2 冷板技术

**技术要求**：
- 材料：高纯度铜（C1100/C1020），真空钎焊/扩散焊接
- 热阻：<0.1°C/W（英伟达要求<0.03°C/W）
- GPU冷板仅用铜材（铝材存在电化学腐蚀风险）
- 微通道翅片：厚度≤0.5mm，间距≤0.5mm，翅高≥3mm
- 接触面：平面度≤0.05mm，粗糙度Ra≤0.8μm

**壁垒分析**：冷板占液冷系统成本40-45%，定制冷板毛利率40-60%（普通冷板20-30%）。技术壁垒在于微通道精密加工（CNC/真空钎焊/扩散焊接），高纯度铜材料依赖。

**技术演进**：从传统铜铝→复合材料→微通道（MLCP）→3D流道。**两相冷板**（热流密度385W/cm²，适配3kW级芯片）是下一代方向。

**供应商**：全球Tier 1为CoolIT、Vertiv、Boyd、AVC、Cooler Master；中国为英维克、曙光数创、精研科技、飞荣达。

### 4.3 液冷快接头（UQD）

UQD是液冷系统中"断点风险"最高的零件，决定系统能否稳定运行。

**技术要求**：双关断平面阀设计、微米级公差、>10000次（英维克达200万次）插拔零泄漏、PEEK特种工程塑料配方。

**全球格局**：

| 供应商 | 地位 |

| **Stäubli（史陶比尔）** | 传统龙头 |
| **Danfoss（丹佛斯）** | 英特尔UQD互插联盟成员 |
| **Amphenol** | QD公端，连接器巨头 |
| **英维克** | **中国UQD龙头**，NV MGX生态，OCP认证，自研率>90% |

**数据来源**：Valuates Reports、天风证券、hansenfluid.com

### 4.4 漏液检测系统

漏液检测是液冷系统的安全保障，核心功能包括：
- 实时监测管路、接头、冷板的泄漏状态
- 检测到泄漏后紧急关断（<1秒响应）
- 定位泄漏点（精度至分支级别）
- 与CDU联动自动隔离

主要供应商：Chilldyne（负压防漏技术）、Vertiv、各CDU厂商集成方案。

## 第八章：投资视角与未来前景

### 8.1 液冷渗透率提升的确定性

**"卖铲子"逻辑——不赌芯片赢家，赌液冷渗透率提升。**

无论哪家AI芯片胜出（英伟达/AMD/Google TPU/国产算力），液冷都是刚需。液冷需求直接取决于核心客户基础设施建设进度，无需受制于下游AI软件应用的商业化落地情况。

**渗透率提升的确定性来自三重驱动**：

1. **物理定律驱动**：芯片功耗超300W、机柜超50kW时风冷已达天花板——这是不可逆的
2. **英伟达强制驱动**：GB200强制液冷→Rubin 100%液冷→Rubin Ultra 600kW/柜，每一代需求翻倍
3. **政策驱动**：中国新建大型数据中心PUE≤1.25，风冷实测1.3-1.6，液冷1.05-1.15——合规必选项

**渗透率数据**：
- AI训练服务器液冷渗透率：2024年15%→2026年74%→2027年80%（高盛）
- 中国整体液冷渗透率：2026年37%→2030年82%
- Goldman Sachs预计2026年底**76%的AI服务器**将采用液冷

### 8.3 产业链传导路径

**时间差逻辑**：上游材料→中游组件→下游集成，各环节放量节奏不同。

1. **上游材料（最先受益）**：氟化液、铜材、密封材料——3M退出立即创造替代需求
2. **中游组件（核心受益）**：冷板、CDU、UQD——GB200/Rubin放量直接驱动
3. **下游集成（最后受益）**：液冷系统方案、数据中心基础设施——建设周期12-18个月

**价值量集中趋势**：产业链价值向上游高壁垒环节（CDU、UQD、氟化液）集中，毛利率CDU 40-60% > UQD 40-50% > 冷板 20-60% > Manifold 30-35%。

## 第九章：总结

### 9.1 一句话定位

**AIDC液冷散热是AI算力时代的"水电煤"——不是选择题，而是必答题，且需求将随每一代GPU功耗翻倍而持续爆发。**

### 9.2 核心结论

1. **液冷已跨过拐点，进入加速渗透期**：2026年AI机柜平均密度27kW已超过30kW TCO拐点，Goldman Sachs预计2026年底76% AI服务器将液冷。冷板式DLC是当前绝对主流（>90%），Rubin之前不会改变。

2. **英伟达是最大驱动力，每一代需求翻倍**：GB200强制液冷（132kW/柜）→ Rubin 100%液冷（190-230kW/柜）→ Rubin Ultra（600kW/柜）。从B200开始液冷不再是选项而是架构需求。

3. **电源液冷是被严重低估的第二战场**：SST（1.25MW/柜）、PCS（450kW/单元全液冷SiC，99.3%效率）、800V DC架构、液冷PSU（6600W）都在驱动电源系统液冷需求。Rubin平台将首次实现100%液冷覆盖（含电源）。

4. **3M退出创造百亿国产替代空间**：巨化股份（5000吨/年规划）、新宙邦（台积电2nm认证）、长芦新材料（300吨试车成功）等加速替代。预计2-3年后国产高端电子级氟化液市占率从<10%提升至40%+。

5. **A股标的需区分产业链环节和业绩兑现节奏**：曙光数创（纯液冷弹性最大）、英维克（全链条+NV MGX）、申菱环境（CDU第一）、麦格米特（唯一NV AI电源）各有定位。当前PE普遍较高（70-110倍），业绩兑现节奏是核心变量。
