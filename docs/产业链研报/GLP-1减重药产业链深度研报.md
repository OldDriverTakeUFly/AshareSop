## 一、GLP-1 是什么——减重"神药"原理

```
GLP-1（胰高血糖素样肽-1）：
  天然激素 → 调节血糖 + 抑制食欲 + 减缓胃排空
    ↓
GLP-1 药物：
  长效 GLP-1 受体激动剂 → 降糖 + 减重（"神药"）
    ↓
减重机制：
  ① 抑制食欲（大脑）→ 少吃
  ② 延缓胃排空 → 饱腹感
  ③ 增加能量消耗
```

**减重效果**：司美格鲁肽 68 周平均减重 **15%**，替尔泊肽最高减重 **22.5%**——接近胃旁路手术效果。

## 二、产业链结构

```
上游：多肽原料药（API）+ 注射笔/给药装置
  ↓
中游：
  ├── 创新药（信达/恒瑞/华东）——原研 GLP-1
  ├── 生物类似药（联邦/华东）——司美格鲁肽仿制
  └── 多肽 CDMO（药明/凯莱英）——生产外包
  ↓
下游：医院/药店/线上 → 糖尿病+减重消费医疗
```

## 三、全球市场：500 亿美元双雄争霸

| 维度 | 数据 |
|------|:---:|
| 2025 全球 GLP-1 市场 | **~500 亿美元** |
| 司美格鲁肽（诺和诺德） | **~361 亿美元** |
| 替尔泊肽（礼来） | **~365 亿美元** |
| 2030E 全球减重药 | **$1000-1500 亿** |
| 减重市场属性 | 消费医疗（自费为主，复购属性强） |

> **核心洞察**：GLP-1 是全球增长最快的药品品类——5 年内从几十亿暴增至 500+ 亿美元，且减重需求远未饱和（全球肥胖人口 10 亿+）。减重药的消费属性使其更像"消费品"而非"药品"——复购率高、价格弹性大。
