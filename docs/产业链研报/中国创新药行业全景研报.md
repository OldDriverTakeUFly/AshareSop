## 一、中国创新药发展全景：三个阶段

| 阶段 | 时间 | 特征 | 标志 |
|------|------|------|------|
| **仿制期** | 2000-2015 | 仿制药为主，创新极少 | 仿制药一致性评价 |
| **跟随创新** | 2015-2022 | Fast-follow，PD-1 爆发，药监局改革 | 百济/信达/君实 PD-1 上市 |
| **全球首创** | **2023-至今** | License-out 爆发，双抗/ADC/CAR-T 全球领先 | **2025 年 BD 交易 1357 亿美元** |

> **核心洞察**：中国创新药已进入第三阶段——不再是"跟随西方"，而是"全球药企来中国扫货"。2026Q1 MNC（跨国药企）与中国企业完成 17 笔合作，总金额 480 亿美元，占当季 License-out 的 **81%**。

## 三、方向详解

### 方向 1：ADC（抗体偶联药物）——中国全球领先

| 维度 | 数据 |
|------|------|
| 原理 | 抗体（精准靶向）+ 连接子 + 毒素（杀伤）= "生物导弹" |
| 全球市场 | ~$150 亿+，CAGR >30% |
| 中国地位 | **在研 ADC 管线占全球约 50%** |
| 代表交易 | 百利天恒-BMS **84 亿美元**（双抗 ADC） |
| 代表企业 | 百利天恒、科伦博泰、恒瑞、荣昌生物 |
| 投资逻辑 | 管线丰富+BD 交易持续放量 |

### 方向 2：双抗（含双抗 ADC）——中国在研管线全球第一

| 维度 | 数据 |
|------|------|
| 原理 | 同时靶向两个抗原，增强疗效/安全性 |
| 中国地位 | **全球在研双抗管线约一半在中国** |
| 代表交易 | 三生制药-辉瑞 **60.5 亿美元**（PD-1/VEGF 双抗） |
| 下一代 | 双抗 ADC（ADC+双抗结合）= 最热前沿 |
| 代表企业 | 康方生物、三生制药、百利天恒、信达 |

### 方向 3：PD-1/PD-L1——红海但仍是基石

| 维度 | 数据 |
|------|------|
| 全球市场 | ~$500 亿（K 药 250 亿/年） |
| 中国现状 | **6+ 家国产 PD-1 上市，内卷严重**（年费从 20 万降至 3-5 万） |
| 出路 | ① 联合用药（PD-1+ADC/靶向）② 出海（License-out）③ 新适应症 |
| 投资逻辑 | 管线内卷=不利，但作为" backbone"联合用药有价值 |

### 方向 4：CAR-T 细胞治疗——追赶中

| 维度 | 数据 |
|------|------|
| 原理 | 提取患者 T 细胞→基因改造→回输杀癌 |
| 全球市场 | ~$30 亿，CAGR >25% |
| 中国地位 | 多款上市（复星凯特/药明巨诺），但商业化不及预期 |
| 瓶颈 | ① 价格极高（100 万+/次）② 自体定制（产能受限） |
| 下一代 | 通用型 CAR-T（off-the-shelf）+ 实体瘤 CAR-T |
| 代表企业 | 复星凯特、药明巨诺、科济药业 |

### 方向 5：GLP-1 减重/代谢——最大的增量市场

| 维度 | 数据 |
|------|------|
| 全球市场 | ~$500 亿+（诺和诺德司美格鲁肽+礼来替尔泊肽） |
| 中国机会 | ① 仿制/生物类似药 ② 口服制剂 ③ 多靶点激动剂 |
| 减重市场 | 全球减重药市场预计 2030 年达 **$1000-1500 亿** |
| 中国企业 | 信达（玛仕度肽）、恒瑞（HRS9531）、华东医药（利拉鲁肽生物类似药） |
| 投资逻辑 | 减重是消费医疗=需求弹性巨大+复购属性 |

### 方向 6：自免（自身免疫）

| 维度 | 数据 |
|------|------|
| 全球市场 | ~$300 亿（修美乐曾年销 200 亿） |
| 中国机会 | ① 生物类似药（修美乐/司库奇尤单抗）② TCE 双抗扩展自免 |
| 特征 | 慢性病=长期用药=复购属性强 |
| 代表企业 | 三生国健、康方生物（自免适应症拓展） |

### 方向 7：小核酸/核酸药物——新兴前沿

| 维度 | 数据 |
|------|------|
| 原理 | ASO/siRNA/mRNA 直接调控基因表达 |
| 全球市场 | ~$20 亿（快速成长） |
| 中国地位 | **起步阶段，被列为 2026 重点方向** |
| 代表企业 | 圣诺医药（RNAi）、瑞博生物（siRNA） |
| 投资逻辑 | 新技术平台=高壁垒+长尾效应 |

### 方向 8：AI 制药——从概念到商业化

| 维度 | 数据 |
|------|------|
| 现状 | MNC 在 AI 制药领域合作超 30 项，总价值超 100 亿美元 |
| 2026 拐点 | **从"概念验证"进入"临床验证"** |
| 瓶颈 | 临床成功率仅 12%（AI 有望提升至 14%） |
| 投资逻辑 | 平台型公司（技术授权）vs 管线型公司（自研药物） |
| 风险 | 商业化路径未验证 |

### 方向 9：CXO 产业链——确定性受益

| 维度 | 数据 |
|------|------|
| 全称 | CRO（研发外包）+ CMO/CDMO（生产外包） |
| 中国地位 | **全球第一（产能+成本优势）** |
| 2025 业绩 | 22 家 CXO 上市公司收入 +13.4%，归母净利 **+85.3%** |
| 投资逻辑 | 无论哪个创新药方向爆发，CXO 都受益（"卖铲人"） |
| 代表企业 | 药明康德、药明生物、凯莱英、泰格医药 |

### 方向 10：中药创新药——独有赛道

| 维度 | 数据 |
|------|------|
| 政策 | 国家鼓励中药创新，审评审批加速 |
| 市场 | ~$100 亿（中国独有） |
| 特征 | 政策驱动+消费属性+品牌壁垒 |
| 代表企业 | 以岭药业、天士力、康缘药业 |

## 六、投资逻辑与风险

### 6.1 核心投资逻辑

| # | 逻辑 |
|---|------|
| L1 | **BD 出海验证全球竞争力**——License-out 首付款堪比 IPO |
| L2 | **双抗/ADC 全球管线第一**——中国从"跟随"变"引领" |
| L3 | **GLP-1 减重是消费医疗**——需求弹性+复购属性 |
| L4 | **CXO 是"卖铲人"**——无论谁赢都受益 |
| L5 | **AI 制药 2026 拐点**——从概念到临床验证 |

### 6.2 风险

| # | 风险 |
|---|------|
| R1 | 集采压价（PD-1 从 20 万降至 3-5 万） |
| R2 | 临床失败率（创新药 inherent 风险） |

## 七、如何选择细分方向（供讨论）

### 7.1 按"确定性×弹性"矩阵

```
          高确定性
             ↑
     CXO     |    ADC/双抗
   (卖铲人)   |  (全球领先)
              |
  ────────────┼──────────→ 高弹性
              |
   中药创新   |    GLP-1 减重
  (政策独有)   |  (消费医疗)
              |
          低确定性
```

### 7.2 推荐讨论方向

| 方向 | 为什么值得讨论 |
|------|----------------|
| **ADC/双抗** | 中国全球第一，BD 交易最大（84 亿美元），确定性+弹性兼备 |
