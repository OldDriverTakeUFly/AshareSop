## 一、光通信制造设备 vs 测试设备——概念边界

| 维度 | 制造/组装设备（本报告） | 测试设备（既有报告已覆盖） |
|------|------------------------|--------------------------|
| **功能** | **制造**光模块/光芯片——固晶、耦合、封装、老化 | **测量**成品性能——BER、眼图、65GHz 采样 |
| **工艺位置** | 产线中段（芯片→模块） | 产线末端（模块/系统验收） |
| **代表标的** | 罗博特科/ficonTEC、科瑞技术、博众精工、新益昌、快克智能 | 联讯仪器、Keysight、Anritsu |
| **价值占比** | 耦合 ~40% + 固晶 ~20% + 封装 ~12% + 老化 ~12% | 测试仪表 ~15% |
| **既有覆盖** | **零深度覆盖**（新益昌/快克仅一行带过） | 深度覆盖（联讯仪器两篇） |

> **核心洞察**：制造设备和测试设备是完全不同的赛道。测试设备的壁垒在"高速信号测量精度"（65GHz 采样），制造设备的壁垒在"亚微米级光学对准精度"（5nm 级主动对准）。两者的客户群、技术路线、竞争格局都不同。**本报告专攻制造设备这个既有报告的盲区。**

## 二、完整生产线工艺流程与设备地图

光模块/光芯片从晶圆到成品的完整制造流程，每一步对应特定设备：

```
晶圆 → [晶圆级测试 KGD] → 切片 → [固晶/贴片] → [共晶] → [光纤耦合/对准] → [键合] → [封装/盖板] → [老化/筛选] → [终检] → 成品
         ↑                    ↑          ↑           ↑              ↑        ↑            ↑             ↑          ↑
      ficonTEC/             新益昌/     ficonTEC/   ficonTEC/      快克/     博众/        联讯/         联讯/
      Onto/Camtek           ASMPT       ASMPT       ficonTEC       ASMPT    科瑞         定制          Keysight
```

### 各环节设备详解

| 工序 | 功能 | 技术壁垒 | A 股标的 | 国际龙头 |
|------|------|:---:|----------|----------|
| **晶圆级测试（KGD）** | 封装前筛选合格芯片（CPO 必须，报废成本极高） | ★★★★ | —（联讯是模块级非晶圆级） | Onto Innovation、Camtek、ficonTEC |
| **固晶/贴片** | 激光器/探测器芯片贴装到基座 | ★★★ | 新益昌(688323)、凯格精机(301338) | ASMPT(0522.HK)、BESI |
| **共晶** | 高精度低阻合金键合贴装 | ★★★★ | **罗博特科/ficonTEC** | ASMPT、BESI |
| **光纤耦合/对准** | 光纤阵列与光芯片亚微米级对准（**最高价值~40%**） | ★★★★★ | **罗博特科/ficonTEC**、科瑞技术(002957) | ficonTEC、Mpi、Fujikura |
| **键合** | 金线/铜线电气互连 | ★★ | 快克智能(603203) | ASMPT、K&S |
| **封装/盖板** | 气密性/COB 封装 | ★★★ | 博众精工(688097)、科瑞技术(002957) | ASMPT |
| **老化/筛选** | 100% 老化（CPO 不可现场更换，必须全检） | ★★★ | —（联讯有老化系统） | 定制厂商 |
| **整线集成** | 交钥匙自动化产线（供中际旭创/新易盛） | ★★★★ | **罗博特科/ficonTEC**、博众精工、科瑞技术 | ASMPT |

## 四、光纤耦合——最高价值最高壁垒环节

### 4.1 为什么耦合是最高壁垒

光纤耦合/对准是将光纤阵列（FAU）与硅光芯片/光引擎进行**亚微米级（<1μm）六自由度主动对准**。对准精度直接决定光耦合效率（插入损耗），差 0.1μm 可能导致损耗翻倍。

| 技术指标 | 要求 |
|----------|------|
| 对准精度 | **亚微米级（<1μm），先进达 5nm 级** |
| 自由度 | 六自由度（x/y/z/θx/θy/θz）主动对准 |
| 耦合效率 | 插入损耗 <1dB |
| 速度 | 全自动（CPO 量产要求） |

### 4.2 供应商格局——近垄断

| 供应商 | 硅光封装份额 | 800G 全自动封装份额 | 地位 |
|--------|:---:|:---:|------|
| **ficonTEC（罗博特科）** | **~55%** | **>80%** | **近垄断** |
| ASMPT | ~15% | ~10% | 第二梯队 |
| Mpi（德国） | ~10% | — | 细分 |
| Fujikura | ~5% | — | 细分 |
| 其他 | ~15% | ~10% | 长尾 |

### 5.2 罗博特科 vs 新益昌 vs 快克智能——差异化

| 维度 | 罗博特科(300757) | 新益昌(688323) | 快克智能(603203) |
|------|:---:|:---:|:---:|
| 核心工序 | **光学耦合/对准**（最高价值） | 固晶/贴片 | AOI 检测/焊接 |
| 技术来源 | **收购德国 ficonTEC** | 自研 | 自研 |
| 全球地位 | 硅光封装 #1（~55%） | 固晶国产龙头 | AOI 细分 |
| CPO 覆盖 | 全流程 | 固晶环节 | 检测环节 |
| 客户 | Intel/Broadcom/NVIDIA/TSMC/中际旭创 | LED/半导体 | 3C/半导体 |

> **核心洞察**：三者分工不同——新益昌"把芯片贴上去"，快克智能"检查贴得好不好"，**罗博特科"精密对准光路"**。耦合是价值最高、壁垒最高的环节，罗博特科通过 ficonTEC 独占这个位置。
