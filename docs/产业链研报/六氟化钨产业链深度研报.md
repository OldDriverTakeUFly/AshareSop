## 一、WF₆ 技术背景与应用

### 1.1 什么是六氟化钨

六氟化钨（WF₆，Tungsten Hexafluoride）是唯一能在室温下稳定存在的钨的气态化合物，分子量 297.8，是无色/淡黄色气体。它在半导体制造中用于**化学气相沉积（CVD）和原子层沉积（ALD）工艺中的钨金属薄膜生长**。

### 1.2 应用场景——钨金属沉积

| 工艺步骤 | 功能 | WF₆ 角色 |
|----------|------|----------|
| **接触孔填充（Contact Plug）** | 连接晶体管与金属布线 | WF₆ 还原沉积钨金属填满接触孔 |
| **通孔填充（Via Plug）** | 多层金属布线层间导通 | 同上 |
| **DRAM 字线/位线** | 存储器金属化 | 先进制程必需 |
| **3D NAND 选择管** | 存储堆叠层连接 | 堆叠层数↑→WF₆ 用量↑↑ |

> **核心洞察**：WF₆ 不是可选材料，而是**钨沉积的唯一气态前驱体**——没有替代品。这意味着只要半导体制造用钨（逻辑芯片+存储芯片都需要），就必然消耗 WF₆。3D NAND 从 32 层向 128/232/300+ 层演进，每翻倍堆叠 WF₆ 用量约翻倍——这是需求增长的几何级数基础。

### 1.3 纯度要求

| 纯度等级 | 应用 | 技术壁垒 |
|----------|------|:---:|
| 5N（99.999%） | 成熟制程（28nm+） | 中 |
| **6N（99.9999%）** | **先进制程（14nm 以下）/先进存储** | **高** |
| 7N+ | 极先进制程 | 极高（全球少数厂商能做） |

国产 WF₆ 目前主流在 5N-6N，中船特气、南大光电已达到 6N。

## 三、国产替代窗口：出口管制催化

### 3.1 事件链

```
2026 上半年：中国对日本实施钨出口管制
    ↓
日本 WF₆ 厂商（关东电化/中央硝子）面临原料断供
    ↓
全球 WF₆ 供给收缩 → 价格暴涨 232.7%
    ↓
国内晶圆厂加速转向国产 WF₆（中船特气/中巨芯/昊华/南大）
    ↓
国产替代率快速提升 + 量价齐升
```

### 3.2 国产替代进度

| 环节 | 国产化率 | 趋势 |

| 5N WF₆ | ~40-50% | ↑↑ 出口管制加速 |
| 6N WF₆ | ~20-30% | ↑ 突破中 |
| 7N+ WF₆ | <5% | 研发阶段 |

## 五、客户认证壁垒

半导体电子特气的核心壁垒是**晶圆厂认证周期**（通常 2-3 年），一旦认证进入供应链很难被替换。

| 公司 | 主要客户 | 认证状态 |
|------|----------|----------|
| 中船特气 | 中芯国际、长江存储、华虹、台积电、Intel | 多家 FAB 认证 |
| 中巨芯 | 中芯国际、长江存储 | 电子级氢氟酸已认证，WF₆ 认证中 |
| 南大光电 | 中芯国际、长江存储 | MO 源已认证，WF₆ 6N 突破 |
| 华特气体 | ASML 认证（气体供应商） | WF₆ 非核心 |

> **核心洞察**：客户认证是比产能更高的壁垒。中船特气凭借先发认证优势（已进入中芯/长存/台积电供应链），即使其他厂商产能跟上，短期内也难以抢走已认证客户。
