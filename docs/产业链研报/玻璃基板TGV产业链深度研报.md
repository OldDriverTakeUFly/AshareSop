### 1.2 TGV技术能力与专利

**核心技术参数**:
- **最小孔径**: 3-5μm（国内领先）
- **最大深宽比**: 100:1~150:1（全球第一梯队）
- **可加工玻璃厚度**: 0.05-1.1mm
- **可加工玻璃类型**: 钠钙玻璃、硼硅玻璃、无碱玻璃、石英玻璃
- **常规加工尺寸**: 510×515mm大板
- **通孔直径范围**: 30-150μm
- **累计专利**: 500余项相关专利
- **加工良率**: 稳定在92%以上

**工艺路线**: 自主研发超快激光+湿法混合刻蚀工艺，掌握"薄化→镀膜→TGV→精密镀铜→多层线路制作"全制程工艺，是全球少数具备全制程产业化能力的企业之一。

**关键工艺突破**:
1. 高深宽比TGV通孔激光精密加工
2. 超高精度孔金属化
3. 超薄玻璃减薄与键合
4. 多层高密度布线
5. 全玻璃基多层线路叠层及玻璃互联键合技术

## 第三章 TGV技术深度解析

### 3.1 什么是TGV？

**TGV (Through Glass Via，玻璃通孔)** 是一种先进三维集成电路技术，在玻璃基板上制作微米级通孔并进行金属化填充，实现上下层电路的垂直电气互联。它是实现玻璃基封装基板的核心工艺。

**TGV市场预测**: 
- 2022年TGV市场约6000万美元
- 预计2029年达4.805亿美元
- CAGR 34.2%（2023-2029）

### 3.2 TGV vs TSV 核心对比

| 参数 | TGV（玻璃通孔） | TSV（硅通孔） |
|------|-----------------|---------------|
| **基板材料** | 高阻玻璃（绝缘体） | 硅（半导体） |
| **介电常数(εr)** | ~5.0（低） | 11.9（高） |
| **90GHz插入损耗** | 0.5dB（低） | 1.5dB（高） |
| **5GHz螺旋电感Q值** | 34+ | 23+ |
| **互连密度** | 中等 | 10⁴-10⁵点/mm²（极高） |
| **热导率** | 1.0-1.4 W/mK（低） | 148 W/mK（高） |
| **基板成本** | 低（约为硅的1/3~1/8） | 高 |
| **工艺复杂度** | 较简单（无需绝缘层） | 复杂（需SiO₂绝缘层） |
| **翘曲问题** | 极低 | 显著 |
| **大尺寸面板** | 支持（510×515mm） | 受限于晶圆尺寸 |
| **光学透明** | 是 | 否 |
| **工艺成熟度** | 早期/成长期 | 成熟 |
| **适用场景** | 高频低损耗(>50GHz)、CPO、射频、显示 | 高密度3D集成(HBM/存储) |

### 3.3 TGV制造工艺

**核心工艺流程**:

```
玻璃基板 → 激光钻孔/刻蚀 → 通孔成型 → 种子层沉积(Ti/Cu溅射) 
→ 电镀铜填孔 → CMP平坦化 → RDL布线 → 多层叠合 → 完成
```

**关键工序**:

1. **TGV通孔加工**: 超快激光+湿法刻蚀（主流方案）；激光诱导刻蚀法(LIDE)
   - 孔形: 沙漏形（主流）、直孔、锥形
   - 最小孔径已达3-5μm

2. **金属化填孔**: 
   - 全铜填孔（主流趋势）或侧壁镀铜+介电质填充
   - 关键挑战: 无空洞(void-free)填充
   - 种子层: Ti/Cu溅射确保附着力

3. **RDL布线**: 
   - 线宽线距可达2μm/2μm
   - 线密度超500线/mm
   - 半加成法(SAP)工艺

4. **多层叠合与键合**:
   - 全玻璃基多层互联叠层
   - 玻璃-玻璃键合工艺

### 3.4 TGV关键挑战

1. **玻璃脆性**: 加工过程中易产生裂纹、微裂痕，尤其面板厚度降至200μm以下时
2. **铜填充空洞**: 深孔(>300μm)中空洞率超15%，需精密电镀控制
3. **金属附着力**: 玻璃表面光滑，铜层易分层/脱落
4. **热膨胀系数匹配(CTE)**: 玻璃与铜CTE差异大，热循环应力导致可靠性问题
5. **大面板翘曲**: 500×600mm面板退火中翘曲超150μm/m
6. **检测难度**: 微裂纹在最终工序才显现，报废成本高
7. **量产良率**: 从实验室到量产的良率爬坡是最大瓶颈

### 3.5 成本对比

**TGV成本优势**:
- 基板成本: 玻璃基板约为硅中介层的**1/8**
- TGV生产成本约为TSV的**1/3~1/2**
- 无需额外绝缘层沉积工序，工艺步骤大幅减少
- 面板级加工效率为晶圆级4.5倍

**成本结构对比**:

| 成本项目 | TSV | TGV |
|----------|-----|-----|
| 通孔蚀刻/加工 | 44% | ~20%(激光) |
| 通孔填充 | 25% | ~30% |
| 减薄 | 24% | 不需要/极少量 |
| 设备投资 | >$5M | 少15-20% |
| 材料利用率 | 85-90% | >95% |

## 第五章 玻璃基板在CPO/光互连中的应用

### 5.1 CPO技术概述

**CPO (Co-Packaged Optics，共封装光学)** 将光芯片（PIC）与电芯片（EIC）与交换芯片（ASIC）集成在同一封装基板上，显著缩短光电转换距离，降低功耗和延迟。

**CPO市场预测**:
- 2026年全球CPO光模块市场: 突破20亿美元
- 2030年: 攀升至180亿美元以上
- CAGR超70%
- 配套玻璃基封装基板市场CAGR超55%

### 5.2 玻璃基板对CPO的独特优势

**电学优势**:
- 低介电常数(~5.0 vs 硅的11.9): 高频信号传输损耗低30%以上
- 低介电损耗: 损耗因子比硅低2-3个数量级
- TGV+RDL结构带宽可达110GHz
- 超平整度，无翘曲问题

**光学优势（关键差异化）**:
- **光波导集成**: 玻璃透明特性允许在基板内部制作低损耗光波导
- **离子交换(IOX)光波导**: 传播损耗低至0.034 dB/cm
- **光纤模式匹配**: IOX光波导模式直径8-10μm，直接匹配光纤，耦合损耗<0.65dB
- **光电一体化**: 同一玻璃基板承载电信号(TGV+RDL)和光信号(光波导)
- **康宁方案**: 单一玻璃基板承载光电功能，显著简化工艺流程

**系统级优势**:
- 从"有机基板+硅中介层"两步方案 → 玻璃基板一步方案
- 面板级加工(510×515mm)，产能效率为晶圆级4.5倍
- 单个515mm面板可容纳16个PIC+1个ASIC

### 5.3 康宁玻璃基板CPO方案（全球领先）

**技术路线**:
1. 离子交换(IOX)制备单模光波导（传播损耗0.034 dB/cm）
2. 玻璃表面精密加工浅腔，嵌入高密度RDL和TGV
3. 光纤通过导销实现被动对准 → 玻璃光波导 → 绝热渐逝耦合到硅光芯片
4. 半环路插入损耗仅1.7dB

**GlassBridge平台（OFC 2026展示）**:
- 515×515mm玻璃面板
- 放置16个PIC + 1个ASIC
- 所有电信号路径在13mm以内
- 单个PIC放置时间<5分钟

### 5.4 中国CPO玻璃基板进展

| 企业 | 方向 | 进展 |
|------|------|------|
| **沃格光电** | CPO玻璃基载板 | 1.6T光模块载板批量送样验证中 |
| **深光谷科技** | TGV CPO光引擎 | 国内首条CPO光电封装产线（年产100万只），2025年5月完工 |
| **岭芯光电** | 2.5D/3D封装芯片 | 1.6T传输能力，玻璃基3D光波导产线已投产3条 |
| **友达光电** | Micro LED+CPO | 完成样品开发 |
| **长电科技** | TGV射频IPD | 性能验证优于硅基方案 |
| **通富微电** | TGV封装 | 具备封装技术能力 |

### 5.5 玻璃基板在光互连中的应用前景

**玻璃波导技术路线**:

| 技术 | 传播损耗 | 特点 |
|------|----------|------|
| 离子交换(IOX) | 0.034 dB/cm | 主流，模式匹配好 |
| 飞秒激光直写 | 较高 | 灵活但精度有限 |
| 溅射/沉积SiN | 较低 | 需额外沉积工艺 |

**CPO应用时间线**:
- 2024-2025: CPO方案商用启动
- 2026-2027: 规模上量，2027年CPO占比有望达30%
- 2028+: 数据中心内光互连逐步采用玻璃基板集成方案
- 2030: Micro LED CPO市场超100亿美元

## 第六章 投资逻辑与风险提示

### 6.1 核心投资逻辑

**产业趋势确定性**:
1. AI算力需求爆发 → 先进封装产能紧缺 → 玻璃基板成为下一代平台材料
2. 英特尔量产、台积电/苹果跟进 → 全球产业趋势不可逆转
3. 2026-2028年为规模化量产关键窗口
4. 2025年全球规模约10亿美元 → 2030年突破1000亿元（CAGR>40%）

**A股投资主线**:

```
主线一：TGV全制程加工龙头
  → 沃格光电(603773): 国内唯一TGV全制程，技术参数全球第一梯队

主线二：封装基板国产替代
  → 深南电路(002916): 内资最大封装基板供应商，FC-BGA快速突破
  → 兴森科技(002436): ABF载板国产化先行者，玻璃基板技术储备

主线三：先进封装材料
  → 华海诚科(688535): 并购衡所华威后全球EMC出货量第二

主线四：设备国产化
  → 帝尔激光(300776)/大族激光(002008): TGV激光钻孔设备
  → 天承科技(688603): TGV电镀填孔

主线五：显示跨界先进封装
  → 京东方A(000725): 面板级封装载板，与康宁联合
```

### 6.2 沃格光电投资分析

**看多因素**:
- TGV全制程技术壁垒高，国内独一无二
- 多领域送样验证，1.6T CPO方向想象空间巨大
- 与北极雄芯合作的全玻璃基AI计算芯片为行业首创
- 通格微TGV量产线已建成，具备规模化能力
- AI算力+CPO+先进封装三重概念加持
- 政策支持TGV国产化

**看空因素**:
- 连续亏损且亏损扩大（2025年亏1.58亿，2026Q1亏5110万）
- 通格微营收极小，距规模商业化尚远
- 新项目折旧压力大（资产减值-0.69亿）
- 客户认证周期长，量产时间不确定
- 玻璃基板产业化受多因素影响，有不确定性
- 部分市场传闻（华为/台积电合作）未经证实

**估值判断**: 当前沃格光电的定价反映的是TGV技术的"期权价值"而非盈利能力。核心跟踪指标为: (1)通格微营收季度环比变化；(2)CPO/先进封装送样验证结果；(3)大客户量产订单落地时间。

### 6.3 风险提示

1. **技术良率风险**: TGV金属化与Micro LED巨量转移良率能否如期提升
2. **产业化进度不及预期**: 玻璃基板全面商用化可能推迟至2028-2030年
3. **竞争加剧**: 传统有机基板持续迭代挤压新技术的市场空间
4. **供应链依赖**: 高端特种玻璃及部分核心设备仍依赖进口
5. **估值泡沫**: 部分概念股短期涨幅过大，缺乏业绩支撑
6. **宏观经济**: 消费电子需求偏弱拖累部分企业短期业绩
7. **地缘政治**: 半导体产业链脱钩风险
