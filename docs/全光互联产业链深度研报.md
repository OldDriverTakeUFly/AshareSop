# 全光互联产业链深度研报：A股参与者全景图与海外认证状态

> 研究日期：2026年6月3日
> 研究范围：A股全光互联产业链标的全景分析，覆盖光模块、光芯片、光器件、光纤连接器、原材料及设备六大环节
> 核心关注：海外产业链认证状态（英伟达/谷歌/Meta/亚马逊等）与国产替代进度

---

## 一、核心摘要

### 全光互联的定义与范围

全光互联（All-Optical Interconnect）是指数据中心与AI算力集群内部，以光信号替代电信号实现高速数据传输的技术架构。其核心产品包括800G/1.6T/3.2T高速光模块、硅光芯片、CPO（共封装光学）光引擎、OCS（全光交换机）、高密度光纤连接器等。

**本报告覆盖28家A股上市公司，按产业链环节分为六大类别，并逐一梳理其海外认证状态。**

### 核心发现

1. **中国光模块已主导全球市场**：2025年LightCounting TOP10光模块供应商中，7家为中国企业，中际旭创以53亿美元营收居首，新易盛首次超越Coherent升至第二
2. **认证分化明显**：中际旭创、新易盛、天孚通信已深度嵌入英伟达/谷歌/Meta供应链；源杰科技、剑桥科技等正处于认证加速期
3. **核心短板仍在光芯片**：25G以上光芯片国产化率仅约20%，100G以上EML芯片国产化率不足5%，200G产品几乎100%依赖进口
4. **硅光方案加速替代**：2026年硅光渗透率将超50%，成为突破EML芯片短缺的关键路径
