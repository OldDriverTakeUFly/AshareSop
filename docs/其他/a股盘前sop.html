一、概述

1.1 SOP目的

本SOP的核心目标是将投资纪律流程化，确保每个交易日开盘前：

  1. 对已持标的形成明确的当日操作预期（持有/减仓/加仓/清仓）
  2. 对潜在标的完成结构化筛选和可行性分析
  3. 对市场环境有清晰的认知和应对预案

1.2 使用节奏

前一晚 21:00-22:00  →  预研阶段（宏观扫描 + 周期分析 + 持仓检查）
     ↓
当日   8:30-9:30    →  确认阶段（数据确认 + 集合竞价 + 最终决策）
     ↓
     9:30          →  开盘执行

三、行业周期分析框架

3.1 周期分析基础理论

行业投资的核心原则是：在景气周期中参与，在产能出清期等待。

3.1.1 四阶段周期模型

每个行业都会经历四个阶段的循环：

                    ┌──────────────┐
         收入↑ 成本↓ │   繁荣期     │ 收入↑↑ 成本↑
        盈利↑↑      │  (过热期)    │ 盈利↓
                    └──────┬───────┘
                           │
    ┌──────────────┐       │       ┌──────────────┐
    │   复苏期     │       │       │   衰退期     │
    │ (产能出清末) │◄──────┴──────►│ (产能过剩)   │
    │ 收入→ 成本↓  │               │ 收入↓ 成本→  │
    │ 盈利↑        │               │ 盈利↓↓       │
    └──────────────┘               └──────────────┘
阶段 特征 投资策略 典型信号
复苏期 供需改善，价格企稳，产能开始收缩 最佳建仓期 库存去化完成、价格止跌、龙头率先扩产
繁荣期 需求旺盛，量价齐升，企业扩产 持有+逐步止盈 行业整体高盈利、媒体广泛报道、散户涌入
衰退早期 产能集中释放，供过于求，价格下跌 减仓/清仓 大量新产能公告、价格战开始、行业PE压缩
衰退晚期 行业亏损面扩大，落后产能退出 观望+储备标的 企业倒闭/被收购、政策开始救市、估值见底

3.1.2 判断周期位置的核心方法

方法一：量价信号追踪

  • 价格趋势：产品价格是周期最直接的指标

    • 持续上涨 > 3个月 → 复苏/繁荣
    • 持续下跌 > 3个月 → 衰退
    • 底部横盘 > 6个月 → 衰退晚期/复苏前夜

  • 库存信号：库存与价格的关系揭示周期位置

    • 主动去库存（价格↓ + 库存↓）→ 衰退晚期
    • 被动去库存（价格↑ + 库存↓）→ 复苏期 ← 最佳入场点
    • 主动补库存（价格↑ + 库存↑）→ 繁荣期
    • 被动补库存（价格↓ + 库存↑）→ 衰退早期 ← 离场信号

方法二：产能信号追踪

  • 在建产能/现有产能比 > 30% → 供给压力巨大，谨慎
  • 产能利用率 > 85% → 供需紧张，利好
  • 落后产能退出公告增多 → 出清进行中，关注底部
  • 行业集中度（CR3/CR5）提升 → 龙头受益，利好头部企业

方法三：财务信号追踪

  • 行业整体毛利率趋势：持续提升/稳定/持续下降
  • 行业ROE趋势：> 15%为优秀，< 8%为低迷
  • 资本开支/折旧比：> 2.0 为激进扩张，< 1.0 为收缩
  • 自由现金流：全行业FCF为负 → 产能竞赛/烧钱期

3.2 科技板块周期分析

科技板块的特点：周期较短（1-3年）、政策敏感、技术突破驱动、估值弹性大。

3.2.1 AI板块

核心周期逻辑：技术突破周期 → 资本开支周期 → 应用落地周期

周期指标 复苏信号 繁荣信号 衰退信号
大模型能力 新架构/新范式出现 能力涌现，应用爆发 技术瓶颈，迭代放缓
算力资本开支 云厂商开始扩产 全面扩产，GPU供不应求 资本开支放缓/削减
应用渗透率 < 10%（早期） 10-50%（快速增长） > 50%（增长放缓）
估值水平 PE < 30x（被低估） PE > 60x（高估风险） PE回落但业绩下降
政策态度 鼓励创新 加速推动+监管框架 监管收紧

赛道拥挤度判断：

  • 🟡 拥挤警告：所有公司都在宣称AI转型、非AI公司跨界入场、一级市场AI融资泡沫
  • 🔴 严重拥挤：AI概念股遍地、监管出手规范、龙头股高位放量滞涨
  • 🟢 拥挤出清：AI概念退潮、多数公司不再提AI、只剩真龙头在持续投入

3.2.2 半导体板块

核心周期逻辑：库存周期（3-4年）+ 产能周期（5-7年）+ 国产替代周期

周期指标 复苏信号 繁荣信号 衰退信号
库存水平 库存降至正常水位以下 库存持续去化，缺货 库存持续累积
产能利用率 回升至80%+ 95%+满产 降至70%以下
晶圆代工价格 折价减少/开始涨价 全面涨价 大幅打折
下游需求 手机/PC销量企稳 新应用驱动（AI/IoT） 终端需求疲软
国产替代率 政策推动、客户开始验证 国产率快速提升 替代放缓/外部限制

库存周期追踪：

全球半导体销售额同比增速（月度）：
  < -10% → 底部区域（关注建仓机会）
  -10% ~ 0% → 触底回升中
  0% ~ 20% → 景气上行
  > 20% → 景气过热（关注顶部风险）

3.2.3 软件板块

核心周期逻辑：技术创新周期（云计算→SaaS→AI）+ IT支出周期 + 信创周期

周期指标 复苏信号 繁荣信号 衰退信号

订单/合同增速 单季转正 连续加速 大单推迟/取消

续费率/ARR 稳定/小幅回升 快速增长 流失率上升

3.3 新能源板块周期分析

新能源板块的特点：强政策驱动、产能周期明显（2-4年）、价格战激烈、全球竞争。

观察要点：

时间 观察项 信号解读

  竞价高开 > 2%  + 放量 → 买盘强势，可观察开盘走势
  竞价高开 > 2%  + 缩量 → 可能高开低走，谨慎
  竞价平开        + 平量 → 无明显信号
  竞价低开 < -1% + 放量 → 卖压较大，注意支撑位
  竞价低开 < -2% + 巨量 → 可能有突发利空，准备止损

对关注的板块：
  板块内多数个股高开 → 板块整体偏强
  板块内分化明显 → 需个股独立分析

    1. 快速查找是否有突发利空
    2. 检查是否触及止损线
    3. 如果是系统性下跌（板块整体低开）：按板块操作
    4. 如果是个股利空：考虑9:25-9:30之间挂单减仓

六、新增标的筛选流程

6.1 第一步：周期定位（确定赛道方向）

原则：只在处于复苏期或繁荣早期的赛道中寻找标的，不在衰退期赛道中做抄底动作。

操作步骤：

  1. 取出最新的板块周期位置速查表（3.6节）
  2. 筛选出周期位置为"复苏"或"繁荣早期"的赛道
  3. 排除赛道拥挤度≥7分的赛道
  4. 在通过筛选的赛道中，选择1-2个最值得关注的赛道

决策逻辑：

可选赛道 = 周期位置∈{复苏, 繁荣早期} AND 拥挤度 < 7分
IF 可选赛道数量 = 0：
    → 当日不新增标的，保持现有仓位
IF 可选赛道数量 ≥ 1：
    → 在最优赛道中执行后续筛选步骤

优先级 类型 说明
⭐⭐⭐ 龙头企业 市占率前3，有定价权

4. 取排名前10的个股

6. 筛选后保留3-5只候选标的

6.3 第三步：技术确认（择时验证）

对筛选出的候选标的进行技术面检查：

6.3.1 趋势确认

检查项 合格标准 不合格处理
均线系统 5日>20日>60日（多头排列） 等待均线修复后再入场
趋势线 处于上升通道或底部突破确认 不入场

成交量 近期放量上涨日多于缩量下跌日 缺乏资金关注，不入场

6.3.2 入场时机判断

入场类型 技术特征 操作
突破买入 放量突破前期高点/平台/下降趋势线 突破确认后首次回踩时买入

底部买入 长期下跌后底部放量企稳 需要配合基本面催化，谨慎

6.3.3 不入场的技术信号

以下情况应放弃入场或等待：
❌ 连续大涨3天以上（追高风险大）
❌ 距支撑位距离>10%（缺乏安全边际）
❌ 成交量极度萎缩（无人关注）
❌ 即将面临重大不确定性事件（如业绩披露前）
❌ 均线系统空头排列（趋势向下）

🟡中 资金催化 1-2周 北向持续流入、机构调研密集

   - 未被预期 → 真正的催化剂

6.5 第五步：风险收益评估

最终筛选：对通过以上四步的标的进行风险收益比评估。

6.5.1 计算风险收益比

风险收益比（R:R）= 潜在收益 / 潜在亏损

潜在亏损 = 入场价 - 止损价（止损定义见第七章）
潜在收益 = 目标价 - 入场价
  目标价参考：
    1. 下一档技术压力位
    2. 分析师一致预期目标价
    3. 基于业绩增速的合理估值

最低要求：R:R ≥ 2:1（潜在收益至少是风险的2倍）
理想目标：R:R ≥ 3:1

6.5.2 仓位确定

单笔风险 = 总资产 × 单笔最大风险比例（建议1-2%）
买入股数 = 单笔风险 /（入场价 - 止损价）
买入金额 = 买入股数 × 入场价

示例：
总资产 100万，单笔风险比例 2% = 2万
入场价 50元，止损价 45元（止损幅度10%）
买入股数 = 20000 / (50-45) = 4000股
买入金额 = 4000 × 50 = 20万（占总资产20%）

7.1 仓位管理规则

7.1.1 单票仓位限制

约束类型 限制 说明

约束类型 限制 说明

7.1.3 总仓位管理（根据市场环境动态调整）

市场判断 总仓位范围 判断依据
🟢 牛市 80-95% 趋势向上+政策友好+资金涌入+板块普涨
🟡 震荡市 60-80% 方向不明+板块分化+量能一般
🟠 弱势市 30-60% 趋势向下+资金流出+市场信心不足
🔴 危机市 10-30% 系统性风险+恐慌抛售+流动性危机

仓位调整节奏：

市场判断变化时，不一次性调仓：
- 加仓：每周增加10-15%仓位，逐步到位
- 减仓：可以更快，每周减少15-20%仓位
- 危机模式：可一次性大幅减仓

7.2 止损体系

7.2.1 三层止损机制

根据你的投资风格（平衡型 + 逻辑止损为主），设计以下三层止损：

第一层：逻辑止损（主动止损）

这是你的核心止损方式。当买入逻辑被破坏时，无论盈亏，坚决离场。

触发条件 示例
行业周期转折 从复苏转为衰退早期
核心假设被证伪 认为某技术会突破但迟迟未能实现
竞争格局恶化 龙头失去市场份额/价格战加剧
政策方向转向 从支持变为限制
公司基本面恶化 核心指标连续两季低于预期

执行要求：

  • 不需要等到亏损才止损——盈利中逻辑破坏同样要止盈/止损
  • 逻辑止损的决策应在预研阶段做出，不等到盘中恐慌

第二层：技术止损（防御止损）

作为逻辑止损的补充安全网，防止"逻辑还没被证伪但股价已经崩了"的情况。

止损类型 设定方法 适用场景
关键支撑止损 跌破重要支撑位（如20日均线、前低） 所有持仓
硬止损 亏损达到 -12%（从买入价计算） 所有持仓，无条件执行
跟踪止损 盈利超过15%后，从最高点回撤 -8% 即止损 盈利持仓保护利润
放量破位止损 放量（量比>2）跌破关键均线 趋势性持仓

第三层：时间止损（效率止损）

当一笔交易长时间没有按预期发展时，需要重新评估是否值得继续持有。

触发条件 操作
买入后1个月内未见预期走势 重新评估逻辑，如无变化可继续持有
买入后3个月横盘不动 考虑减半仓，释放资金给更有机会的标的
买入后6个月仍未达预期 强制重新评估，大概率应清仓换标的

7.2.2 止损执行纪律

止损铁律（不可违反）：

1. 亏损达到-12%的硬止损，当天必须执行，不接受"等反弹"
2. 逻辑止损一旦确认，次日开盘即执行，不犹豫
3. 止损后不在同一天重新买入同一标的
4. 止损后的资金不急于立即使用，至少等待1个交易日
5. 每月止损次数超过3次 → 强制降低仓位至50%以下，复盘反思

7.3 组合风险管理

7.3.1 组合层面的风险监控

风险指标 预警线 强制线 触发动作
单日组合回撤 -3% -5% 预警→审视持仓 / 强制→减仓至60%
单周组合回撤 -5% -8% 预警→降低仓位 / 强制→减仓至40%
单月组合回撤 -8% -12% 预警→全面复盘 / 强制→减仓至30%+停止交易1周
连续止损次数 3次/月 5次/月 暂停交易+复盘 / 强制停止交易2周

7.3.2 相关性检查

避免"看起来分散但实际风险集中"的情况：

每周检查一次：
□ 是否所有持仓都受同一因素影响？（如全部是出口导向型）
□ 是否多个持仓面对同一风险事件？（如全部受中美关系影响）
□ 如果某一板块下跌20%，组合总损失是多少？（应<8%）

相关性过高的处理：
- 同一产业链上下游可持有（如铜矿+铜加工，风险不完全一致）
- 同一板块纯竞争关系不持有多个（如不同时持有多家锂电正极材料公司）
- 跨板块配置优先（科技+新能源+周期品的组合天然分散）
