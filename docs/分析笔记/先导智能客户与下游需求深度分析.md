### 2.3 宁德时代合作关系深度分析

宁德时代是先导智能最重要的客户和战略合作伙伴，双方合作持续深化：

**合作历程：**
- **2020年9月**：宁德时代通过定增入股先导智能（持股7.14%），成为第二大股东，签署首轮战略合作协议
- **2021年2月**：签署补充协议，宁德承诺不低于电芯段核心设备新投资额50%给予先导优先权，合作范围覆盖涂布、卷绕、化成等
- **2024年11月**：签署新一轮三年期战略合作协议，合作范围扩大至涂布、辊压、模切、卷绕、叠片、物流、化成容量等全环节电芯核心设备，并拓展模组PACK及固态电池、钙钛矿等新领域
- **2025年全年预计关联交易规模约85亿元**，2025Q1实际已达21亿元（接近2024年全年实际规模的60%），宁德订单有望重回2020-2021年高峰规模

**宁德时代扩产需求：**
- 2024年宁德时代产能676GWh，在建产能219GWh，产能利用率76.3%
- 2024年7月起产能利用率环比提升，9月升至约90%，目前已处于满产状态
- 2025年新增产能预计150-200GWh（含海外基地35GWh）
- 匈牙利德布勒森工厂：总产能72GWh（一期34GWh预计2025年投产，二期38GWh预计2027年投产），港股IPO募资353亿港元中90%（276亿港元）投入该工厂
- 西班牙合资工厂（与Stellantis合作）：规划50GWh，预计2026年底试生产
- 印尼合资工厂：规划15GWh，预计2027年投产

## 四、下游需求分析——动力电池

### 4.1 全球电动车市场趋势

**全球EV电池需求爆发式增长：**
- 2024年全球EV电池安装量约901 GWh（IEA数据）
- 2025年全球EV电池安装量达1,187 GWh，同比+31.7%（SNE Research）
- IEA预计2030年EV电池需求将超过3 TWh（STEPS情景）

**2025年全球锂离子电池关键数据：**
| 指标 | 数据 | 来源 |
|---|---|---|
| 全球锂电池部署总量 | ~1.74 TWh | Analysis Atlas综合 |
| 其中EV电池 | 1,187 GWh (+31.7% YoY) | SNE Research |
| 其中储能电池 | 550 GWh (+79% YoY) | 多源综合 |
| 全球锂电池市场规模 | >1500亿美元 (+20% YoY) | IEA |
| 平均锂电池包价格 | $108/kWh（创历史新低） | BloombergNEF |
| BEV专用包价格 | $99/kWh（连续两年低于$100） | BloombergNEF |

### 4.2 全球电池产能扩张

**全球电池产能严重过剩，但结构性分化：**

| 指标 | 2025年数据 | 来源 |
|---|---|---|
| 全球名义电池产能 | 4.8 TWh | Benchmark MI |
| 其中中国产能 | ~2,560 GWh | CRU Group |
| 全球实际出货量 | ~1,875 GWh（中国） | CRU Group |
| 中国产能利用率 | ~73.2% | CRU Group |
| 2030年规划产能 | >10 TWh | Benchmark MI |
| 77%规划产能归属 | 中国公司 | Benchmark MI |

**关键判断：**
- 全球2030年规划产能（10 TWh）远超合理需求预测（4-5 TWh），意味着大量项目将被取消或延迟
- 中国产能利用率从2022年的45%提升至2025年的73.2%，但距85%的理想水平仍有差距
- **设备需求≠产能扩张**：设备需求与产能利用率挂钩——当利用率达到80-90%，厂商才有动力大规模扩产
- 头部厂商（宁德时代、比亚迪）利用率已达90%+，扩产意愿强烈；二三线厂商利用率偏低，扩产更谨慎

### 4.3 电池技术趋势对设备需求的影响

| 技术趋势 | 设备影响 | 先导智能布局 |
|---|---|---|
| **LFP占比持续提升** | LFP设备需求结构性增长，2025年LFP占EV电池超50%、储能超90% | 已为多家头部企业完成GWh级LFP项目交付 |
| **固态电池产业化** | 设备价值量大幅提升（干法电极、等静压、固态叠片等新工艺），毛利率有望达50%+ | 全球唯一全固态整线解决方案；已向欧美日韩客户发货 |
| **钠离子电池** | 新增产线需求，但设备与锂电有较大通用性 | 已与法国Tiamat合作；完成产线批量交付 |
| **46系列大圆柱** | 需要新型卷绕/组装设备 | 已向保时捷交付全极耳圆柱电芯产线 |
| **全极耳技术** | 新增极耳整形、焊接设备需求 | 已掌握全极耳电池整线技术 |

**长期展望：**

- 累计装机到2036年将达2.9 TW / 10.5 TWh
- 储能电池需求占锂电总需求的比例从2023年的16%升至2025年的19%，2026年1-4月中国储能需求占比已达26%

### 5.2 储能电池产线设备与动力电池的差异

储能电池产线设备与动力电池有共性也有差异：

| 维度 | 动力电池 | 储能电池 |
|---|---|---|
| 电池形态 | 方形/圆柱/软包 | 以大方形为主（280Ah以上大容量电芯） |
| 核心设备 | 涂布、卷绕/叠片、组装、化成分容 | 与动力电池基本通用，但化成分容工段要求更高（长循环寿命验证） |
| 设备价值量 | 标准GWh产线约2-3亿元 | 类似，但大容量电芯对涂布宽度、叠片精度有特殊要求 |
| 产线特点 | 追求高能量密度、快充 | 追求长寿命（>6000次循环）、低成本、高安全性 |
| 技术趋势 | 固态、半固态、46大圆柱 | LFP为主流（>90%），钠离子渗透加速 |

**核心结论：储能电池产线与动力电池产线在设备上具有较高通用性（约70-80%设备可共用），这意味着储能需求爆发可以直接转化为锂电设备需求增量，无需完全新建产线类型。** 先导智能已为多家全球头部企业完成GWh级储能项目交付，累计订单规模行业领先。

## 六、光伏设备客户与需求

### 6.1 光伏设备业务概况

先导智能光伏设备业务2025年实现营收11.23亿元（+29.48%），占比7.77%。公司提供光伏组件和光伏电池制造设备及整线解决方案。

**核心产品矩阵：**
- **光伏组件端**：BC/0BB/SMBB串焊机、划焊一体设备、叠瓦焊接设备
- **光伏电池端**：丝印、烧结、测试分选、湿法工艺设备、TOPCon/HJT/XBC/钙钛矿整线

### 6.2 主要客户

光伏设备客户以行业头部企业为主（年报未单独披露具体客户名称）：
- **头部BC电池企业**：累计获得超30GW的XBC设备订单，客户已实现量产
- **TOPCon头部客户**：打造行业首条TOPCon3.0标杆整线，量产效率≥26.5%
- **钙钛矿前沿客户**：为中国钙钛矿行业首条GW级生产线提供国内首台激光划线设备
- **全球领先电动汽车及清洁能源企业**（推断为特斯拉）：光伏组件焊接设备已交付

### 6.3 光伏设备需求趋势

| 趋势 | 影响 |
|---|---|
| **BC电池加速产业化** | 2025年GW级标段不断涌现，先导XBC串焊机出货量全球第一 |
| **TOPCon持续迭代** | 从TOPCon2.0→3.0→4.0，每次迭代均释放设备更新需求 |
| **钙钛矿量产加速** | 先导已布局实验室/中试线/产线级全方案 |
| **0BB技术渗透** | 降低银浆用量，新设备需求释放 |
| **全球光伏装机持续增长** | 2024年全球光伏新增装机约500GW+，带动设备需求 |

**先导在光伏领域的核心壁垒：**
- XBC串焊机出货量居全球首位（累计超30GW）
- 全球唯一提供XBC整线焊接方案的供应商
- TOPCon3.0整线交钥匙工程行业首条
- 在BC、TOPCon、HJT、钙钛矿多技术路线形成"预研一代、量产一代"的迭代优势

## 七、3C智能装备客户与需求

### 7.1 业务概况

3C智能装备业务由子公司**立导科技**运营，2025年实现营收6.07亿元（-11.92%），占比4.20%。

**核心能力：**
- 自研3D+AI视觉算法（HySmart平台）
- 精密流体技术（HyFluid平台）
- 集成测试平台（HyTest平台）
- 产品覆盖：视觉测量、AI瑕疵检测、五轴高速点胶、大流量封胶、成像测试、电气测试、可靠性测试及3D组装

### 7.2 客户与市场

立导科技覆盖三大智能领域：
- **消费电子**：MR、TWS耳机、智能手表、手机、平板等（据公司发展史记载，曾用于"苹果手机锂电池生产"，华为曾授予"技术突破奖"）
- **智能汽车**：Pack&CTP、智能电动、智能座舱、智能驾驶
- **数字能源**：储能集装箱、逆变器、超充充电桩

## 九、需求可持续性评估

### 9.1 锂电设备需求：周期性还是结构性？

**核心判断：锂电设备需求具有"结构性增长+周期性波动"的双重特征。**

**结构性增长驱动因素（长期看多）：**
1. **全球电动化趋势不可逆**：2025年全球每售出4辆汽车就有1辆是电动车，IEA预计2030年EV电池需求超3TWh
2. **储能市场爆发**：成为锂电设备第二增长曲线，2025年全球储能GWh装机同比+61%，2026年预计再增28-41%
3. **技术迭代驱动设备更新**：固态电池、钠离子电池、46大圆柱、全极耳等新技术路线带来全新设备需求，且设备价值量更高
4. **海外产能建设**：欧美日韩本地化产能建设带来全新增量市场，先导已成功切入大众、BMW、特斯拉等供应链
5. **设备折旧周期**：锂电产线设备寿命约5-8年，2020-2022年集中建设的产线将陆续进入更新周期

**周期性波动因素（需警惕）：**
1. **电池产能过剩**：全球名义产能远超需求，2025年中国产能利用率仅73.2%
2. **国内需求增速放缓**：2024年国内锂电设备市场受下游扩产放缓影响，先导锂电设备营收一度下降39%
3. **价格战压缩电池厂利润**：电池价格持续下跌（2025年BEV包价$99/kWh），削弱电池厂扩产意愿
4. **政策不确定性**：美国IRA补贴政策调整、欧盟贸易壁垒等

### 9.2 设备需求与产能利用率的关系

**核心逻辑：设备需求 ≈ 新增产能需求 + 旧线改造/更新需求**

- 当产能利用率 < 70%：电池厂扩产意愿极低，设备需求主要来自旧线更新
- 当产能利用率 70-85%：电池厂开始规划新产能，设备订单逐步释放
- 当产能利用率 > 85%：电池厂大规模扩产，设备需求爆发

**当前状态：**
- 宁德时代利用率已达90%+（满产），219GWh在建产能加速落地 → 设备需求旺盛
- 比亚迪利用率维持高位，海外基地布局加速 → 设备需求增长
- 二三线厂商利用率分化，部分已停产或转型 → 设备需求集中在头部

### 9.3 先导智能的抗周期能力评估

| 维度 | 评估 | 评级 |

| **客户多元化** | 对宁德依赖度从高点降至~21%，海外客户占比持续提升 | ★★★★ |

### 9.4 需求展望（2025-2027）

**乐观情景（概率40%）：**
- 宁德时代扩产超预期（年扩产200GWh+），海外整车厂自建电池产能加速落地
- 固态电池产业化提速，先导获得大量新技术产线订单
- 储能需求持续超预期（年增50%+）
- 预计公司年营收可达200亿元+

**基准情景（概率45%）：**
- 头部电池厂稳步扩产，海外订单持续释放
- 储能需求保持30-40%增速
- 国内二三线厂商扩产谨慎
- 预计公司年营收160-180亿元

**悲观情景（概率15%）：**
- 全球经济衰退导致EV需求大幅放缓
- 地缘政治冲突加剧，海外市场受限
- 国内电池产能严重过剩，设备验收持续延迟
- 预计公司年营收120-140亿元

## 十、核心结论与投资建议

### 10.1 核心结论

1. **客户结构持续优化**：宁德时代仍是最大客户（~21%），但依赖度持续下降；海外客户占比已达22%+，且毛利率更高（40%+ vs 综合35%）；客户覆盖国内所有头部电池企业+海外主流整车厂/电池厂

2. **下游需求正从"单一动力电池驱动"转向"动力+储能双轮驱动"**：2026年1-4月中国储能需求同比+90%，占锂电总需求26%，成为最大增量来源

3. **技术迭代是设备需求的"永动机"**：固态电池（设备价值量翻倍）、BC电池、钙钛矿、钠离子等新技术路线持续释放新设备需求，先导在所有前沿技术均有完整布局

4. **海外产能建设是未来3-5年最大增量**：欧洲500GWh产能缺口+美国IRA驱动本土化+中国电池企业出海，先导作为核心设备供应商将充分受益

5. **设备需求具有结构性增长特征，但短期存在周期性波动**：2024年是本轮周期底部（先导锂电设备营收-39%），2025年拐点已确立（+23%），2026年有望持续改善
