## 一、光芯片：DFB（分布式反馈激光器）

### 1.1 国产化率总览

DFB芯片是光模块中最基础的光源芯片，覆盖从接入网到数据中心的完整速率需求。当前国产替代进度呈现明显的速率梯度：

| 速率/品类 | 国产化率（2024） | 代表企业与进度 |
|---|---|---|
| 10G DFB | ~100%（基本完成替代） | 源杰科技全球市占率约20% |
| 25G DFB | ~30% | 光迅科技自供比例>70% |
| 50G DFB | <10% | 光迅科技、源杰科技小批量 |
| CW DFB（硅光光源） | ~15% | 源杰科技2024年出货突破百万颗 |
| 25G以上高端光芯片整体 | <5% | 国产替代早期阶段 |

**关键判断**：10G DFB已完成国产化，25G正处于替代加速期，50G及以上仍是海外厂商主导。CW DFB作为硅光模块的核心光源，是当前国产突破的重点方向。

### 1.2 核心公司分析

#### 源杰科技（688498.SH）— DFB全球龙头

源杰科技是国内DFB芯片领域毫无争议的领军企业，采用IDM全流程模式，外延良率达90%以上。

| 指标 | 数据 |

| 2025年营收 | 6.01亿元（+138.5%） |
| 数据中心收入 | 3.93亿元（占比65.57%） |
| 制造模式 | IDM全流程，外延良率90%+ |

**主要客户**：中际旭创、海信宽带、博创科技、铭普光磁

**投资逻辑**：源杰的CW DFB产品线直接受益于硅光模块渗透率提升。2025年营收增速138.5%，其中数据中心收入占比已超65%，说明公司已从传统接入网成功切换到AI算力驱动的数据中心赛道。CW 100mW通过验证后，有望进入更高功率硅光模块供应链。

## 二、光芯片：EML（电吸收调制激光器） — 最高价值段

### 2.1 为什么EML是最高价值段？

EML（Electro-absorption Modulated Laser）将激光器和调制器集成在同一芯片上，相比DFB+外调制方案，具备更小体积、更低功耗、更优信号质量的优势。在200G/400G/800G高速光模块中，EML是主流光源方案。

**核心逻辑**：速率越高，EML的不可替代性越强。200G PAM4 EML是当前800G光模块的核心芯片，400G EML将决定3.2T模块的竞争格局。

### 2.2 核心公司分析

#### 东山精密 / 索尔思光电（002384.SZ）— 国内EML绝对龙头

2025年10月，东山精密全资收购索尔思光电，使索尔思成为东山精密全资子公司。索尔思是国内唯一量产200G EML的企业，在这一最高价值段建立了显著的先发优势。

**核心数据：**

| 指标 | 数据 |
|---|---|
| 100G PAM4 EML累计出货 | >1000万颗 |
| 200G PAM4 EML | 已量产（2025年） |
| 光芯片月产能（2025H1） | ~300万颗 |
| 2026Q4目标月产能 | 2200万颗 |
| EML自供率 | 99%+ |

**主要客户**：英伟达、谷歌、微软等北美头部云厂商

**投资逻辑**：索尔思的核心壁垒在于200G EML的量产能力。全球范围内能批量交付200G EML的厂商屈指可数，索尔思不仅填补了国内空白，更直接打入北美头部云厂商供应链。2026Q4月产能目标2200万颗，意味着未来一年产能将扩张7倍以上，对应的是1.6T光模块规模化量产的巨大需求。

#### 源杰科技（688498.SH）— EML追赶者

源杰在EML领域是追赶者角色，但进展迅速。

| 速率 | 进度 |
|---|---|
| 25G EML | 规模化量产（国内最早） |
| 100G PAM4 EML | 通过客户验证 |
| 200G PAM4 EML | 完成开发 |

**产能规划**：

| 时间节点 | 年产能 |
|---|---|
| 2025年底 | ~1200万颗/年 |
| 2026年 | 3000万颗/年 |

**投资逻辑**：源杰的EML路径是从低速率向高速率逐步爬坡。25G EML已率先量产建立工艺基础，100G通过验证意味着即将进入贡献期，200G完成开发则说明公司在技术端已追近第一梯队。2026年3000万颗/年的产能规划，对应的是对800G/1.6T模块市场的大举切入。

#### 长光华芯（688048.SH）— 高端EML新锐

长光华芯在EML领域的特点是直接切入高端，100G和200G PAM4 EML的产品节奏紧凑。

**核心数据：**

- 100G PAM4 EML：2024年9月发布，2025H1量产
- 200G PAM4 EML：2025年3月发布并送样
- 6英寸产线年产能：1000万颗+
- 2025年前三季度营收3.39亿元（+67.42%），实现扭亏

**投资逻辑**：长光华芯的亮点在于从亏损到扭亏的业绩拐点已经出现。6英寸产线年产能1000万颗以上，为后续EML放量提供了产能基础。200G EML从发布到送样的间隔仅半年，产品迭代速度很快。

#### 光迅科技（002281.SZ）

| 速率 | 进度 |
|---|---|
| 25G EML | 量产，良率>70% |
| 50G EML | 量产 |
| 100G EML | 小批量，月产~10万颗 |
| 200G EML | 良率~40% |

光迅的EML覆盖面广，但200G EML良率40%仍需提升。作为华为昇腾AI的独家供应商，EML自供对降低光模块成本至关重要。

#### 华工科技 / 云岭光电

- 10G/25G EML：已量产
- 56G Baud EML：小批量
- 100G EML：头部客户测试中

#### 三安光电（600703.SH）

- 100G EML：2026年4月正式推出

三安作为化合物半导体龙头，进入EML赛道时间较晚，但IDM体量和客户资源不可忽视。

### 2.3 EML国产化进度总览

| 速率 | 与海外差距 | 国内进度 |
|---|---|---|
| 25G EML | 基本追平 | 多家企业量产 |
| 50G / 56G Baud | 2-3年 | 光迅量产，华工小批量 |
| 100G PAM4 | 1-2年 | 索尔思大量出货，源杰/长光验证通过 |
| 200G PAM4 | 1-2年（索尔思已量产） | 索尔思唯一量产 |
| 400G | 同步竞争 | 技术研发阶段 |

**核心结论**：EML国产化在25G已实现追平，200G PAM4因索尔思的量产使差距缩短至1-2年，400G处于同步竞争阶段。索尔思的200G EML量产是国产EML领域最重要的里程碑事件。

## 三、光芯片：VCSEL（垂直腔面发射激光器）

### 3.1 技术定位

VCSEL芯片因制造工艺与DFB/EML不同（面发射而非边发射），在短距光互连（数据中心内部）和消费电子（3D sensing）两大场景有广泛应用。在光通信领域，100G PAM4 VCSEL是下一代短距光模块的核心方案。

### 3.2 核心公司分析

#### 长光华芯（688048.SH）

- 100G PAM4 VCSEL量产出货（850nm）
- 获Lightwave + BTR年度创新奖

长光华芯是少数同时布局EML和VCSEL的企业，双产品线共享化合物半导体制造平台。

#### 三安光电（600703.SH）— 国内VCSEL产能最大

| 产品线 | 月产能 |
|---|---|
| 通信VCSEL | 20KK |
| 消费VCSEL | 150KK |
| 合计 | 170KK+（国内最大） |

- 2026年4月：宽温VCSEL突破车规级

三安的VCSEL产能规模在国内遥遥领先，消费VCSEL月产能150KK说明在手机3D sensing领域已有稳固份额。宽温VCSEL突破车规级打开了车载光通信和车载感知的新市场。

#### 立昂微（605358.SH）— 激光雷达VCSEL新势力

立昂微通过旗下立昂东芯，在激光雷达VCSEL领域实现了独特突破。

- 全球唯二量产二维可寻址激光雷达VCSEL的企业
- 2025年VCSEL销量同比增长近7倍
- 主要客户：禾赛科技（比亚迪激光雷达供应商）、速腾聚创、华为系

**投资逻辑**：二维可寻址VCSEL是固态激光雷达的核心光源，全球能批量交付的厂商极少。立昂东芯的7倍销量增长说明激光雷达VCSEL正在快速从样品走向量产。比亚迪搭载禾赛激光雷达的量产车型是需求端的核心驱动。

#### 乾照光电（300102.SZ）

- 控股股东：海信视像
- 10G/25G VCSEL送样验证中

背靠海信集团的光通信资源，乾照光电在VCSEL领域的进展值得关注。

#### 非上市企业及A股关联

**老鹰半导体（非上市）**：国内首家100G VCSEL量产；2025年B+轮融资超7亿元

**纵慧芯光（非上市）**：VCSEL国内市占率第一；累计出货超6亿颗

A股关联持股：

|---|---|---|
| 东阳光（600673） | 纵慧芯光 | ~2.51% |
| 华西股份（000936） | 纵慧芯光 | ~2.48% |

## 四、激光器 / 光源器件

### 4.1 泉浦激光器（Pump Laser）

泵浦激光器是光放大器的核心光源，在硅光模块中CW光源充当泵浦源角色。

| 公司 | 产品与进度 | 光通信暴露度 |
|---|---|---|
| 源杰科技 | CW 70mW/100mW/300mW，硅光模块核心泵浦源 | 高 |
| 长光华芯 | 70mW CWDM4 CW Laser | 中 |
| 锐科激光（300747.SZ） | 子公司睿芯特种光纤（掺铒/掺镱），2024年营收0.71亿元 | 极低（<5%） |

**重要提示**：锐科激光主业是工业激光器，光通信收入占比不到5%，投资逻辑应以工业激光为主。

### 4.2 可调谐激光器

可调谐激光器是相干光模块和DWDM系统的核心器件。这一领域A股纯正标的极度稀缺。

**A股相关标的**：

| 公司 | 进度 |
|---|---|
| 源杰科技 | C波段DFB已量产 |
| 光迅科技 | 可调谐激光器组件 |
| 云岭光电 | 新三板 |

**全球格局**：可调谐激光器市场被Lumentum、NeoPhotonics（已被海信收购）、II-VI等海外巨头掌控，国产替代空间巨大但技术壁垒也最高。

### 4.3 LiDAR与光通信交叉

部分激光芯片企业同时覆盖LiDAR和光通信两大场景，形成双赛道交叉优势：

| 公司 | 交叉特点 |
|---|---|
| 长光华芯 | EML + VCSEL + LiDAR光源，最典型的双赛道激光芯片公司 |
| 源杰科技 | DFB + EML + CW光源 + LiDAR光源 |

### 4.4 重要纠正

**瀚天天成（02726.HK）是SiC外延公司，与光通信完全无关。** 市场部分研报将其归入光通信供应链，这是错误的。SiC外延服务于功率半导体和新能源汽车，不涉及光通信。

## 五、光通信测试设备

### 5.1 行业背景

光通信测试设备是产业链的"卖水人"。光模块从400G升级到800G、1.6T、3.2T，每代速率迭代都需要全新的测试仪器。随着AI算力爆发驱动的光模块需求激增，测试设备的需求同步放大。

2024年中国光通信测试仪器市场规模33.0亿元，预计2029年增长至65.9亿元，CAGR 14.84%。

### 5.2 核心标的：联讯仪器（688808.SH）

联讯仪器于2026年4月上市，首日涨幅875%，市值约820亿元。上市即获得极高估值，反映了市场对国产高端测试仪器稀缺性的认可。

**核心定位**：全球第二家、国内唯一量产1.6T光模块全套核心测试仪器的企业。

**产品矩阵**：

| 产品 | 参数 |
|---|---|
| 采样示波器 | 65GHz |
| CRU（时钟恢复单元） | 120GBaud |
| BERT（误码率测试仪） | 113GBaud |
| OSA（光谱分析仪） | — |
| 可调谐激光源 | — |
| 晶圆级老化系统 | — |

**财务数据**：

| 年份 | 营收（亿元） | 增速 |
|---|---|---|
| 2022 | 2.14 | — |
| 2023 | — | — |
| 2024 | — | — |
| 2025 | 11.94 | — |
| 2026Q1 | 5.00 | +148.63% |

从2022年2.14亿到2025年11.94亿，三年营收增长约4.6倍。2026Q1单季度营收5亿元，同比增长148.63%，单季度已接近2022年全年营收的2.3倍。

**市场份额**：2024年中国市场份额9.9%（国产品牌第一，整体排名第三）。

**主要客户**：中际旭创、新易盛、光迅、Lumentum、Coherent、Broadcom、住友

**投资逻辑**：联讯仪器的核心逻辑是"测试设备国产替代"与"1.6T放量"的双重驱动。其客户名单同时覆盖国内头部光模块厂商和海外巨头（Lumentum、Coherent、Broadcom），说明产品竞争力已获得全球认可。晶圆级老化系统对应"测试左移"趋势，从模块级测试前移到芯片甚至晶圆级，市场空间进一步打开。

### 5.3 市场竞争格局

2024年中国光通信测试仪器市场份额：

| 排名 | 厂商 | 份额 |
|---|---|---|
| 1 | Keysight（是德科技） | 30.8% |
| 2 | Anritsu（安立） | 22.5% |
| 3 | 联讯仪器 | 9.9% |
| 4 | Tektronix（泰克） | ~8% |
| 5 | VIAVI | ~5% |

海外厂商合计份额约84%，国产替代空间广阔。联讯仪器以9.9%的份额位居国产第一，是唯一进入前三的中国企业。

### 5.4 其他相关公司辨析

| 公司 | 与光通信测试的关系 | 光通信测试暴露度 |
|---|---|---|
| 鼎阳科技 / SIGLENT（688112） | 通用电子测试仪器，无专用光通信测试产品线 | 极低 |
| 烽火通信 | OTDR模块为系统内置组件，非独立销售产品 | 低 |
| 中航光电 | 光电连接器厂商，不生产测试设备 | 无 |

**结论**：在A股光通信测试设备领域，联讯仪器是唯一具备纯正暴露度的标的。鼎阳科技、烽火通信、中航光电虽然在各自领域有优势，但在光通信测试设备方向不具备投资逻辑。

### 5.5 增长驱动因素

1. **速率迭代**：400G → 800G → 1.6T → 3.2T，每代速率升级都需要全新测试仪器
2. **AI算力爆发**：英伟达GB200/GB300驱动1.6T光模块规模化部署
3. **"测试左移"趋势**：测试环节从模块级前移到芯片级甚至晶圆级，单模块测试设备需求量倍增
4. **国产替代政策**：2026年政府工作报告明确推动高端仪器仪表国产化
5. **成本优势**：国产测试设备比海外同类产品低30-50%，性价比优势明显

## 六、风险提示

1. **技术迭代风险**：光芯片技术路线变化（如硅光集成方案对传统分立芯片的替代）可能导致部分细分品类需求不及预期。
2. **产能过剩风险**：多家企业同时扩产EML/VCSEL，若下游需求增速放缓，可能出现价格竞争和产能利用率下降。
3. **客户集中度风险**：部分企业客户集中于少数头部云厂商，大客户订单波动可能对业绩产生较大影响。
4. **估值风险**：光通信板块经历大幅上涨后，部分公司估值已处于历史高位，存在回调风险。联讯仪器上市首日涨幅875%，市值约820亿元，需关注后续业绩兑现能力。
5. **地缘政治风险**：北美云厂商供应链政策变化可能影响国内光芯片和光模块企业的海外业务。
6. **研发失败风险**：200G/400G EML等高端芯片的良率提升和量产进度存在不确定性。
7. **报告数据来源风险**：本报告数据来源于公开研报、公司公告及行业数据库，未进行独立验证，数据准确性依赖于原始信源。部分数据为估算值，实际数据可能存在偏差。
