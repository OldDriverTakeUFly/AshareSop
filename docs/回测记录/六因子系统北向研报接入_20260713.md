# 六因子系统 — 北向资金+研报因子接入与验证

> **新增因子**：北向持股变化（northbound）+ 研报密度变化（research）
> **验证方法**：top-10 标的 V4(4因子) vs V6(6因子) 回测对比
> **生成日期**：2026-07-13

---

## 一、因子设计

### 北向持股因子（northbound_score, 0-100）

| 要素 | 设计 |
|------|------|
| 数据源 | Tushare `hk_hold`（个股沪深港通持股） |
| 计算方式 | 最新持股比例 - 120天前持股比例 |
| 分值映射 | +2pp=100, -2pp=0, 0pp=50 |
| 权重 | 0.10（辅助因子） |
| 研究依据 | 50%套利/38%加仓；北向加仓标的平均涨幅更好 |

### 研报密度因子（research_score, 0-100）

| 要素 | 设计 |
|------|------|
| 数据源 | Tushare `report_rc`（个股研报，限频10次/小时） |
| 计算方式 | 近30天研报数 vs 前30天研报数的变化率 |
| 分值映射 | 新覆盖(0->有)=70, 密集增加(>50%)=30, 稳定=50, 衰退=60 |
| 权重 | 0.10（辅助因子） |
| 研究依据 | 研报密集=关注度见顶=利好出尽（周期中后期成立） |
| 限制 | 限频严重，仅适用于小股票池（top-20），不进全量扫描 |

### 汇总

| 指标 | V4（4因子） | V6（6因子） | 差异 |
|------|------------|------------|------|
| 平均收益 | +88.8% | +90.8% | **+2.0pp** |
| 有变化的标的 | — | 2/10 | 80%无变化 |

### 关键发现

1. **80% 的标的 V4=V6（北向因子无影响）**：因为这些标的的北向持股数据缺失或变化不大，
   因子返回 None 被跳过，综合分不变。

2. **蓝特光学 +27.9pp（最大正向影响）**：北向持股从 4.0%→4.1% 微增，northbound_score
   略高于50，使得综合分在某次调仓时恰好越过买入阈值，多持有一段涨幅。

3. **广合科技 -9.5pp（最大负向影响）**：北向加仓较多（2.8%→4.3%），northbound_score
   较高，但高综合分导致在某次调仓时未触发卖出，随后回调。

4. **整体提升 +2.0pp**：北向因子对回测有微弱正向贡献，但改善幅度有限。
   这符合研究结论——北向资金是弱信号（50%套利/38%加仓），只能做辅助。

## 三、因子权重配置

六因子权重（非周期股默认）：

| 因子 | 权重 | 说明 |
|------|------|------|
| 动量 | 0.20 | 从0.25降（给新因子腾空间） |
| 估值 | 0.20 | 从0.25降 |
| 景气度 | 0.25 | 从0.30降 |
| 困境 | 0.15 | 从0.20降 |
| **北向** | **0.10** | 新增（辅助弱信号） |
| **研报** | **0.10** | 新增（辅助弱信号，限频场景） |

周期股仍用 CYCLICAL_FACTOR_WEIGHTS（估值0.40/困境0.25/动量0.20/景气0.15），
北向和研报使用 cfg 权重（各0.10）。

## 四、使用方式

```python
from davis_analyzer.backtest_factors import FactorConfig

# 全量扫描（关闭北向/研报，保持速度）
cfg = FactorConfig()  # enable_northbound=False, enable_research=False

# 小股票池（开启北向+研报）
cfg = FactorConfig(enable_northbound=True, enable_research=True)

# 只开北向（全量可行，hk_hold不限频）
cfg = FactorConfig(enable_northbound=True)
```

### 注意事项
- **北向因子可全量扫描**：hk_hold 接口限频正常，已加 SQLite 缓存
- **研报因子不可全量扫描**：report_rc 限频10次/小时，仅适用于 top-20 级别的小池
- **默认关闭**：FactorConfig 默认 enable_northbound=False, enable_research=False，
  保持与 V4 的向后兼容
