**买入天赐材料的原因**：动量分最高（96.7），景气度尚可（70.3）。德福科技排第 3，主要拖累项是**景气度 ΔG=-25.4**（增速急剧放缓），在六只中最差。

> **注**：此阶段估值分（valuation）和困境分（distress）均为 0/—，原因是回测初期 PE/PB 日频估值数据尚未缓存到对应日期窗口，daily_basic 拉取范围为 3 年但部分标的的早期数据缺失。这导致 4/29 之前的因子实际只用了动量+景气度两维。

#### 01-12 → 01-19 → 02-02 → 02-09（阳光电源 vs 天赐材料轮换）

这一阶段阳光电源和天赐材料交替第一。阳光电源景气度更高（77.9 vs 70.3），天赐材料动量更高。两者综合分咬得很紧（差距 <3 分），导致频繁切换。

**卖出原因**：每次调仓时排名从 #1 跌到 #2 → 触发卖出。这是定期调仓 top-1 策略的固有特征——**只留综合分最高的 1 只，排名交替即换仓**。

德福科技动量分升至 76.7（6 只最高），但景气度（62.2）和 ΔG（-25.4）拖累，仍排第 3。**差阳光电源仅 2.5 分**。

德福科技动量分持续走高（65→77→73→74→69），稳定排第 2，但景气度（62.2）始终不够。阳光电源靠景气度 77.9 维持第 1。

### 关 发现

1. **动量是德福科技的核心驱动力**：动量分从 62.8（2/9 低谷）→ 100.0（5/11 满分），与股价 27 元 → 130 元的涨幅高度同步。权重 0.25 的动量因子在估值分缺失的阶段贡献了大部分综合分。

2. **ΔG 转正是景气度拐点**：01-05 的 ΔG=-25.4（增速急剧放缓）→ 04-29 的 ΔG=+7.2（增速重新加速），这一变化反映了 2026Q1 季报披露后的基本面改善。ΔG 转正在景气度框架中等于"二次点火"信号。

3. **估值分是一把双刃剑**：06-01 估值分启动后给德福科技打了 0.0（极端高估），直接导致综合分从 74.8 暴跌至 47.8，触发卖出。但随后估值分修复至 23.8 → 30.8，加上动量满分支撑，6/23 重新登顶。

4. **困境分贡献有限**：困境分在 33-36 区间波动，对综合分影响较小（权重 0.20，但分值不高）。

## 七、方法论反思

### 策略有效性

| 维度 | 评价 |
|------|------|
| **选股能力** | ✓ 因子排名成功捕捉了德福科技的动量爆发和 ΔG 转正 |

| **换手成本** | ✗ 19 笔交易成本 ¥11,081，占初始资金 1.1%，频繁切换侵蚀收益 |

### 已知局限

1. **估值因子数据缺口**：4/29 之前估值分全部为 0，导致前半段实际只用动量+景气度两维打分。若估值分全程可用，德福科技可能在更早期就因高估值被压制。
2. **top-1 满仓风险**：单标的满仓导致每次切换都是 100% 换手，回撤极大。top-3 或 top-5 分散持仓会显著降低波动。
3. **定期调仓的刚性**：5 日固定调仓不考虑中途信号变化，可能在趋势中过早卖出（如德福科技 6/2 卖出后继续上涨至 142 元）。
4. **同行业 universe 的选择偏差**：6 只电气设备股票高度相关（β 重叠），并非真正的分散。
