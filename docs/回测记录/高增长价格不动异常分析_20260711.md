- 46% 的异常标的利润直接为负增长（营收涨 50%+，利润跌 50%+）

- 本质：营收增长是靠**降价抢份额/低价扩张/行业产能过剩**换来的，不是真实的需求爆发

## 五、对超级周期检测器的启示

### 应增加的负向过滤器

1. **利润增速必须为正**：G(营收) > 50% 但利润增速为负 = 增收不增利 = 直接排除
   - 这一条就能过滤掉 46% 的异常标的

2. **营收和利润方向一致性检验**：营收增速和利润增速的符号必须一致
   - 营收涨利润跌 = 降价换量，不是真实景气
   - 这是超级周期和伪增长的核心区别

3. **低基数过滤**：G > 200% 的需额外检查利润是否同步高增长
   - 如果利润没跟上，大概率是低基数效应而非结构性增长

### 更新后的超级周期先行信号公式

```
超级周期先行信号 = 原有条件
                  + 利润增速 > 0（排除增收不增利）
                  + 营收增速与利润增速方向一致
```

这个过滤预计能把异常标的（92%下跌）从信号中剔除，进一步提升 precision。
