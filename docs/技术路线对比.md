# 光通信三大技术路线深度对比：硅光 · InP · MicroLED — 投资者技术路线图

> **报告定位**: 光通信产业投资研究系列报告
>
> **核心结论**: 硅光、磷化铟(InP)、MicroLED三大技术路线并非零和博弈，而是走向异构集成。2024-2026年硅光模块+InP光芯片是确定性最高的投资主线；2026-2028年CPO/NPO生态加速落地；2028年后技术融合格局成型。

---

## 一、硅光集成 (Silicon Photonics)

### 1.1 技术原理

硅光集成利用现有CMOS制造基础设施，将光学元件（调制器、探测器、波导、复用器等）集成到硅基芯片上。核心优势在于可复用300mm晶圆CMOS产线，实现大规模、低成本制造。

硅作为间接带隙半导体，无法自身发光，激光器需要外部耦合或通过异质集成III-V材料来解决。这一限制是硅光技术路线的关键瓶颈。

**关键组件成熟度:**

| 组件 | 方案 | 成熟度 |
|---|---|---|
| 调制器 | MZI或MRM | HVM量产 |
| 探测器 | Ge-on-Si | HVM量产 |
| 激光源 | 外部CW激光器(主流)、异质集成(演进中) | 外部CW已量产 |
| 波分复用 | AWG/EDG/微环 | HVM量产 |
| 光纤耦合 | 光栅耦合器/端面耦合 | HVM量产 |

**关键限制:**

- **激光集成**: 需外部耦合，无法在硅上直接生成激光
- **热管理**: 微环调制器温度敏感性约1GHz/°C，需精密温控
- **耦合损耗**: 光纤到芯片的耦合效率仍有优化空间

### 1.2 速率路线图

| 速率 | 时间表 | SiPh渗透率 |
|---|---|---|
| 400G | 2020-2024已成熟 | ~30-40% |
| 800G | 2024-2026主流部署 | 预计60-70%(2026) |
| 1.6T | 2025-2028 | 预计70%+ |
| 3.2T | 2027-2030 | SiPh几乎唯一可行方案 |

**核心判断**: 800G硅光BOM成本约$243，比EML方案(~$310)低22%，且EML芯片产能只能满足2026年1.6T需求的~30%。硅光在3.2T时代将成为"几乎唯一可行方案"，因为传统EML技术路线面临产能天花板。

### 1.3 成本与市场规模

- 硅光PIC市场规模: $3.11B(2025) → $10.36B(2030)，CAGR 27.2% (IDTechEx)
- Intel/Jabil累计出货超800万颗PIC，验证了大规模量产能力

**每Gbps成本趋势:**

| 速率 | SiPh | InP/EML |
|---|---|---|
| 400G | ~$0.75 | ~$1.0 |
| 800G | ~$0.50 | ~$0.65 |
| 1.6T | ~$0.25-0.35 | EML产能不足 |

## 二、磷化铟 (InP)

### 2.1 技术原理

磷化铟是**唯一能在单一材料平台上实现激光器+调制器+探测器单片集成的光子材料**。作为直接带隙半导体，InP覆盖1.26-1.625μm通信波段，天生具备发光能力。

**InP PIC核心能力:**

- DFB/DBR/SGDBR可调谐激光器: >15mW输出功率，>45nm调谐范围
- EAM电吸收调制器
- InGaAs PIN/APD探测器
- SOA半导体光放大器

### 2.2 不可替代领域

InP在以下场景具有极难被替代的技术壁垒:

**长距相干传输(10km-数千km)**: 高功率、窄线宽、可调谐特性，替代难度极高。电信骨干网、跨洋通信离不开InP。

**可调谐激光器**: SGDBR调谐范围>45nm，是DWDM系统的核心光源。

**高功率CW光源**: >300mW输出，是CPO/SiPh方案不可或缺的外部激光源。

**EML高速调制**: 100G/200G EML是当前高速光模块的核心调制方案。

### 2.3 制造瓶颈: 产能是核心矛盾

**晶圆尺寸与价格:**

| 尺寸 | 份额(2026) | 价格 |
|---|---|---|
| 2英寸 | ~46% | $180-260 |
| 3英寸 | ~26% | $320-420 |
| 4英寸 | ~28% | $520-740 |
| 6英寸 | 刚起步 | N/A |

**产能瓶颈的关键数据:**

- 2025年全球InP需求约200-210万片(2英寸当量)，有效产能仅60-70万片，**缺口>70%**
- InP晶圆价格是同等GaAs的3-5倍
- 全球InP衬底被日本住友(~60%)、美国AXT(~35%)垄断
- Infinera正在建设全球首个6英寸InP量产线(CHIPS Act资助)，预计2026投产
- InP-on-GaAs工程衬底有望将成本降至InP原片的<20%

## 三、MicroLED光通信

### 3.1 技术原理

MicroLED光通信利用GaN基MicroLED作为可见光/近红外通信光源。与传统激光方案的根本区别: 无需激光阈值、无需温度稳定、可实现超高密度阵列、与CMOS单片集成。

这意味着MicroLED在功耗效率、可靠性和集成密度上具有天然优势，代价是调制带宽和传输距离目前受限。

### 3.2 技术突破

| 指标 | 最新记录 |
|---|---|
| 单颗调制带宽 | >2 GHz(绿光) |
| 单颗最大速率 | 10.5 Gbps (NRZ-OOK) |
| 系统级速率 | 10.23 Gbps @ 10m自由空间(OFDM) |
| 能耗 | 7.34 pJ/bit(2.125Gbps); Avicena 80 fJ/bit |

### 3.3 产业进展

**Avicena LightBundle™**

- 320通道MicroLED阵列，每通道3.5 Gbps
- 总吞吐量最高896 Gbps
- eKit评估套件计划2026年3月发布

**MediaTek + Microsoft Research (2026年3月)**

- 联合开发Active MicroLED Cable(AOC)
- 比传统VCSEL AOC功耗降低50%
- 可靠性匹配铜缆，是激光方案的100倍
- 可扩展至800 Gbps

### 3.4 投资时间线

- **2023-2025**: 实验室突破 ✅
- **2026**: 原型/评估平台 ✅
- **2026-2027**: 早期商业化(AOC)
- **2028-2030**: 数据中心规模部署

**投资判断**: 2026-2027仍为概念验证/早期导入期，适合长期(2028+)主题投资。

## 五、技术融合: 2030光通信技术栈

三大技术不会互相替代，而是走向异构集成，各自在最擅长的领域发挥价值。

### 5.2 融合路径

| 阶段 | 方案 | 状态 |
|---|---|---|
| 当前主流 | SiPh + InP CW激光器 | HVM量产 |
| Intel已量产 | SiPh + InP异质集成 | HVM，累计>800万颗 |

| 原型阶段 | MicroLED + CMOS(Avicena) | 评估套件2026年发布 |

### 5.3 NVIDIA GTC 2025 标志性意义

NVIDIA发布Quantum-X(2025H2)和Spectrum-X(2026H2) CPO交换机，基于TSMC COUPE平台的3D hybrid bonding封装。这标志着硅光CPO从实验室正式走向产品化，是整个光通信产业的技术分水岭。

## 六、投资策略

### 6.1 风险/回报对比

| 维度 | SiPh | InP | MicroLED |
|---|---|---|---|
| 确定性 | ★★★★★ | ★★★★ | ★★ |
| 成长空间 | ★★★★★ | ★★★★ | ★★★★★(若成功) |
| 估值(PE) | 25-50x | 100-600x | 主题期 |
| 业绩兑现 | 2024-2026已兑现 | 2025-2026开始 | 2028+ |

### 6.2 分阶段投资主线

**短期(2024-2026): SiPh光模块 + InP光芯片**

🥇 中际旭创、新易盛 → 🥈 天孚通信、源杰科技 → 🥉 仕佳光子

**中期(2026-2028): CPO/NPO生态 + InP产能释放**

🥇 中际旭创(NPO/XPO 2027量产)、天孚通信(CPO光引擎) → 🥈 源杰科技/仕佳光子(InP产能释放)

**长期(2028-2030): 技术融合**

🥇 SiPh CPO生态链 → 🥈 MicroLED互连(Avicena/MediaTek) → 🥉 TFLN高端调制

### 6.3 关键风险警示

1. **InP光芯片估值风险**: 源杰PE>600x，业绩兑现压力巨大
2. **SiPh渗透率天花板**: 800G/1.6T已达60-70%+，增量空间需关注CPO新形态
3. **MicroLED商业化不确定性**: 从实验室到数据中心规模部署，技术/成本/产业链均需验证
4. **地缘风险**: InP衬底高度依赖日本住友，国产化替代进度需持续跟踪
5. **CPO大规模部署时间**: 预计2028-2030，短期预期可能跑在基本面之前
