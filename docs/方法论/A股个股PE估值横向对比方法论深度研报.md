# A股个股 PE 估值横向对比方法论

> **研究日期**：2026-06-25 | **方法论定位**：个股绝对 PE 历史分位的补充，解决"个股 PE 中位但市场高位"的估值幻觉
> **实现模块**：`stockhot/valuation/`（用 Tushare `index_dailybasic` + `daily_basic` 自动计算）

---

## 核心问题

个股 PE 处于自身历史中位（如 50% 分位），但市场指数也处于高位（如 90% 分位）。此时个股"看起来不贵"，但实际可能已经随市场系统性抬升而溢价。**横向锚定市场基准，才能判断是否真正便宜。**

---

## 三种方法

### 方法一：相对 PE 溢价率

```
相对PE溢价率 = 个股PE / 基准指数PE
```

**基准选择**（自动匹配，`stockhot/valuation/_benchmark_for()`）：
- 主板个股 → 沪深300（000300.SH）
- 创业板个股 → 创业板指（399006.SZ）
- 科创板个股 → 科创50（000688.SH）

**判断**：用比值的历史分位替代绝对 PE 分位。溢价率分位 <40% = 相对折价（便宜），>75% = 相对溢价（贵）。

### 方法二：ERP 法（股权风险溢价）

```
ERP = 盈利收益率(1/PE) - 无风险利率(10Y国债≈Shibor1Y+0.7pp)
```

ERP ≥3% 明显便宜 / 1.5-3% 合理 / 0-1.5% 偏弱 / <0 偏贵（不如国债）。

### 方法三：PE-Band 二维定位

| 象限 | 个股PE分位 | 市场PE分位 | 含义 |
|:---:|:---:|:---:|------|
| **①** | 高 | 高 | 两者都贵（最危险） |
| **②** | 高 | 低 | 个股贵/市场便宜（相对溢价） |
| **③** | 低 | 低 | 两者都便宜（最佳买点） |
| **④** | 低 | 高 | 个股便宜/市场贵（相对折价） |

> **市场背景**：沪深300 PE_TTM=14.7（3年 98.4% 分位）、创业板指 PE_TTM=55.1（99.9% 分位）——**两大指数均处于 3 年估值极高位**。

### 关键发现

1. **全部 11 只标的落入 Q1 象限**——当前市场整体处于 3 年估值极高位，几乎所有标的都被系统性抬升
2. **阳光电源是唯一亮点**——相对 PE 溢价率仅 0.48x（28.6% 分位，创业板内相对折价）、ERP 1.59%（吸引力合理）。它的"贵"主要来自市场系统性高位，而非个股层面溢价
3. **ERP 普遍为负**（除阳光电源外）——买这些股票的盈利收益率不如买国债，这是市场整体高估的强烈信号
4. **驰宏锌锗/彤程新材**相对溢价率分位达 99%+，即使在市场高位中也属极端

> **方法论价值**：这套横向对比揭示了单纯看个股 PE 分位的盲区——在市场整体高位时，即使个股 PE 分位不高（如佛塑 47%），相对市场基准仍可能无吸引力（ERP 为负）。
