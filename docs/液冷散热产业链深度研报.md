## 一、液冷完整产业链图谱

### 1.1 产业链总览

液冷产业链分为**上游零部件、中游系统集成、下游终端应用**三大环节：

```
┌─────────────────────────────────────────────────────────────────────┐
│                        液冷产业链全景                                 │
├─────────────┬───────────────────────────────────────────────────────┤
│             │ ● 冷却液：氟化液（3M退出后巨化/新宙邦替代）、去离子水 │
│             │ ● 铜/铝材（冷板原材料）                               │
│  上游零部件 │ ● 液冷快接头（UQD）：Stäubli、CPC、丹佛斯、中航光电   │
│             │ ● 泵/阀门/传感器（磁力泵、电磁阀等）                   │
│             │ ● 密封材料（氟橡胶O-ring、EPDM等）                    │
│             │ ● 换热器（钎焊板式换热器）                            │
├─────────────┼───────────────────────────────────────────────────────┤
│             │ ● 冷板（Cold Plate）：微通道/3D流道，壁垒较高          │
│  中游组件   │ ● CDU（冷量分配单元）：液冷系统"心脏"，毛利率最高      │
│  与系统     │ ● Manifold（歧管）：流量分配管道                       │
│             │ ● 漏液检测系统、浸没式液冷机柜（TANK）                  │
│             │ ● 液冷管路（柔性管/硬管）                              │
├─────────────┼───────────────────────────────────────────────────────┤
│             │ ● 服务器液冷集成（ODM/代工厂）                        │
│  下游应用   │ ● 数据中心液冷基础设施                                │
│             │ ● 电源液冷集成                                        │
│             │ ● 算力中心整体液冷方案                                │
└─────────────┴───────────────────────────────────────────────────────┘
```

**数据来源**：中国信通院《智算中心液冷产业全景研究报告》（2025年），国信证券研报（2026年1月）

### 1.2 上游：核心材料与元器件

#### 1.2.1 冷却液/氟化液

**市场格局（3M退出后）**：

3M于2025年底如期完成PFAS制造的全面退出，其Novec和Fluorinert系列氟化液全线停产，撕开约10-15亿美元（百亿级人民币）的全球市场重组空间。

| 企业 | 国别 | 产品/地位 | 进展 |
|------|------|----------|------|
| **3M** | 美国 | Novec/Fluorinert系列（原占70%份额） | 2025年底全面停产 |
| **Solvay（索尔维）** | 比利时 | HFO体系环保制冷剂 | 2025年电子级产品双位数增长 |
| **Chemours（科慕）** | 美国 | Opteon 2P50，专为两相浸没设计 | 2025年正式投放市场 |
| **Honeywell** | 美国 | Solstice E-Cooling平台 | 北美数据中心本地化优势 |
| **Daikin（大金）** | 日本 | 亚太半导体供应链 | PFPE市场约20%份额 |
| **新宙邦(300037)** | 中国 | Boreaf系列电子氟化液 | 已通过台积电2nm认证，2026Q1净利同比+100% |
| **巨化股份(600160)** | 中国 | "巨芯"冷却液项目 | 规划5000吨/年，一期1000吨已投产 |
| **永太科技(002326)** | 中国 | 氟化液布局中 | 产品验证阶段 |

**国产替代进度**：亚化咨询预计未来2-3年，国产高端电子级氟化液市场占有率将从不足10%提升至40%以上。

**重要变化**：北美头部科技巨头（NVIDIA、Meta、Microsoft）在最新一代AI算力集群中，明显向冷板式液冷（Direct-to-Chip）倾斜，主要使用去离子水或水乙二醇等**非氟流体**。非氟流体已占据浸没式液冷市场超过55%份额。

| 中航光电(002179) | 中国 | 已突破海外垄断，量产经验丰富 |
| 英维克(002837) | 中国 | 英特尔UQD互插联盟成员，列入NV MGX生态 |

**数据来源**：Valuates Reports统计、天风证券研报、英特尔官方公告

#### 1.2.3 泵/阀门/传感器/密封材料

- **磁力泵**：高端市场仍以进口为主（丹佛斯等），国产化率约35%→60%提升中
- **电磁阀**：2024年中国市场约23.23亿元，2026年预计达24.35亿元。三花智控、盾安环境等
- **密封材料**：EPDM橡胶双密封圈为主流，航空航天级要求

### 1.3 中游：液冷组件和系统（价值量最集中环节）

#### 核心零部件价值量占比（以GB300 NVL72单机柜为例）

| 零部件 | 价值量占比 | 毛利率 | 壁垒 | 市场现状 |
|--------|----------|--------|------|---------|
| **冷板（Cold Plate）** | 41% | 20%-60% | 微通道/3D流道设计制造 | 英伟达定制冷板40%-60%，普通冷板20%-30% |
| **CDU** | 32% | 40%-60% | **产业链毛利最高环节** | 钎焊板式换热器45%，循环泵35% |
| **UQD（快接头）** | 14% | 40%-50% | 密封工艺+认证壁垒 | 一体化方案毛利率40%-50% |
| **Manifold（歧管）** | 13% | 30%-35% | 定制化设计 | 竞争较激烈 |

**数据来源**：东莞证券研报（2026年3月），硬科技洞察

#### 1.3.1 冷板（Cold Plate）

- **技术演进**：从传统铜铝向复合材料升级，加速向微通道（MLCP）、3D流道结构演进
- **国内供应商**：英维克、飞荣达、中石科技、立敏达、曙光数创等
- **海外/台资供应商**：双鸿科技（Auras）、奇宏科技（AVC）、酷冷至尊（Cooler Master）、Boyd等
- **趋势**：两相冷板（热流密度385W/cm²，覆盖3kW级芯片）是下一代方向

#### 1.3.2 CDU（冷量分配单元）

- **功能**：液冷系统的"心脏"和"控制中枢"，负责冷却液压力调节、温度控制、杂质过滤
- **分类**：列间（In-Row）CDU vs 机架内（In-Rack）CDU；后者是高密度场景主流趋势
- **国内排名（2024年）**：申菱环境在中国液冷数据中心市场CDU厂商排名第一
- **核心厂商**：Vertiv、台达、英维克、申菱环境、高澜股份等

#### 1.3.3 浸没式液冷机柜（TANK）

- 曙光数创是国内唯一实现浸没相变液冷大规模商业化部署的企业
- 第三代C8000单机柜功率密度突破至860kW以上

### 1.4 下游：系统集成与应用

| 下游环节 | 代表企业 | 说明 |
|---------|---------|------|
| 服务器液冷集成 | 工业富联/富士康、超微、广达 | GB200/GB300最大代工厂 |
| 数据中心液冷基础设施 | Vertiv、英维克、申菱环境 | 系统级解决方案商 |
| 电源液冷集成 | 麦格米特、欧陆通、台达 | AI服务器高功率电源+液冷 |

### 1.5 英伟达液冷供应链竞争格局

以英伟达GB300液冷产业链为例，台资企业目前占据全球70%以上液冷组件市场份额：

| 组件 | 已确认/领先供应商 | 优秀供应商（含中国大陆） |
|------|-----------------|----------------------|
| 冷板 | AVC（奇宏）、Auras（双鸿）、Cooler Master | Boyd、CoolIT、品达、立敏达 |
| CDU | Vertiv、台达、光宝 | Cooler Master、英维克、比亚迪电子 |
| UQD | CPC、Stäubli、丹佛斯 | 中航光电、英维克、立敏达 |
| Manifold | 双鸿、奇宏、台达 | 品达、富士康、立敏达 |
| 系统集成 | 富士康、Vertiv、nVent | DCX、Trane、Nidec |

**关键变化**：英伟达GB300液冷供应链放开，谷歌TPU机柜液冷采取直采模式，国产厂商迎来直接切入全球头部供应链的战略机遇。

## 五、投资逻辑与风险提示

### 5.1 核心投资逻辑

#### 逻辑一：液冷是AIDC建设中"确定性最高"的环节

**无论哪家AI芯片胜出（英伟达/AMD/Google TPU/国产算力），液冷都是刚需。**这是典型的"卖铲子"逻辑——不赌具体AI芯片赢家，赌液冷渗透率提升。液冷需求直接取决于核心客户基础设施的建设进度，无需受制于下游AI软件应用的商业化落地情况。

#### 逻辑二：量价齐升的商业属性

单机柜算力密度每隔18-24个月翻倍，即使零部件单位功率成本下降，系统总工程造价依然急剧上升：
- GB200液冷系统造价7-10万美元/柜
- Rubin液冷系统预计10-15万美元/柜
- 液冷BOM占整机成本30%-40%

#### 逻辑三：产业链传导——上游材料→中游组件→下游集成

国产液冷企业正从零部件配套向系统级解决方案升级，产业链价值向上游高壁垒环节（CDU、UQD、氟化液）集中。

### 5.3 风险提示

#### 5.3.1 行业风险

1. **液冷渗透率提升不及预期**：液冷增加数据中心基础设施复杂性，管理成本上升可能影响渗透率
2. **技术路线不确定性**：两相vs浸没式的渗透节奏存在分歧
3. **标准化滞后**：不同液冷路线的标准化进程仍显滞后
4. **产能过剩风险**：当前多家企业扩产，若需求不及预期可能引发价格战

#### 5.3.2 公司特定风险

1. **曙光数创**：净利润下滑（2025年-40.64%），政府补助减少+研发投入增加；冷板液冷市场竞争加剧致毛利率下滑；存货同比+336%
2. **英维克**：2026Q1净利仅0.09亿（同比-82%），液冷尚未规模出货，产能扩张期费用压力大；应收账款增长带来坏账风险
3. **高澜股份**：数据中心液冷收入（高功率密度）同比-19.44%；市值较小，流动性风险
4. **麦格米特**：当前PE极高（419x TTM），业绩高度依赖AI电源放量节奏；汇兑损失风险（海外占比近40%）
5. **新宙邦**：数据中心业务处于早期阶段，对公司现在业绩影响非常小（公司官方提示）；电解液周期拖累
6. **冷板液冷竞争加剧**：市场规模提升吸引行业内外厂家争相入局，短期毛利率呈下降趋势

#### 5.3.3 宏观风险

1. 全球AI投资热潮若降温，液冷需求可能放缓
2. 中美贸易摩擦可能影响国产液冷企业出海
3. PFAS环保监管趋严可能影响氟化液路线
