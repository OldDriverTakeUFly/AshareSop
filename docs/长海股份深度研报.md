## 一、公司概况与发展历程

### 1.1 公司简介

江苏长海复合材料股份有限公司（简称"长海股份"，股票代码300196）成立于2000年5月，注册于江苏省常州市武进区，2011年3月29日在深交所创业板上市。公司是一家**拥有从玻纤生产、玻纤制品深加工到玻纤复合材料制造完整产业链的高新技术企业**，是国内规模最大的**无纺玻纤制品综合生产企业之一**，也是国内少数能制造高端玻纤毡制品并不断进行制品深加工的企业。

公司在**短切毡**和**湿法薄毡**细分领域处于**领军地位**，短切毡产品获评**国家第六批制造业单项冠军产品**。产品畅销全国30多个省市，远销北美、南美、欧洲、中东、东南亚、大洋洲、非洲等全球100多个国家和地区。

### 1.2 发展历程

| 时间 | 里程碑事件 |

### 1.3 子公司与控股结构

公司通过**"长海母公司 + 天马集团"**双主体运营模式，构建了完整的产业链闭环：

- **母公司（长海股份）**：主营玻纤纱、短切毡、湿法薄毡、玻纤织物、复合隔板、涂层毡、网格布、热塑性玻纤及制品等
- **全资子公司常州天马集团有限公司（原建材二五三厂）**：主营不饱和聚酯树脂、胶衣树脂、乙烯基酯树脂、丙烯酸树脂、玻纤粘结剂及乳液、玻纤增强材料等精细化工产品

天马集团是国家**专精特新小巨人企业**，建有江苏省玻璃钢/复合材料工程技术研究中心、江苏省不饱和聚酯树脂工程研究中心。两家公司形成紧密的产业链协同：天马集团的化工辅料是长海玻纤制品的重要原材料，而天马集团的玻纤增强材料产品则使用长海生产的短切毡、薄毡等。

## 三、主营业务与产品矩阵

### 3.1 业务架构

公司形成了**"玻纤纱 → 玻纤制品 → 树脂/复合材料"**的完整产业链：

```
┌─────────────────────────────────────────────────────┐
│                 长海股份产业链布局                       │
├─────────────────────────────────────────────────────┤
│                                                       │
│  上游原料          中游制造              下游应用       │
│  叶腊石/石英砂 → 玻纤纱生产 → 短切毡 → 汽车顶棚       │
│  石灰石/硼酸   → 池窑拉丝   → 湿法薄毡 → 建筑装饰       │
│  天然气/电力   → 深加工     → 玻纤织物 → 风电叶片       │
│                             → 涂层毡   → 电子基材       │
│                             → 网格布   → 蓄电池隔板     │
│                             → 复合隔板 → 储罐/管道      │
│                             → 热塑性   → 轨道交通       │
│                                       → 新能源汽车      │
│  天马集团协同：                                       │
│  树脂 ←→ 玻纤制品 ←→ 复合材料                          │
└─────────────────────────────────────────────────────┘
```

### 3.2 产品矩阵

| 产品类别 | 应用领域 | 市场地位 |
|----------|---------|---------|
| **玻纤纱** | 基材，供给下游制品加工 | 自用为主，部分外销 |
| **短切毡** | 汽车车顶棚、货车箱体、卫浴洁具、大型储罐、透明板材 | **细分龙头**，市占率国内超50%（车顶毡） |
| **湿法薄毡** | 内外墙装饰、屋面防水、电子基材 | **细分龙头** |
| **玻纤织物（多轴向织物）** | 风电叶片、复合材料增强 | 快速增长 |
| **复合隔板** | 蓄电池隔板 | 稳定供应 |
| **涂层毡** | 建筑装饰 | 成熟产品 |
| **网格布** | 建筑增强 | 成熟产品 |
| **热塑性玻纤及制品** | 汽车轻量化、轨道交通 | 战略新兴，填补国内空白 |
| **不饱和聚酯树脂（天马）** | 复合材料基体树脂 | 产业链协同 |

### 4.2 60万吨高性能玻纤智能制造基地项目

这是公司最重要的在建项目，也是未来增长的核心驱动力：

- **总投资**：63.47亿元（固定资产投资59.67亿元，流动资金3.80亿元）
- **选址**：江苏省常州市经开区遥观镇，新增用地520亩
- **开工时间**：2023年2月21日
- **建设规划**：分二期建设
- **一期进展**：第一条15万吨产线已于2024年Q3点火投产，目前处于产能爬坡期
- **全部达产后**：新增60万吨高强度高模量玻纤复合材料、低介电玻纤及织物产能
- **产品方向**：
  - 超高模量玻纤 → 超大功率海上风电叶片
  - 低介电玻纤及织物 → 5G高频PCB电路板（有望打破国外垄断）

### 4.3 产能关停与优化

2024年，公司积极进行产能结构优化：
- 2024年初和2024年10月先后关停两条老旧池窑生产线
- 累计关停产能近**12万吨**
- 以新换旧，实现高端化、智能化转型

### 4.4 制造技术水平

- **2个省级示范智能车间**：主打产品高性能玻纤短切原丝毡全流程智能化
- **粉料加工厂配套**：新建产线配备自有粉料加工厂，预计可带来吨成本下降约100元
- **能效优化**：智能产线天然气、电力单耗下降，预计带来100-150元/吨成本下降空间
- **综合成本节降**：新线预计带来**100-200元/吨**的综合成本下降

## 六、行业分析：中国玻纤市场全景

### 6.1 中国玻纤产业地位

中国是全球最大的玻纤生产国：
- **2024年国内玻纤纱总产量756万吨**，同比增长4.6%，连续两年保持低速增长
- **2025年国内玻纤纱总产量843万吨**，同比增长11.5%，结束低速增长，恢复较强增长
- 国内产量占全球比重超**70%**
- **2025年底国内在产池窑产能约838万吨**

| 新增产能（万吨） | 38 | 86 | — |
| 新建大型池窑（条） | — | 7 | — |

### 6.3 产能结构调控

2023年以来，行业龙头主动进行产能调控：
- **七大玻纤集团累计关停15条大型池窑生产线**，关停总产能达101.1万吨
- **长海股份**：2024年关停两条产线，累计关停近12万吨
- 头部企业持续推进产线冷修与高端化转型

### 6.4 行业盈利修复

- 2024年行业分别于3月、4月、5月、6月、11月及2025年2月进行**6轮复价**
- 复价产品从普通直接纱延伸至电子纱、薄布、短切毡、风电纱等中高端产品
- **2025年1-6月**：规模以上玻纤企业主营业务收入同比+13.5%，利润总额同比+142.5%

### 6.6 行业发展趋势

1. **低端产能出清**：环保政策（如欧盟CBAM碳关税）和成本压力加速中小企业退出
2. **高端化转型**：5G/AI带动电子级玻纤需求爆发，低介电常数产品毛利率高
3. **集中度提升**：预计2026-2030年行业CR5将提升至75%，CR10突破90%

## 九、增长驱动力与战略展望

### 9.1 核心增长驱动力

#### 驱动力一：产能释放（量增逻辑）
- **60万吨项目**一期第一条15万吨产线已投产，后续产线将根据市场需求有序推进
- **天马集团8万吨技改**已完成，贡献高端产品增量
- 2025年玻纤制品销量39.19万吨，同比+30.1%
- 未来随着60万吨项目全部达产，公司总产能有望翻倍增长

#### 驱动力二：产品高端化（价升逻辑）
- 超高模量玻纤 → 超大功率海上风电
- 低介电玻纤 → 5G/AI PCB板
- 热塑性复合材料 → 汽车轻量化
- 2024-2025年行业6轮复价，价格持续回升

#### 驱动力三：成本节降（利润率提升）
- 智能制造产线带来100-200元/吨成本下降
- 粉料加工厂配套降低原料成本
- 天然气、电力单耗下降
- 规模效应摊薄固定成本

#### 驱动力四：行业景气修复
- 玻纤行业从2024年Q1触底回升
- 产能投放明显放缓，供需结构改善
- 2025年行业利润总额同比增长78.4%

### 9.2 未来发展战略

1. **产能战略**：稳步推进60万吨项目后续产线投放，根据市场需求灵活调节节奏
2. **产品战略**：从"规模扩张"迈向"高端化、智能化、绿色化"
3. **市场战略**：深耕国内市场（内需修复），优化海外出口产品结构
4. **技术战略**：持续加大研发投入（2024年研发费用率4.99%），攻坚高端产品
5. **产业链战略**：强化玻纤-树脂-复材产业链协同，提升一体化竞争优势

### 11.2 特别关注：行业周期波动

玻纤行业的周期性是投资长海股份最核心的考量因素：

```
2018-2019年：行业低迷 → 2020年：疫情后复苏 → 
2021-2022年：景气周期高点（量价齐升） → 
2023-2024年：行业调整期（价格下跌、盈利下滑） → 
2025年：新一轮景气上行周期启动
```

> **关键判断**：当前处于行业景气修复期，但需警惕如果产能投放过快、需求恢复不及预期，行业可能再次陷入价格竞争。

## 十二、投资结论

### 12.1 投资逻辑总结

长海股份的投资逻辑可概括为**"产能弹性+制品龙头+行业修复"**三重逻辑：

1. **产能弹性最大**：60万吨项目（总投资63.5亿元）是同梯队企业中最大的产能扩张计划，全部达产后产能翻倍。15万吨新线已投产，增量兑现中。

2. **制品领域龙头**：在短切毡（国内车顶毡市占率超50%）、湿法薄毡等细分领域具有定价权。制造业单项冠军产品构筑竞争壁垒。

3. **行业景气修复**：2024年Q1触底，6轮复价推动价格回升。2025年行业利润总额同比+78.4%，公司全年营收+17.9%，归母净利润+18.8%。

4. **产业链一体化优势**：从纱→制品→树脂→复合材料的完整闭环，国内少数企业具备。

5. **估值安全边际**：P/B 1.22倍在行业处于偏低水平，2026E PE约11-14倍，若产能释放顺利，存在估值修复空间。

- ✅ 60万吨项目后续产线点火投产节奏
