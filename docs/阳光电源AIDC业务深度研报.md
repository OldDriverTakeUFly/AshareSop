# 阳光电源(300274.SZ) AIDC业务深度研报

> 调研日期：2026年5月31日 | 股价：约153元 | 市值：约2200亿元
> 数据截止：2025年年报 + 2026年Q1公开信息

---

## 核心结论（TL;DR）

**阳光电源在AIDC领域属于"跨界新兵"，技术储备深厚但业务处于极早期阶段。** 公司2025年5月正式宣布进入AIDC电源赛道，8月成立AIDC事业部，2026年力争实现产品落地和小规模交付。AIDC业务目前对公司营收贡献几乎为零，但被定位为继光伏逆变器、储能系统之后的**"第三增长曲线"**。

**核心定位差异**：阳光电源不走传统UPS路线，而是押注**SST（固态变压器）+800V HVDC+构网型储能**的前沿技术架构，瞄准的是AIDC供电架构的下一代变革，而非存量替代市场。

**投资判断**：AIDC概念为阳光电源提供了估值想象空间，但2026-2027年是验证期，短期内不产生实质性营收贡献。股价催化剂来自客户签约进展和产品发布节奏。

## 一、阳光电源AIDC产品线全景

### 1.1 已发布/规划中的AIDC相关产品

| 产品/技术 | 状态 | 说明 |
|-----------|------|------|
| **SST固态变压器** | 研发中，2026年发布 | 35kV/6MW样机已完成，面向数据中心中压直连，预研近10年 |
| **800V HVDC电源系统** | 研发中，2026年Q4推出 | 兼容±400V和800V架构，适配英伟达GB200/Rubin平台 |
| **构网型储能（Grid-Forming BESS）** | 已有成熟产品 | PowerTitan 3.0液冷储能系统，7.14MW/6.9MWh，RTE达92%+ |
| **液冷储能系统** | 已量产 | PowerTitan 2.0/3.0系列，全球规模交付 |
| **光储一体化方案** | 已有成熟方案 | 园区级光伏+储能+微电网，适配AIDC绿电需求 |
| **"从网到芯"整体方案** | 白皮书已发布 | GRES 2026发布《未来就绪AI数据中心供电解决方案》白皮书 |

### 1.2 技术逻辑与优势

阳光电源切入AIDC的核心逻辑是**技术降维**：

1. **高压平台降维**：公司逆变器和储能系统长期运行在1500V平台，800V直流对其是"降规格"操作，技术迁移门槛低
2. **SST预研积累**：阳光电源在35kV固态变压器上已预研近10年，**这个积累在现有数据中心电源竞争者里几乎找不到可比对象**（来源：芯流AIDC专栏）
3. **构网型技术**：Stem-Cell Grid-Forming Tech 2.0已实现20ms快速响应、GW级黑启动能力，可支持AIDC毫秒级负荷波动平抑
4. **SiC（碳化硅）功率器件**：PowerTitan 3.0已采用全液冷SiC PCS，最大效率99.3%，与800V HVDC体系天然契合

根据GRES 2026发布会及公司公告，阳光电源AIDC产品路线图如下：

### 3.2 2026年3月31日投资者关系活动记录表（1000+机构参加）

关键问答摘录：

**Q：公司AIDC订单的进展？**

> **A：** 储能在AIDC中主要有供电+备电、平抑负荷波动两大应用场景。前者是传统的光储提供电源的项目，**相关订单一直都有**；后者是数据中心内的电能质量管理，需要实现毫秒级响应，**公司正在配合客户做定制化研发，目前还没有订单**。

**Q：AIDC配储需求的潜力和公司的优势，及SST的进展？**

> **A：** 公司AIDC事业部会在中压直连技术、高密度模块化电源技术、直流配电技术等方面进行重点研发投入和产品规划。行业发展很快，现在有户外电源、预制舱、户内机架式电源和板载电源等不同方面的需求，未来我们产品会覆盖到AIDC的各种场景。客户方面，**目前已经与部分国际头部云服务厂家及国内头部互联网企业在开展一些产品架构的定义**，目标是在今年有产品落地和小规模交付，明年开始批量。**AIDC的复合增长率比光伏、风电、储能要快**，互联网企业的优势是比较能接受新鲜事物，他们很开放和主动，我们会借助原来电力电子领域的经验，在800V方面尽快发力，**我们对SST的研发、产品、客户服务等方面充满信心**，现在投入很大，希望到今年下半年出一些成果。

### 5.3 阳光电源的差异化优势

1. **SST预研深度独一无二**：35kV/6MW样机已完成，覆盖60kV及以下交直流机型，输出240-800VDC，功率0.5-15MW。在AIDC领域，SST被视为供电架构的"终局形态"
2. **储能+电源一体化**：全球仅阳光电源同时具备大储系统集成能力和电力电子技术，可实现"绿电→储能→供电"全链条
3. **构网型技术（Grid-Forming）**：已实现20ms快速响应、GW级黑启动，可支持AIDC毫秒级负荷波动平抑，这是传统UPS厂商不具备的
4. **全球品牌和渠道**：海外收入占比>60%，全球520家服务网点，BloombergNEF可融资性排名第一

### 5.4 阳光电源的挑战

1. **数据中心不允许试错**：这是一个容错率极低的行业，新供应商从测试到规模化订单需要1-2年
2. **客户关系从零开始**：维谛、伊顿、施耐德等在全球头部云厂商中经营多年，客户粘性极高
3. **认证周期长**：英伟达认证周期12-18个月，且已有台达、光宝、麦格米特三家占据超95%份额
4. **AIDC事业部核心团队来自外部**（台达、伊顿），需要时间磨合
5. **SST/800V HVDC属于前沿路线**，如果行业技术路线选择不同，押注可能落空

## 七、风险提示

1. **AIDC业务仍处极早期**：目前营收贡献为零，2026-2027年为验证期
2. **信息可靠性**：部分客户进展信息（如AWS 8GWh订单、Meta签约等）来自二级市场信息聚合平台，尚未在公司正式公告中独立确认
3. **竞争加剧**：宁德时代、德业股份等同样在布局AIDC储能，竞争格局远未定型
4. **技术路线不确定性**：SST/800V HVDC是否成为行业主流标准尚无定论
5. **地缘政治风险**：阳光电源是中国企业，进入北美数据中心供应链面临合规审查
6. **估值风险**：市场可能已部分price in AIDC概念，但实际业务贡献尚需2-3年
