# Source Traceability Checklist

在报告撰写完成后、终审前使用此清单。验证报告中每一个数据点均有可追溯的来源标注。

## tushare 数据标注

- [ ] 每个来自tushare的数字后标注（来源：tushare YYYY-MM-DD）
- [ ] 财务报表字段标注具体接口名（如 income / balancesheet / cashflow / fina_indicator / daily_basic）
- [ ] 同一数字多处引用时，每处均保留标注

## 公告 / 年报数据标注

- [ ] 每个来自公告/年报的数字后标注（来源：{公司}{年份}年报/公告 YYYY-MM-DD）
- [ ] 公告数据标注披露日期（巨潮资讯网 / 交易所公告）
- [ ] 年报附注数据标注所在章节或页码

## 券商 / 协会数据标注

- [ ] 每个来自券商报告/行业协会的数字后标注（来源：{机构名} {报告名}）
- [ ] 网络调研数据标注 source URL 或机构名 + 发布日期
- [ ] 海外数据（如年报 10-K / 协会统计）标注原文出处

## 密度检查

- [ ] 统计source标注密度 ≥80%（数字总数中有标注的占比）
- [ ] 无 source 的裸数字逐一排查，补标或删除

## 可追溯抽查

- [ ] 抽查10个关键数字，每个能在evidence/目录找到原始数据文件
- [ ] 抽查数字覆盖不同来源类型（tushare / 公告 / 券商 / 协会）
- [ ] 抽查结果记录：N/10 可追溯，不可追溯项已处理
